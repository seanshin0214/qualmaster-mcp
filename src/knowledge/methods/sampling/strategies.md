# 질적연구 표집 전략 (Sampling Strategies)

## 목적적 표집 (Purposeful Sampling)

### 정의
연구 목적과 연구 질문에 가장 적합한 정보 풍부한 사례를 의도적으로 선택하는 전략

### vs 확률 표집
| 확률 표집 | 목적적 표집 |
|---------|-----------|
| 무작위 선택 | 의도적 선택 |
| 대표성 추구 | 정보 풍부함 추구 |
| 일반화 목적 | 심층 이해 목적 |
| 표본→모집단 | 사례→현상 |

## 16가지 목적적 표집 전략 (Patton, 2015)

### 1. 극단/일탈 사례 표집 (Extreme/Deviant Case)
- 비정상적, 특이한 사례
- 경계 조건 탐색
- 예: 탁월한 성공, 심각한 실패

### 2. 강도 표집 (Intensity Sampling)
- 현상을 강하게 나타내는 사례
- 극단은 아니지만 정보 풍부
- 예: 뛰어난 교사 (최고는 아님)

### 3. 최대 변이 표집 (Maximum Variation)
- 가장 다양한 사례
- 변이의 전 범위 포착
- 공통 패턴 더 의미 있음

```
예: 학교 적응 연구
- 도시/농촌
- 남/여
- 고학년/저학년
- 높은 적응/낮은 적응
```

### 4. 동질 표집 (Homogeneous Sampling)
- 유사한 특성 공유 사례
- 특정 하위집단 초점
- 포커스 그룹에 적합

### 5. 전형적 사례 표집 (Typical Case)
- 평균적, 일반적 사례
- "보통" 상황 이해
- 프로그램 평가에 유용

### 6. 층화 목적적 표집 (Stratified Purposeful)
- 하위집단별 사례 선택
- 그룹 간 비교
- 변이 포착 + 비교

### 7. 결정적 사례 표집 (Critical Case)
- 논리적으로 결정적인 사례
- "여기서도 ~면 어디서도 ~"
- 일반화 근거 제공

```
예: "이 어려운 학교에서 프로그램이 효과 있다면,
    다른 학교에서도 효과 있을 것"
```

### 8. 눈덩이/연쇄 표집 (Snowball/Chain)
- 참여자가 다음 참여자 추천
- 접근 어려운 집단
- 네트워크 활용

**절차**
1. 초기 정보제공자 확보
2. "이 경험 가진 다른 분 아세요?"
3. 추천받은 사람 접촉
4. 반복

### 9. 기준 표집 (Criterion Sampling)
- 사전 정의된 기준 충족
- 품질 보증 목적
- 명확한 선정 근거

```
예: 참여자 기준
- 경험 5년 이상
- 특정 사건 경험자
- 특정 직위 보유자
```

### 10. 이론 기반 표집 (Theory-Based)
- 이론적 구성개념 대표
- 이론 검증/발전
- 연역적 접근

### 11. 확인/불확인 사례 (Confirming/Disconfirming)
- 분석 후 추가 표집
- 패턴 확인 또는 반박
- 분석적 엄격성

### 12. 편의 표집 (Convenience)
- 접근 용이한 사례
- 시간/비용 절약
- 품질 위험 있음 (주의)

### 13. 정치적 중요 사례 (Politically Important)
- 관심/가시성 높은 사례
- 정책 영향력
- 이해관계자 관심

### 14. 기회 표집 (Opportunistic)
- 현장에서 예상치 못한 기회
- 유연한 대응
- 창발적 설계

### 15. 무작위 목적적 표집 (Random Purposeful)
- 목적적 표집 내 무작위
- 편향 감소
- 신뢰성 향상

### 16. 복합 표집 (Combination/Mixed)
- 다중 전략 사용
- 삼각화
- 유연성과 엄격성

## 이론적 표집 (Theoretical Sampling)

### 근거이론 전용

**정의**
분석 결과가 다음 데이터 수집을 안내하는 방법

**특징**
- 분석과 수집 동시 진행
- 이론 개발 목적
- 범주 발전 추구
- 포화까지 지속

**절차**
```
1단계: 초기 표집 (목적적)
       ↓
2단계: 데이터 수집 & 분석
       ↓
3단계: 범주 발견
       ↓
4단계: 범주 속성 탐색 위한 표집 결정
       (누구를? 어디서? 무엇을?)
       ↓
5단계: 추가 수집 & 분석
       ↓
6단계: 포화 도달?
       - 예 → 종료
       - 아니오 → 3단계로
```

## 표본 크기 결정

### 전통별 가이드라인

| 전통 | 권장 크기 | 근거 |
|------|---------|------|
| 현상학 | 5-25명 | 경험의 본질 |
| 근거이론 | 20-30+명 | 이론적 포화 |
| 문화기술지 | 1집단 | 문화 전체 |
| 내러티브 | 1-3명 | 심층 이야기 |
| 사례연구 | 1-5 사례 | 맥락적 심층 |

### 포화 개념

**데이터 포화**
- 새로운 정보 없음
- 반복되는 내용

**이론적 포화**
- 범주가 잘 발달
- 관계가 명확
- 변이 설명 가능

**코드 포화**
- 새 코드 출현 없음
- Guest et al. (2006): 12회 면담 후 92% 코드

### 실용적 고려

**영향 요인**
- 연구 목적
- 현상의 복잡성
- 데이터 질
- 연구자 경험
- 자원 제약

**권장 접근**
- 최소 수 시작
- 포화 모니터링
- 유연하게 조정
- 결정 근거 기록
