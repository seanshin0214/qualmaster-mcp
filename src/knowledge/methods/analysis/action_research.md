# Action Research: 실천과 변화를 위한 연구 마스터 가이드

## 1. Action Research의 본질

### 정의와 철학적 기반
Action Research는 연구자와 실천가가 협력하여 현실 문제를 해결하면서 동시에 지식을 생산하는 참여적 연구 패러다임이다.

```
Action Research 전통:
├── Lewin의 전통 (1940s)
│   ├── 사회 변화 지향
│   ├── 민주적 참여
│   └── 순환적 과정
│
├── Participatory Action Research (PAR)
│   ├── Freire의 비판적 의식화
│   ├── 권력 재분배
│   └── 해방적 목적
│
├── Action Science (Argyris)
│   ├── 조직 학습
│   ├── Single/double-loop learning
│   └── 방어적 루틴 극복
│
└── Appreciative Inquiry (AI)
    ├── 긍정적 탐구
    ├── 강점 기반 접근
    └── 4D 사이클
```

### 핵심 원칙
```
1. 참여 (Participation)
   - 연구 대상이 연구 주체로
   - 공동 의사결정
   - 지식의 민주화

2. 실천 (Action)
   - 이론이 아닌 변화 지향
   - 즉각적 적용
   - 실천적 지식 생산

3. 반성 (Reflection)
   - 체계적 성찰
   - 경험에서 학습
   - 이론 수정

4. 순환 (Iteration)
   - 계획-실행-관찰-반성
   - 지속적 개선
   - 나선형 발전
```

## 2. Action Research 사이클

### Lewin의 고전적 사이클
```
        Planning
           ↓
    ┌──→ Acting ──→ Observing
    │                   ↓
    └──── Reflecting ←─┘
         (새로운 사이클)

각 단계:
1. Planning: 현상 진단, 가설 설정
2. Acting: 개입/행동 실시
3. Observing: 결과 데이터 수집
4. Reflecting: 성찰 및 수정
```

### Kemmis & McTaggart의 나선형 모델
```
사이클 1: 정찰(Reconnaissance)
├── 현장 상황 파악
├── 이해관계자 식별
└── 초기 계획 수립

사이클 2: 1차 실행
├── 계획 실행
├── 관찰 및 기록
└── 성찰 및 수정

사이클 3: 수정된 실행
├── 수정된 계획 실행
├── 심화된 관찰
└── 이론화 시도

사이클 N: 지속적 개선
├── 정착 또는 확산
└── 연구 종료 결정
```

## 3. 주요 유형

### Participatory Action Research (PAR)
```
핵심 특성:
- 연구자-참여자 동등 파트너십
- 사회정의/해방 지향
- 커뮤니티 역량 강화
- 집합적 지식 생산

Freire의 영향:
- Conscientização (의식화)
- Praxis (실천-반성 통합)
- Dialogue (대화적 관계)

적용 분야:
- 지역사회 개발
- 보건교육
- 여성/소수자 역량강화
- 환경정의
```

### Action Science (Argyris & Schön)
```
핵심 개념:

Single-loop Learning:
- 현 체계 내 개선
- 가정 변화 없음
- "How to do it better?"

Double-loop Learning:
- 근본 가정 질문
- 패러다임 변화
- "Why are we doing this?"

Triple-loop Learning:
- 학습 방식 자체 변혁
- 메타인지적 성찰
- "How do we learn?"

방어적 루틴 극복:
- Espoused theory vs Theory-in-use
- Left-hand column 기법
- Ladder of inference
```

### Appreciative Inquiry (AI)
```
4D 사이클:

1. Discovery (발견)
   - 조직의 강점 탐색
   - 최상의 경험 공유
   - "What gives life?"

2. Dream (꿈)
   - 이상적 미래 상상
   - 집합적 비전
   - "What might be?"

3. Design (설계)
   - 실현 가능한 제안
   - 도발적 명제(Provocative Propositions)
   - "What should be?"

4. Destiny/Deliver (실현)
   - 행동 계획
   - 지속적 실행
   - "What will be?"

원칙:
- Constructionist - 사회적 구성
- Simultaneity - 질문=변화 시작
- Poetic - 조직은 열린 텍스트
- Anticipatory - 이미지가 행동 유도
- Positive - 긍정에서 에너지
```

## 4. 데이터 수집과 분석

### 참여적 데이터 수집
```
기법:
├── Community mapping
│   └── 참여자가 직접 자원/문제 지도화
│
├── Photovoice
│   └── 참여자가 사진으로 현실 표현
│
├── Focus groups
│   └── 집단 성찰 및 논의
│
├── Participatory diagramming
│   └── 관계도, 시간선 함께 그리기
│
└── Reflective journals
    └── 참여자의 성찰 일지
```

### 협력적 분석
```
과정:
1. 데이터 공유
   - 모든 참여자에게 데이터 접근권

2. 공동 코딩
   - 함께 주제/패턴 식별

3. 해석 논의
   - 다양한 관점 통합

4. 검증
   - 참여자가 분석 확인

5. 공동 보고
   - 연구 결과 함께 작성
```

## 5. 엄밀성과 품질

### Anderson & Herr (2005)의 타당도 기준
```
1. Outcome Validity (결과 타당도)
   - 행동이 문제 해결에 기여했는가?
   - 새로운 질문/사이클 촉발했는가?

2. Process Validity (과정 타당도)
   - 연구 과정이 충분히 탐구적이었는가?
   - 다중 관점/삼각검증 사용했는가?

3. Democratic Validity (민주적 타당도)
   - 모든 이해관계자 참여했는가?
   - 권력 관계 어떻게 다루었는가?

4. Catalytic Validity (촉매적 타당도)
   - 참여자의 인식 변화 촉진했는가?
   - 행동 동기 증가시켰는가?

5. Dialogic Validity (대화적 타당도)
   - 동료와 비판적 대화했는가?
   - 가정을 검토했는가?
```

### 문서화 전략
```
연구 일지:
□ 각 사이클 상세 기록
□ 의사결정 근거
□ 예상치 못한 결과
□ 참여자 반응
□ 연구자 성찰

증거 수집:
□ 회의록/녹음
□ 사진/비디오
□ 산출물/문서
□ 설문/피드백
□ 외부 관찰자 기록
```

## 6. 저널 출판

### Action Research 전문 저널
```
Action Research (Sage)
- 분야 대표 저널
- 모든 AR 전통 포함
- IF: ~1.8

Educational Action Research
- 교육 분야 특화
- 교사연구자 논문 환영
- 실천 중심

Systemic Practice and Action Research
- 시스템 사고 통합
- 조직 변화 초점

International Journal of Action Research
- 유럽 관점
- 학제간 접근
```

### 주류 저널 출판 전략
```
도전 과제:
- 전통적 "객관성" 기준과 충돌
- 일반화 가능성 질문
- 연구자 역할 혼란

대응 전략:

1. 방법론적 정당화 강화
   "Action research is appropriate because
   the research question requires..."

2. 엄밀성 명시적 논의
   "We ensured process validity through..."

3. 이론적 기여 부각
   "This action research contributes to
   theory by revealing..."

4. 전이가능성 논의
   "The principles developed may inform
   similar contexts where..."
```

### 보고 구조
```
1. Introduction
   - 문제 맥락과 중요성
   - 왜 Action Research인가

2. Theoretical Framework
   - AR 인식론적 위치
   - 관련 실천 이론

3. Research Context
   - 현장 상세 기술
   - 연구자-현장 관계

4. Methodology
   - AR 전통 명시
   - 사이클 설계
   - 참여자 역할
   - 데이터 수집/분석
   - 윤리적 고려

5. The Action Research Process
   - 사이클별 서술
   - 계획-실행-관찰-반성
   - 변화와 학습

6. Findings/Outcomes
   - 실천적 결과
   - 이론적 통찰

7. Discussion
   - 타당도 논의
   - 전이가능성
   - 한계

8. Implications
   - 실천적 함의
   - 이론적 기여
   - 향후 연구/실천
```

## 7. 일반적 도전과 해결책

### 도전 1: 연구자-실천가 이중 역할
```
문제:
- 객관성 vs 참여
- 비판적 거리 유지 어려움

해결책:
- 역할 명확화 및 협상
- Critical friend 활용
- 지속적 성찰 일지
- 다중 관점 통합
```

### 도전 2: 참여자 권력 불균형
```
문제:
- 위계 내 참여 어려움
- 목소리 불균형

해결책:
- 익명 피드백 채널
- 소그룹 논의 활용
- 외부 퍼실리테이터
- 권력 분석 명시적 수행
```

### 도전 3: 시간과 자원
```
문제:
- 여러 사이클 소요
- 참여자 시간 부담

해결책:
- 현실적 범위 설정
- 기존 회의/활동 활용
- 단계적 참여 수준
- 역량 강화를 인센티브로
```
