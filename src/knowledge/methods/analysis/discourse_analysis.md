# 담론 분석 (Discourse Analysis)
## 완전 가이드

---

## 담론 분석이란?

### 정의

```
담론 분석:
"언어 사용이 어떻게 사회적 현실을 구성하고,
 권력 관계를 반영하며, 주체를 형성하는지 분석"

핵심 가정:
├── 언어 = 현실 구성의 도구
├── 담론 = 지식과 권력의 체계
├── 의미 = 맥락 의존적
├── 주체 = 담론에 의해 형성
└── 현실 = 사회적으로 구성됨
```

### 유형

```
1. 담화 분석 (Discourse Analysis - 협의)
├── 언어학적 초점
├── 대화 구조
├── 언어 사용 패턴
└── 상호작용 분석

2. 비판적 담론 분석 (CDA)
├── 권력과 이데올로기
├── 사회적 불평등
├── 지배와 저항
└── Fairclough, Van Dijk

3. 푸코적 담론 분석 (FDA)
├── 담론적 형성
├── 지식-권력
├── 주체화
└── 통치성

4. 담론 심리학
├── 일상 담론
├── 정체성 구성
├── 수사적 전략
└── Potter, Wetherell
```

## 이론적 배경

### Foucault의 담론 이론

```
핵심 개념:

1. 담론적 형성 (Discursive Formation)
├── 진술들의 규칙적 집합
├── 대상, 주체, 개념, 전략
└── 특정 시대의 지식 체계

2. 지식-권력 (Knowledge-Power)
├── 지식은 권력을 수반
├── 권력은 지식을 생산
└── 분리 불가능

3. 주체화 (Subjectification)
├── 담론이 주체를 생산
├── 가능한 위치 제공
└── 자아 테크놀로지

4. 통치성 (Governmentality)
├── 인구 관리
├── 자기 통치
└── 규율과 생명권력
```

### Fairclough의 비판적 담론 분석

```
3차원 모델:

1차원: 텍스트 (Text)
├── 어휘 선택
├── 문법 구조
├── 응집성
└── 텍스트 구조

2차원: 담론 실천 (Discursive Practice)
├── 생산
├── 분배
├── 소비
└── 상호텍스트성

3차원: 사회적 실천 (Social Practice)
├── 이데올로기
├── 헤게모니
├── 권력 관계
└── 사회 변화
```

## 연구 설계

### 연구 질문 유형

```
기술적 질문:
├── "이 담론의 특징은 무엇인가?"
├── "어떤 언어가 사용되는가?"
└── "어떤 패턴이 있는가?"

해석적 질문:
├── "이 담론은 어떤 의미를 구성하는가?"
├── "어떤 주체 위치를 제공하는가?"
└── "어떤 현실을 구성하는가?"

비판적 질문:
├── "이 담론은 누구에게 이익인가?"
├── "어떤 권력 관계를 반영/강화하는가?"
├── "무엇이 배제되는가?"
└── "어떤 불평등을 재생산하는가?"
```

### 데이터 선정

```
텍스트 유형:
├── 공식 문서 (정책, 보고서)
├── 미디어 텍스트 (뉴스, 광고)
├── 조직 문서 (미션, 전략)
├── 인터뷰 전사본
├── 연설/프레젠테이션
├── 소셜미디어
└── 시각 자료

선정 기준:
├── 연구 질문과 관련성
├── 담론의 대표성
├── 시간적 범위
├── 다양한 관점 포함
└── 접근 가능성
```

## 분석 절차

### Fairclough 접근

```
1단계: 텍스트 분석

어휘 분석:
├── 어떤 단어가 사용되는가?
├── 어떤 은유가 있는가?
├── 어떤 범주화가 있는가?
└── 명명(naming) 전략

문법 분석:
├── 행위자는 누구인가?
├── 수동태/능동태 사용
├── 명사화 (Nominalization)
└── 양태 (Modality)

텍스트 구조:
├── 장르 관습
├── 논증 구조
├── 정보 흐름
└── 응집 장치

2단계: 담론 실천 분석

생산:
├── 누가 생산했는가?
├── 어떤 조건에서?
├── 어떤 자원을 사용?
└── 어떤 제약 하에?

분배:
├── 어떻게 유통되는가?
├── 누가 접근하는가?
├── 어떤 채널을 통해?
└── 얼마나 널리?

소비:
├── 누가 읽는가?
├── 어떻게 해석되는가?
├── 어떤 반응이?
└── 어떤 효과가?

상호텍스트성:
├── 다른 텍스트 참조
├── 담론 혼합
├── 장르 혼합
└── 담론 질서

3단계: 사회적 실천 분석

이데올로기:
├── 어떤 가정이 자연화?
├── 어떤 가치가 촉진?
├── 어떤 이익이 은폐?
└── 상식화된 것은?

헤게모니:
├── 어떤 담론이 지배적?
├── 경쟁 담론은?
├── 동의는 어떻게?
└── 저항은?
```

### 푸코적 접근

```
분석 초점:

1. 담론적 대상 (Objects)
├── 무엇이 담론의 대상인가?
├── 대상은 어떻게 구성되는가?
├── 대상 간 관계는?
└── 시간에 따른 변화는?

2. 발화 양식 (Enunciative Modalities)
├── 누가 말할 수 있는가?
├── 어디서 말하는가?
├── 어떤 자격으로?
└── 어떤 권위로?

3. 개념 (Concepts)
├── 어떤 개념이 사용되는가?
├── 개념 간 관계는?
├── 개념의 규칙은?
└── 변형 패턴은?

4. 주제/전략 (Themes/Strategies)
├── 어떤 주제가 다뤄지는가?
├── 전략적 선택은?
├── 이론적 일관성은?
└── 배제된 것은?
```

## 구체적 분석 기법

### 어휘 분석

```
1. 키워드 분석
├── 빈도 분석
├── 핵심어 식별
├── 연어 (Collocation)
└── 의미장

2. 은유 분석
├── 개념적 은유
├── 은유 클러스터
├── 은유의 함의
└── 대안 은유

3. 명명 분석
├── 어떻게 명명하는가
├── 범주화 방식
├── 포함/배제
└── 정체성 구성
```

### 주체 위치 분석

```
분석 질문:
├── 어떤 주체 위치가 제공되는가?
├── 누가 행위자로 구성되는가?
├── 누가 대상으로 구성되는가?
├── 어떤 정체성이 가능한가?
└── 어떤 관계가 설정되는가?

예시:
담론: "직원들은 고객 만족을 위해 헌신해야 한다"
├── 직원 = 헌신하는 주체
├── 고객 = 만족받아야 할 대상
├── 관계 = 봉사 관계
└── 대안 배제 = 직원 권리
```

### 상호텍스트성 분석

```
유형:
├── 명시적 인용
├── 암묵적 참조
├── 장르 혼합
├── 담론 혼합
└── 목소리 다중성

분석:
├── 어떤 텍스트가 참조되는가?
├── 어떻게 재맥락화되는가?
├── 어떤 효과를 내는가?
└── 어떤 권위를 빌려오는가?
```

## 조직/경영 담론 분석

### 적용 영역

```
1. 전략 담론
├── 전략 텍스트 분석
├── 전략 수사
├── 정당화 전략
└── 권력 관계

2. 리더십 담론
├── 리더 정체성 구성
├── 카리스마 구성
├── 권위 정당화
└── 팔로워 위치

3. 조직 변화 담론
├── 변화 정당화
├── 저항 담론
├── 정체성 재구성
└── 의미 투쟁

4. HRM 담론
├── 인적 자원 구성
├── 성과 담론
├── 개발 담론
└── 주체화
```

### 분석 예시

```
텍스트: "우리는 인재가 가장 중요한 자산입니다"

분석:

1. 어휘
├── "인재" - 차별화, 선별
├── "자산" - 경제적 가치화
├── "가장 중요한" - 수사적 강조
└── "우리" - 집단 정체성

2. 이데올로기
├── 인간의 자본화
├── 조직 관점 지배
├── 도구적 관계
└── 불평등 자연화

3. 주체 위치
├── 인재 = 가치 있는 자원
├── 비인재 = 암묵적 배제
├── 조직 = 인재 활용 주체
└── 대안 담론 배제

4. 효과
├── 경쟁 정당화
├── 차별 자연화
├── 관계의 상품화
└── 비판 봉쇄
```

## 글쓰기

### 담론 분석 논문 구조

```
1. Introduction
├── 연구 맥락
├── 연구 질문
├── 담론 분석의 정당성
└── 기여

2. Theoretical Framework
├── 담론 이론
├── 분석 접근
├── 핵심 개념
└── 분석적 렌즈

3. Methodology
├── 데이터 선정
├── 분석 절차
├── 품질 기준
└── 연구자 위치성

4. Analysis
├── 담론 특성
├── 주제/패턴
├── 권력/이데올로기
└── 예시와 해석

5. Discussion
├── 이론적 함의
├── 실무적 함의
├── 비판적 반성
└── 한계와 향후 연구
```
