# 면담 유형 (Interview Types)

## 구조화 정도에 따른 분류

### 1. 비구조화 면담 (Unstructured Interview)

#### 특징
- 최소한의 구조
- 참여자 주도
- 탐색적 목적
- 대화형

#### 적합한 상황
- 현상학적 연구
- 초기 탐색 단계
- 미지의 영역
- 참여자 관점 우선

#### 시작 질문 예시
```
"~에 대한 경험을 이야기해 주시겠어요?"
"~에 대해 어떻게 생각하세요?"
```

#### 장단점
| 장점 | 단점 |
|------|------|
| 풍부한 데이터 | 분석 복잡 |
| 참여자 관점 존중 | 비교 어려움 |
| 유연성 | 시간 소요 |
| 예상치 못한 발견 | 연구자 기술 의존 |

### 2. 반구조화 면담 (Semi-structured Interview)

#### 특징
- 면담 가이드 사용
- 핵심 질문 + 유연한 탐색
- 가장 많이 사용
- 비교와 깊이 균형

#### 적합한 상황
- 대부분의 질적연구
- 특정 주제 탐구
- 다수 참여자 비교
- 이론 기반 연구

#### 면담 가이드 구조
```
1. 도입부 (5-10분)
   - 라포 형성
   - 연구 설명
   - 동의 확인

2. 도입 질문 (Ice-breaker)
   - 쉬운 질문으로 시작
   - 배경 정보 수집

3. 전환 질문 (Transition)
   - 핵심 주제로 연결
   - 맥락 설정

4. 핵심 질문 (Key Questions)
   - 연구 질문 관련 핵심
   - 탐색 질문 (Probes) 준비
   - 5-10개 핵심 질문

5. 마무리 질문 (Closing)
   - 추가할 내용 확인
   - 경험 요약 기회
   - 감사 표현
```

### 3. 구조화 면담 (Structured Interview)

#### 특징
- 표준화된 질문
- 정해진 순서
- 동일한 방식
- 양적 분석 가능

#### 적합한 상황
- 대규모 조사
- 양적-질적 혼합
- 응답 비교 필요
- 신뢰도 강조

#### 장단점
| 장점 | 단점 |
|------|------|
| 비교 용이 | 유연성 제한 |
| 분석 효율 | 깊이 제한 |
| 훈련 용이 | 새로운 주제 놓침 |
| 신뢰도 높음 | 참여자 주도 어려움 |

## 형식에 따른 분류

### 개인 면담 (Individual Interview)

#### 특징
- 1:1 심층 대화
- 프라이버시 보장
- 개인 경험 집중

#### 적합한 상황
- 민감한 주제
- 개인적 경험
- 심층적 탐구

### 포커스 그룹 (Focus Group)

#### 특징
- 6-10명 그룹
- 상호작용 활용
- 집단 역학

#### 적합한 상황
- 공유된 경험
- 다양한 관점 수집
- 상호작용 관찰

#### 진행 요령
- 모더레이터 역할
- 발언 균형
- 지배적 참여자 관리

### 다이아드/트라이아드 (Dyad/Triad)

#### 특징
- 2-3명 소그룹
- 관계적 역학
- 친밀한 대화

#### 적합한 상황
- 부부, 친구, 동료
- 관계적 경험
- 상호 보완

## 매체에 따른 분류

### 대면 면담 (Face-to-face)

**장점**
- 비언어적 단서
- 라포 형성 용이
- 환경 관찰

**단점**
- 시간/비용
- 지리적 제약
- 일정 조율

### 전화 면담 (Telephone)

**장점**
- 지리적 접근성
- 비용 효율
- 익명성 제공

**단점**
- 비언어적 단서 부재
- 라포 형성 어려움
- 집중 시간 제한

### 화상 면담 (Video)

**장점**
- 대면의 장점 + 원격
- 녹화 용이
- 화면 공유 가능

**단점**
- 기술적 문제
- 디지털 피로
- 환경 통제 어려움

### 비동기 면담 (Asynchronous)

**형태**
- 이메일 면담
- 온라인 게시판
- 메신저

**장점**
- 참여자 편의
- 성찰 시간
- 기록 자동화

**단점**
- 즉각적 탐색 불가
- 중도 포기
- 맥락 손실

## 특수 면담 유형

### 엘리트 면담 (Elite Interview)
- 고위직, 전문가
- 접근 어려움
- 권력 역학

### 아동 면담
- 발달 수준 고려
- 윤리적 고려
- 창의적 기법

### 취약 집단 면담
- 특별한 윤리적 배려
- 권력 불균형 인식
- 지원 체계 연결

## 면담 기술

### 적극적 경청
- 눈 맞춤
- 고개 끄덕임
- 언어적 인정 ("음", "네")

### 탐색 질문 (Probing)
```
- "좀 더 자세히 말씀해 주시겠어요?"
- "예를 들어주시겠어요?"
- "그때 어떤 느낌이셨나요?"
- "그건 무슨 의미인가요?"
```

### 침묵 활용
- 생각할 시간 제공
- 불편함 견디기
- 깊은 반성 유도

### 요약과 반영
```
"지금까지 말씀하신 것을 정리하면..."
"~라고 느끼셨다는 거죠?"
```

## 피해야 할 것

### 질문 유형
- 유도 질문: "그게 힘들었죠?"
- 이중 질문: "A와 B는 어떠세요?"
- 전문 용어: 참여자 언어 사용

### 행동
- 판단하기
- 조언하기
- 말 끊기
- 자기 이야기 과다
