# 신뢰성 (Trustworthiness)

## Lincoln & Guba (1985) 기준

### 개요
실증주의의 타당도, 신뢰도, 객관성을 대체하는 질적연구 품질 기준

## 4가지 핵심 기준

### 1. 신빙성 (Credibility)

#### 정의
연구 결과가 참여자의 관점과 경험을 정확하게 반영하는 정도

#### 확보 전략

**장기간 관여 (Prolonged Engagement)**
- 충분한 시간 투자
- 맥락과 문화 이해
- 신뢰 관계 구축
- 왜곡 최소화

**지속적 관찰 (Persistent Observation)**
- 심층적 탐구
- 핵심 요소 식별
- 세부 사항 주목

**삼각화 (Triangulation)**
- 데이터 삼각화: 다양한 출처
- 연구자 삼각화: 다수 연구자
- 이론적 삼각화: 다양한 관점
- 방법론적 삼각화: 다양한 방법

**동료 브리핑 (Peer Debriefing)**
- 비이해당사자와 논의
- 분석 과정 검토
- 가정과 편견 도전
- 다음 단계 탐색

**참여자 확인 (Member Checking)**
- 참여자에게 해석 확인
- 데이터, 범주, 해석 검토
- 수정 기회 제공
- 가장 중요한 신빙성 확보 방법

**부정적 사례 분석 (Negative Case Analysis)**
- 패턴에 맞지 않는 사례 탐색
- 결론 수정 또는 정교화
- 가설 지속적 검토

### 2. 전이가능성 (Transferability)

#### 정의
연구 결과가 다른 맥락이나 집단에 적용될 수 있는 정도

#### 확보 전략

**풍부한 기술 (Thick Description)**
- 맥락의 상세한 기술
- 문화적, 사회적 배경
- 참여자 특성
- 연구 환경

**목적적 표집의 명확화**
- 표집 기준과 근거
- 참여자 선정 과정
- 이론적 다양성 추구

#### 연구자의 역할
- 풍부한 데이터베이스 제공
- 전이 판단은 독자의 몫
- 맥락 비교 가능하도록 정보 제공

### 3. 의존가능성 (Dependability)

#### 정의
연구 과정이 일관되고 추적 가능하며, 다른 연구자가 따라갈 수 있는 정도

#### 확보 전략

**감사 추적 (Audit Trail)**
기록해야 할 것:
- 원자료 (녹음, 전사본)
- 데이터 축소와 분석 산물 (현장노트, 메모)
- 데이터 재구성 산물 (주제, 범주)
- 과정 노트 (절차, 설계 결정)
- 연구자 의도와 기대 기록
- 도구 개발 정보

**외부 감사 (External Audit)**
- 독립적인 연구자의 검토
- 과정과 산물 평가
- 결론의 지지 여부 확인

**탐구적 감사 (Inquiry Audit)**
- 과정의 적절성 평가
- 결론의 데이터 지지 여부

### 4. 확인가능성 (Confirmability)

#### 정의
연구 결과가 연구자의 편견이 아닌 데이터에 근거함을 보여주는 정도

#### 확보 전략

**감사 추적**
- 원자료에서 결론까지 추적 가능
- 해석의 근거 제시
- 분석적 논리 기록

**삼각화**
- 다양한 방법과 출처로 확인
- 연구자 편견 감소

**반성적 저널 (Reflexive Journal)**
- 연구자 가정과 가치 기록
- 결정 과정 기록
- 편견 인식과 관리

## 추가 기준

### 진정성 (Authenticity) - Guba & Lincoln (1989)

**공정성 (Fairness)**
- 모든 관점의 동등한 표현
- 주변화된 목소리 포함

**존재론적 진정성 (Ontological Authenticity)**
- 참여자의 현실 이해 향상

**교육적 진정성 (Educative Authenticity)**
- 타자의 관점 이해 증진

**촉매적 진정성 (Catalytic Authenticity)**
- 행동 촉진

**전술적 진정성 (Tactical Authenticity)**
- 행동 역량 강화

## Tracy (2010) 8가지 기준

### Big-Tent 접근
1. **가치 있는 주제 (Worthy Topic)**
2. **풍부한 엄밀성 (Rich Rigor)**
3. **진정성 (Sincerity)**
4. **신빙성 (Credibility)**
5. **공명성 (Resonance)**
6. **의미 있는 기여 (Significant Contribution)**
7. **윤리 (Ethics)**
8. **의미 있는 일관성 (Meaningful Coherence)**

## 실천적 체크리스트

### 신빙성 (Credibility) ✓
- [ ] 장기간 현장 관여
- [ ] 삼각화 적용
- [ ] 참여자 확인 수행
- [ ] 동료 브리핑 실시
- [ ] 부정적 사례 분석

### 전이가능성 (Transferability) ✓
- [ ] 풍부한 맥락 기술
- [ ] 참여자 특성 상세 기술
- [ ] 표집 전략 명시

### 의존가능성 (Dependability) ✓
- [ ] 감사 추적 유지
- [ ] 연구 절차 상세 기록
- [ ] 외부 검토 수행

### 확인가능성 (Confirmability) ✓
- [ ] 데이터에서 결론 추적 가능
- [ ] 반성적 저널 작성
- [ ] 연구자 가정 명시

## 주의사항

### 과잉 적용 피하기
- 모든 전략을 다 사용할 필요 없음
- 연구 유형과 목적에 맞게 선택
- 실행 가능성 고려

### 기계적 적용 경계
- 체크리스트 채우기가 목표 아님
- 연구의 본질적 품질 추구
- 유연한 적용
