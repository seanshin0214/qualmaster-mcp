# 고전적 문화기술지 (Classical Ethnography)

## 개요

### 정의
문화기술지는 문화 공유 집단(culture-sharing group)을 장기간 현장 몰입을 통해 연구하는 질적연구 전통입니다.

### 기원
- 인류학적 뿌리: Malinowski, Boas
- 시카고 학파 사회학: Park, Thomas
- 1920년대~ 발전

## 핵심 특징

### 문화 (Culture) 개념
- 공유된 패턴: 행동, 언어, 신념
- 학습된 것: 세대 간 전승
- 상징 체계: 의미의 망

### Geertz의 두꺼운 기술 (Thick Description)
```
얇은 기술: "그는 눈을 깜빡였다"
두꺼운 기술: "그는 공모의 신호로 윙크했다. 이 집단에서
             윙크는 비밀 공유를 의미하며..."
```

## 연구 절차

### 1. 현장 선정

**기준**
- 접근 가능성
- 문화적 경계
- 연구 질문 적합성

**진입 전략**
- 게이트키퍼 확보
- 신뢰 구축
- 역할 협상

### 2. 참여관찰

#### 참여 수준
| 수준 | 설명 | 장점 | 단점 |
|------|------|------|------|
| 완전 참여자 | 구성원으로 활동 | 내부자 시각 | 객관성 위험 |
| 참여적 관찰자 | 참여하며 관찰 | 균형 | 역할 갈등 |
| 관찰적 참여자 | 관찰 중심 | 거리 유지 | 표면적 이해 |
| 완전 관찰자 | 순수 관찰 | 최소 개입 | 내부자 시각 부재 |

#### 관찰 기록

**현장노트 구조**
1. 기술적 노트: 객관적 관찰
2. 반성적 노트: 연구자 생각, 감정
3. 방법론적 노트: 연구 결정
4. 이론적 노트: 분석적 아이디어

### 3. 면담

**유형**
- 비공식 대화: 일상적 상호작용
- 반구조화 면담: 주제 가이드
- 생애사 면담: 개인 역사

**핵심 정보제공자 (Key Informant)**
- 문화 지식 풍부
- 표현력 있음
- 협조적

### 4. 문서/인공물 분석

- 공식 문서
- 사진, 영상
- 물질문화
- 소셜미디어

## 분석

### Wolcott (1994) 3단계

1. **기술 (Description)**
   - 무슨 일이 일어나는가?
   - 맥락 상세 기술

2. **분석 (Analysis)**
   - 어떻게 작동하는가?
   - 패턴, 관계 규명

3. **해석 (Interpretation)**
   - 무슨 의미인가?
   - 이론적 설명

### 문화적 주제 분석

**Spradley의 도메인 분석**
- 포함 관계: X는 Y의 한 종류
- 인과 관계: X는 Y의 결과
- 수단-목적: X는 Y를 위한 방법

## 민족지적 글쓰기

### 구조
1. 현장과 사람 소개
2. 일상 기술
3. 문화적 주제 분석
4. 이론적 해석
5. 연구자 반성

### 재현의 위기 (Crisis of Representation)
- 누구의 목소리?
- 권력 관계
- 타자화 문제

## Emic vs Etic

| Emic (내부자 관점) | Etic (외부자 관점) |
|------------------|------------------|
| 참여자의 범주 | 연구자의 범주 |
| 토착 개념 | 분석적 개념 |
| 문화 특수적 | 비교 가능 |

**좋은 문화기술지**
- Emic에서 출발
- Etic으로 번역
- 둘의 대화

## 현장 시간

### 일반적 기간
- 최소 6개월
- 전통적: 1-2년
- 현대: 다양 (focused ethnography)

### 현장 떠나기
- 이론적 포화
- 자원 한계
- 관계 고려

## 윤리적 고려

### 핵심 이슈
- 정보 제공 동의
- 익명성과 기밀
- 호혜성
- 문화적 민감성

### 현장 관계
- 보고(Rapport) 구축
- 경계 유지
- 떠남 관리
