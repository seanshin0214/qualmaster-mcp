# Yin의 사례연구 (Yin's Case Study)

## Robert K. Yin 접근

### 개요
Yin(2018)의 사례연구는 현실 맥락에서 현상을 심층적으로 탐구하는 실증적 연구 방법입니다.

### 정의
"사례연구는 특히 현상과 맥락 사이의 경계가 명확하지 않을 때, 현실 맥락 내에서 현대 현상을 심층적으로 탐구하는 실증적 연구이다" (Yin, 2018, p.15)

## 핵심 특징

### 적합한 상황
1. 연구 질문이 "어떻게" 또는 "왜"
2. 연구자가 사건 통제 불가
3. 현재 또는 최근 현상에 초점
4. 맥락이 중요

### 사례의 경계
- 시간적 경계
- 공간적 경계
- 분석 단위 정의

## 사례연구 유형

### 목적에 따른 분류

| 유형 | 목적 | 예시 |
|------|------|------|
| 탐색적 | 가설 개발 | 새로운 현상 이해 |
| 기술적 | 현상 기술 | 프로그램 평가 |
| 설명적 | 인과 규명 | 정책 효과 분석 |

### 사례 수에 따른 분류

**단일 사례 설계**
- 결정적 사례: 이론 검증
- 극단적/독특한 사례: 희귀 현상
- 대표적 사례: 전형적 상황
- 폭로적 사례: 새로운 접근 기회
- 종단적 사례: 시간에 따른 변화

**다중 사례 설계**
- 복제 논리 (Replication Logic)
- 문자적 복제: 유사한 결과 예측
- 이론적 복제: 대조적 결과 예측

## 연구 설계

### 5가지 구성요소

1. **연구 질문**: 어떻게/왜
2. **명제 (Propositions)**: 연구 방향
3. **분석 단위**: 사례 정의
4. **논리적 연결**: 데이터와 명제
5. **해석 기준**: 결과 평가

### 설계 유형 (2x2)

| | 단일 사례 | 다중 사례 |
|---|---------|---------|
| **총체적 (단일 분석단위)** | Type 1 | Type 3 |
| **내포적 (복수 분석단위)** | Type 2 | Type 4 |

## 데이터 수집

### 6가지 증거 출처

1. **문서**: 편지, 메모, 보고서
2. **기록물**: 서비스 기록, 조직도
3. **면담**: 개방형, 초점화, 조사형
4. **직접 관찰**: 현장 방문
5. **참여 관찰**: 역할 수행
6. **물리적 인공물**: 도구, 장치

### 데이터 수집 원칙

**원칙 1: 다중 출처 활용**
- 삼각화를 통한 타당도 향상
- 증거 수렴

**원칙 2: 사례연구 데이터베이스 구축**
- 증거 조직
- 감사 추적
- 신뢰도 향상

**원칙 3: 증거 연쇄 유지**
- 보고서 → 데이터베이스 → 원자료
- 추적 가능성

**원칙 4: 전자적 출처 주의**
- 소셜미디어, 온라인 데이터
- 신뢰성 확인

## 데이터 분석

### 4가지 전략

1. **이론적 명제 의존**
   - 명제 기반 분석
   - 연역적 접근

2. **사례 기술 개발**
   - 기술적 프레임워크
   - 귀납적 구성

3. **양적-질적 데이터 활용**
   - 혼합 접근
   - 패턴 확인

4. **경쟁 설명 검토**
   - 대안적 해석
   - 반박 증거 탐색

### 5가지 분석 기법

1. **패턴 매칭**: 예측 패턴과 실제 비교
2. **설명 구축**: 인과적 연결 개발
3. **시계열 분석**: 시간에 따른 변화
4. **논리 모델**: 인과 사슬 추적
5. **교차 사례 종합**: 다중 사례 통합

## 품질 기준

### 4가지 검증

| 검증 | 전술 | 단계 |
|------|------|------|
| 구성 타당도 | 다중 출처, 증거 연쇄 | 수집 |
| 내적 타당도 | 패턴 매칭, 설명 구축 | 분석 |
| 외적 타당도 | 복제 논리 | 설계 |
| 신뢰도 | 프로토콜, 데이터베이스 | 수집 |

## 사례연구 보고서

### 구조 유형

1. **선형-분석적**: 전통적 학술 구조
2. **비교적**: 같은 사례 다중 관점
3. **연대기적**: 시간순
4. **이론-구축**: 이론 전개 중심
5. **비순차적**: 비선형 탐색

### 보고서 작성 원칙
- 증거 제시: 충분한 인용
- 경쟁 해석 고려
- 유의미한 측면 표시
- 독자 참여 유도
