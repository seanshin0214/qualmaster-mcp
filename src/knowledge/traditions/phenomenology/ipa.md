# 해석적 현상학적 분석 (IPA)
## Interpretative Phenomenological Analysis

## 개요

IPA는 Jonathan Smith가 1990년대 중반 개발한 질적연구 접근법으로, 개인이 특정 경험을 어떻게 의미화하는지 탐구합니다.

### 이론적 기반
1. **현상학**: 경험의 상세한 검토
2. **해석학**: 의미의 해석
3. **개별기술학(Idiography)**: 개인 사례의 깊이 있는 분석

## 핵심 특징

### 삼중 해석학 (Double Hermeneutic)
```
참여자 → 자신의 경험을 해석
     ↓
연구자 → 참여자의 해석을 해석
     ↓
독자 → 연구자의 해석을 해석
```

연구자는 참여자가 자신의 경험을 이해하려고 하는 것을 이해하려고 합니다.

### 개별기술적 접근
- 각 참여자를 개별 사례로 심층 분석
- 사례 간 패턴 탐색
- 수렴과 발산 모두 보고

## 연구 설계

### 적합한 연구 질문
> "X를 경험한 사람들에게 그것은 어떤 의미인가?"

예시:
- "만성 통증과 함께 사는 경험은 어떠한가?"
- "첫 아이를 갖는 것은 부부에게 어떤 의미인가?"
- "승진을 거부당한 경험은 어떠한가?"

### 표본 추출
| 항목 | 권장 |
|------|------|
| 표본 크기 | 3-6명 (학위논문), 1-3명 (저널) |
| 표본 전략 | 목적적, 동질적 |
| 동질성 | 특정 경험 공유 집단 |

### 데이터 수집
- **반구조화 면담** (1-2시간)
- 개방적 질문으로 시작
- 탐색적 후속 질문
- 참여자 주도 허용

## 분석 절차 (Smith et al., 2009)

### 1단계: 읽기와 재읽기
```
- 전사본 여러 번 읽기
- 녹음 다시 듣기
- 전체적 인상 기록
- 선입견 괄호치기
```

### 2단계: 초기 노팅
```
세 가지 유형의 코멘트:
1. 기술적 (Descriptive): 내용 기술
2. 언어적 (Linguistic): 언어 사용 주목
3. 개념적 (Conceptual): 해석적 질문
```

### 3단계: 발현 주제 개발
- 초기 노트를 주제로 변환
- 참여자 언어에서 연구자 언어로
- 심리학적/현상학적 개념화

### 4단계: 주제 간 연결
- 추상화 (Abstraction)
- 포섭 (Subsumption)
- 맥락화 (Contextualization)
- 기능 (Function)

### 5단계: 다음 사례로 이동
- 각 사례를 독립적으로 분석
- 이전 분석에서 괄호치기
- 새로운 주제 허용

### 6단계: 사례 간 패턴 탐색
- 수렴 (Convergence) 확인
- 발산 (Divergence) 확인
- 상위 주제 구성

## 결과 제시

### 주제 표 예시
```
상위 주제 1: [주제명]
├── 하위 주제 1.1: [하위주제명]
│   ├── 참여자 1: 페이지/줄
│   ├── 참여자 2: 페이지/줄
│   └── 참여자 3: 페이지/줄
└── 하위 주제 1.2: [하위주제명]
    └── ...
```

### 서술 형식
1. 주제 소개
2. 참여자 인용문 (근거)
3. 연구자 해석 (분석)
4. 수렴/발산 논의

## 품질 기준 (Yardley, 2000)

### 맥락 민감성
- 관련 문헌 검토
- 참여자 관점 존중
- 사회문화적 맥락 고려

### 헌신과 엄격성
- 적절한 표본
- 면담의 질
- 분석의 깊이

### 투명성과 일관성
- 연구 과정 상세 기술
- 반성적 실천
- 일관된 논증

### 영향과 중요성
- 이론적/실무적 함의
- 독자에게의 의미

## 흔한 실수

### 피해야 할 것
❌ 너무 많은 참여자 (10명 이상)
❌ 피상적 분석
❌ 인용문 나열만
❌ 해석 없는 기술
❌ 이질적 표본

### 지향해야 할 것
✅ 소수 참여자 심층 분석
✅ 적극적 해석
✅ 인용문 + 분석 통합
✅ 경험의 의미 탐구
✅ 동질적 표본

## IPA 논문 예시 구조

```
1. Introduction
   - 현상 소개
   - 기존 연구 검토
   - 연구 필요성

2. Method
   - 참여자 정보
   - 데이터 수집 절차
   - 분석 과정

3. Results
   - 주제 표
   - 주제별 상세 기술
   - 인용문 + 해석

4. Discussion
   - 기존 문헌과 연결
   - 함의
   - 한계
   - 향후 연구
```
