# 구성주의 근거이론 (Constructivist Grounded Theory)
## Kathy Charmaz 접근

## 개요

Kathy Charmaz가 개발한 근거이론의 현대적 해석으로, 전통적 GT의 방법을 유지하면서 구성주의 인식론을 통합합니다.

## 고전 GT vs 구성주의 GT

| 측면 | Glaser/Strauss | Charmaz |
|------|---------------|---------|
| 인식론 | 객관주의/실증주의 | 구성주의 |
| 이론 | 발견 (Discover) | 구성 (Construct) |
| 연구자 | 중립적 관찰자 | 공동 구성자 |
| 데이터 | 객관적 사실 | 구성된 산물 |
| 범주 | 현실 반영 | 해석적 구성 |

## 핵심 원리

### 1. 상호작용적 데이터 구성
```
데이터는 연구자-참여자 상호작용의 산물
├── 면담은 공동 구성의 장
├── 연구자의 해석이 데이터에 반영
└── 맥락이 의미 형성에 영향
```

### 2. 연구자 반성성
- 자기 위치성 인식
- 가정과 편견 검토
- 해석 과정 투명화

### 3. 다중 현실과 관점
- 여러 관점의 공존 인정
- 발산적 경험 포용
- 복잡성 수용

## 코딩 절차

### 1단계: 초기 코딩 (Initial Coding)

#### 줄단위 코딩 (Line-by-Line)
```
데이터: "처음에는 무서웠지만, 점점 익숙해졌어요"

줄단위 코드:
- 초기 두려움 경험하기
- 적응 과정 거치기
- 익숙함 획득하기
```

#### 게런드(-ing) 코딩
동사 형태 사용으로 과정 포착
- Experiencing fear → 두려움 경험하기
- Adapting gradually → 점진적 적응하기
- Becoming familiar → 익숙해지기

### 2단계: 집중 코딩 (Focused Coding)

가장 빈번하고 의미 있는 초기 코드 선택하여 분석적 방향 설정

```
초기 코드들:
- 불확실성 마주하기
- 두려움 느끼기
- 정보 찾기
- 지지 구하기
- 점진적 익숙해지기
- 자신감 획득하기

집중 코드:
→ "불확실성 속에서 적응하기"
```

### 3단계: 이론적 코딩 (Theoretical Coding)

집중 코드 간 관계를 개념화

```
이론적 코드:
- 선행 조건 (Antecedent)
- 맥락 (Context)
- 조건 (Condition)
- 상호작용 (Interaction)
- 전략 (Strategy)
- 결과 (Consequence)
```

## 메모 작성 (Memoing)

### 메모의 유형
1. **코드 메모**: 코드의 의미와 속성
2. **이론적 메모**: 범주 간 관계
3. **과정 메모**: 연구 결정과 방향
4. **반성적 메모**: 연구자 성찰

### 메모 예시
```markdown
## 메모: "불확실성 관리하기" (2024-01-15)

### 코드 정의
불확실한 상황에서 참여자들이 사용하는 인지적, 행동적 전략

### 속성
- 능동적 vs 수동적
- 개인적 vs 사회적
- 단기적 vs 장기적

### 관련 코드
- 정보 탐색하기
- 지지 구하기
- 의미 재구성하기

### 이론적 질문
- 불확실성 유형에 따라 전략이 다른가?
- 시간에 따라 전략이 변화하는가?
- 성공적 관리의 조건은 무엇인가?

### 다음 단계
- 이론적 표본추출: 다른 불확실성 상황 경험자
- 속성 차원 더 탐색
```

## 이론적 표본추출

### 원칙
```
초기 표본 → 분석 → 개념 발현 →
이론적 질문 → 표본추출 방향 →
추가 데이터 → 범주 포화
```

### 질문 예시
- 이 범주를 더 발전시키려면 누구와 면담해야 하는가?
- 어떤 상황/맥락이 부족한가?
- 부정적 사례는 어디서 찾을 수 있는가?

## 이론적 포화

### Charmaz의 포화 기준
1. 새로운 속성이 나타나지 않음
2. 범주 간 관계가 명확해짐
3. 범주가 잘 발달됨
4. 변이가 설명됨

### 포화 평가 체크리스트
- [ ] 핵심 범주가 다른 범주와 연결되는가?
- [ ] 범주의 속성과 차원이 충분한가?
- [ ] 과정이 명확히 드러나는가?
- [ ] 변이가 이론에 포함되는가?

## 결과 제시

### 범주와 속성 표
```
핵심 범주: 불확실성 속에서 적응하기

하위 범주 1: 위협 인식하기
├── 속성: 강도 (낮음-높음)
├── 속성: 영역 (건강, 관계, 직업)
└── 속성: 지속성 (일시적-지속적)

하위 범주 2: 자원 동원하기
├── 속성: 유형 (정보, 정서, 도구)
├── 속성: 출처 (내부, 외부)
└── 속성: 접근성 (용이-어려움)

하위 범주 3: 의미 재구성하기
├── 속성: 방식 (수용, 재해석, 성장)
└── 속성: 시점 (초기, 중기, 후기)
```

### 이론적 모델 제시
```
[선행조건] → [핵심과정] → [결과]
     ↑              ↓
   [맥락]      [전략/상호작용]
```

## 글쓰기

### Charmaz 스타일
- 연구자의 목소리 포함
- 반성성 표현
- 구성 과정 투명화
- 참여자 목소리 존중

### 예시 문장
> "참여자들과의 대화를 통해 나는 '불확실성 관리하기'라는 과정을 구성하게 되었다. 이 해석은 내 학문적 배경과 참여자들의 경험이 만나 공동으로 만들어진 것이다."

## 품질 기준

### Charmaz의 4기준 (2006)
1. **신빙성 (Credibility)**
   - 데이터와 범주의 긴밀한 연결
   - 범위와 깊이

2. **독창성 (Originality)**
   - 새로운 통찰
   - 이론적 의의

3. **공명 (Resonance)**
   - 참여자 경험 반영
   - 독자에게 의미

4. **유용성 (Usefulness)**
   - 실무적 적용
   - 향후 연구 방향
