## 동기 연구와 질적연구

### 양적연구 한계

```
한계점:
├── 동기의 복잡성 포착 어려움
├── 맥락 효과 무시
├── 과정 (Process) 포착 한계
├── 개인 의미 손실
├── 자기보고 편향
└── 무의식적 동기 측정 한계

질적연구의 가치:
├── 동기의 주관적 경험
├── 동기 형성 과정
├── 맥락적 요인 이해
├── 새로운 동기 요인 발견
├── 동기 간 상호작용
└── 시간에 따른 변화
```

### 주요 연구 주제

```
1. 내재적/외재적 동기
├── 내재적 동기의 경험
├── 외재적 보상의 영향
├── 내면화 과정
└── 크라우딩 아웃 효과

2. 의미와 목적
├── 일의 의미 찾기
├── 소명 (Calling)
├── 목적 지향
└── 가치 정렬

3. 자기결정
├── 자율성 경험
├── 유능감 발달
├── 관계성 형성
└── 욕구 좌절

4. 목표와 열정
├── 목표 설정 과정
├── 열정의 형성
├── 몰입 경험
└── 목표 추구 전략

5. 동기 갈등
├── 다중 목표
├── 접근-회피
├── 내적 갈등
└── 우선순위 협상
```

## 주요 이론적 프레임워크

### 1. 자기결정이론 (Self-Determination Theory)

```
핵심 개념:
├── 자율성 (Autonomy)
├── 유능감 (Competence)
├── 관계성 (Relatedness)
└── 내면화 연속선

질적연구 질문:
├── 자율성은 어떻게 경험되는가?
├── 유능감 발달 과정은?
├── 관계성이 동기에 미치는 영향은?
├── 내면화는 어떻게 일어나는가?
└── 욕구 좌절의 경험은?
```

### 2. 소명 이론 (Calling Theory)

```
핵심 개념:
├── 초월적 부름
├── 목적과 의미
├── 친사회적 지향
└── 소명 인식 vs 실현

질적연구 가치:
├── 소명 인식 과정
├── 소명 실현 장벽
├── 소명과 행복
├── 맥락적 차이
└── 문화적 변이
```

### 3. 열정 모델 (Passion Model - Vallerand)

```
두 유형:
├── 조화적 열정 (Harmonious)
├── 강박적 열정 (Obsessive)

질적연구 주제:
├── 열정 발달 과정
├── 열정 유형 경험 차이
├── 열정의 결과
├── 열정 유지/감소
└── 맥락 효과
```

### 4. 의미 있는 일 (Meaningful Work)

```
차원:
├── 긍정적 의미
├── 목적 지향
├── 자기 일치
└── 기여/영향

질적연구 접근:
├── 의미 형성 과정
├── 의미 원천
├── 의미 상실
├── 의미 회복
└── 개인차
```

## 권장 방법론

### 1. 현상학적 연구

```
적합한 주제:
├── 몰입 경험 (Flow)
├── 열정의 경험
├── 내재적 동기 순간
├── 동기 상실 경험
└── 의미 경험

IPA 예시 질문:
├── "가장 동기부여 된 순간은?"
├── "그 경험이 어땠나요?"
├── "그것이 당신에게 어떤 의미?"
└── "무엇이 그것을 가능하게 했나요?"
```

### 2. 내러티브 연구

```
적합한 주제:
├── 동기 발달 역사
├── 열정 형성 이야기
├── 소명 발견 서사
├── 커리어 동기 변화
└── 전환점 경험

분석:
├── 줄거리 구조
├── 주요 사건
├── 전환점
├── 현재 의미
└── 미래 전망
```

### 3. 근거이론

```
적합한 주제:
├── 동기 형성 과정
├── 내면화 과정
├── 열정 발달 과정
├── 동기 유지 전략
└── 동기 회복 과정

분석:
├── 조건-상호작용-결과
├── 핵심 범주
├── 과정 모델
└── 이론적 포화
```

### 4. 경험표집법 (ESM) + 질적

```
혼합 접근:
├── 실시간 동기 상태 포착
├── 맥락 정보 수집
├── 후속 심층 면담
├── 경험 해석
└── 양적+질적 통합
```

## 데이터 수집

### 동기 경험 면담

```
구조:

1. 배경
├── 현재 상황/맥락
├── 역할과 책임
└── 일반적 동기 수준

2. 구체적 경험
├── 동기 높았던 구체적 순간
├── 상황 묘사
├── 감정 탐색
├── 생각/인지
└── 결과/행동

3. 동기 부여 요인
├── 무엇이 동기를 높이나?
├── 어떤 조건에서?
├── 누가 영향 미치나?
└── 어떤 보상이?

4. 동기 저하 경험
├── 동기 잃었던 경험
├── 원인 탐색
├── 대처 방식
└── 회복 과정

5. 의미와 목적
├── 왜 이 일을 하나?
├── 어떤 의미가 있나?
├── 미래 전망
└── 성취감 원천
```

### 종단적 접근

```
설계:
├── 여러 시점 면담
├── 동기 변화 추적
├── 사건-경험 연결
├── 일지/다이어리
└── 실시간 포착 (ESM)

시점 예시:
├── 입사 초기
├── 3개월 후
├── 1년 후
├── 전환점 후
└── 주요 사건 후
```

## 분석 접근

### 동기 경험 주제 분석

```
주제 유형:

1. 동기 촉발 요인
├── 내재적 요인
├── 외재적 요인
├── 관계적 요인
└── 맥락적 요인

2. 동기 경험의 특성
├── 정서적 측면
├── 인지적 측면
├── 신체적 측면
└── 행동적 측면

3. 동기 유지 전략
├── 자기 조절
├── 환경 조성
├── 사회적 지지
└── 의미 부여

4. 동기 결과
├── 수행
├── 웰빙
├── 성장
└── 관계
```

### Gioia 적용 예시

```
1차 개념:
├── "목표 달성하면 뿌듯해요"
├── "인정받으면 더 열심히"
├── "성장한다는 느낌이 좋아요"

2차 주제:
├── 성취 기반 만족
├── 사회적 인정 동기
├── 성장 욕구

총괄 차원:
├── 내재적 동기
├── 외재적 동기
├── 성장 동기
```

## 글쓰기 전략

### 동기 경험 기술

```
풍부한 기술:
├── 구체적 상황 묘사
├── 참여자 언어 사용
├── 정서 상태 포함
├── 인지 과정 기술
├── 행동 결과 연결
└── 맥락 정보 제공

예시:
"참여자 3은 프로젝트 완료 후
 '온몸에 전율이 흘렀다'고 표현했다.
 이 성취감은 팀원들의 인정과 결합되어
 다음 프로젝트에 대한 강한 동기로 이어졌다."
```
