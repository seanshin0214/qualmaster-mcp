# 교육행정학 질적연구 완벽 가이드

## 1. 교육행정학의 학문적 위치

### 핵심 연구 영역
```
교육행정학 영역:
├── 교육 리더십 (Educational Leadership)
│   ├── 학교장 리더십
│   ├── 분산적 리더십
│   ├── 교수 리더십
│   └── 변혁적 리더십
│
├── 조직 연구 (Organizational Studies)
│   ├── 학교 조직문화
│   ├── 조직 변화
│   ├── 학교 개선
│   └── 전문학습공동체
│
├── 정책과 거버넌스 (Policy & Governance)
│   ├── 교육정책 분석
│   ├── 학교 자율성
│   ├── 책무성
│   └── 교육 재정
│
└── 인적자원 관리 (Human Resource)
    ├── 교사 채용/유지
    ├── 전문성 개발
    ├── 교원 평가
    └── 교원 노동시장
```

### 이론적 프레임워크
```
1. 제도주의 (Institutionalism)
   - DiMaggio & Powell: 제도적 동형화
   - 신제도주의적 분석
   - 느슨한 결합 (Loose Coupling)

2. 센스메이킹 (Sensemaking)
   - Weick: 조직에서의 의미구성
   - 교육개혁 수용 과정
   - 정책 해석과 실행

3. 분산적 리더십 (Distributed Leadership)
   - Spillane: 리더십 실천
   - 행위자-도구-상황 상호작용
   - 리더십의 사회적 분배

4. 비판적 이론 (Critical Theory)
   - 권력과 불평등
   - 사회정의 리더십
   - 비판적 정책 분석
```

## 3. 핵심 연구 주제

### 학교장 리더십 연구
```
연구 질문:
├── 효과적 학교장의 특성은?
├── 리더십 실천은 어떻게 전개되는가?
├── 맥락이 리더십에 어떤 영향을?
├── 리더십과 학교 개선의 관계는?
└── 학교장의 전문적 성장 과정은?

방법론:
- 심층 인터뷰 (학교장, 교사, 학부모)
- 새도우잉 (Shadowing)
- 일지/저널 분석
- 학교 민족지
- 생애사 연구

Gioia 적용 예시:
1st Order: "직접 수업 볼 시간이 없어요"
2nd Order: Time allocation tensions
Aggregate: INSTRUCTIONAL LEADERSHIP BARRIERS
```

### 정책 실행 연구
```
이론적 틀:

Spillane의 인지적 접근:
- 정책 → 해석 → 실행
- 사전 지식의 영향
- 의미구성 과정

Coburn의 규모 확대 연구:
- 깊이 (Depth)
- 지속가능성 (Sustainability)
- 확산 (Spread)
- 소유권 이전 (Shift in ownership)

방법론:
- 다중 사례연구
- 종단 연구
- 정책 담론 분석
- 실행 과정 추적
```

### 전문학습공동체 (PLC) 연구
```
핵심 특성 (DuFour):
├── 공유된 미션/비전/가치
├── 협력적 탐구
├── 집단적 책임
├── 행동 지향
├── 지속적 개선
└── 결과 중심

연구 질문:
- PLC는 어떻게 형성되는가?
- 효과적 PLC의 특성은?
- 장애요인과 촉진요인은?
- PLC와 학생 성과의 관계는?

방법론:
- 참여관찰
- 회의 녹화 분석
- 교사 인터뷰
- 산출물 분석
```

## 4. 연구 방법론

### 사례연구 설계
```
Yin 접근 적용:

단일 사례:
- 극단적/독특한 학교
- 혁신적 프로그램
- 결정적 사례

다중 사례:
- 학교 유형별 비교
- 성공/실패 사례 비교
- 지역별 비교

분석 단위:
- 개인 (학교장)
- 팀 (리더십 팀)
- 조직 (학교)
- 학구 (교육청)
- 정책 (국가 수준)
```

### 민족지적 접근
```
학교 민족지:

연구 기간:
- 최소 1학기 이상
- 이상적: 1-2년

관찰 초점:
├── 공식 회의
├── 비공식 상호작용
├── 교실 수업
├── 교무실 문화
├── 학교 의례/행사
└── 의사결정 과정

자료 수집:
- 참여관찰
- 심층 인터뷰
- 문서 분석
- 사진/비디오
```

### 내러티브 탐구
```
적용 주제:
├── 학교장 경력 경로
├── 교사 리더 정체성
├── 개혁 경험 서사
└── 위기 대응 이야기

분석 접근:
- 시간적 구조
- 전환점 식별
- 주제적 분석
- 수행적 분석
```

## 5. 데이터 구조 예시

### 분산적 리더십 연구
```
1st Order Concepts
├── "팀장들이 각자 영역을 맡아요"
├── "결정은 함께 내려요"
├── "교장이 다 못하니까요"
├── "전문성에 따라 역할이 나뉘어요"
└── "회의에서 모두 발언권이 있어요"
         ↓
2nd Order Themes
├── Role differentiation
├── Shared decision-making
├── Capacity acknowledgment
├── Expertise-based allocation
└── Voice and participation
         ↓
Aggregate Dimensions
├── STRUCTURAL DISTRIBUTION
├── COLLABORATIVE PRACTICE
└── EMPOWERMENT CULTURE

이론적 통합:
→ Emergent Distributed Leadership Model
```

## 6. 저널 출판 전략

### EAQ 타겟팅
```
성공 요소:
□ 명확한 이론적 기여
□ 엄밀한 방법론
□ 정책/실천 함의
□ 깔끔한 글쓰기

구조:
1. Introduction (문제 제기)
2. Conceptual Framework
3. Research Context
4. Methods (상세)
5. Findings
6. Discussion
7. Implications for Practice/Policy

피해야 할 것:
× 단순 기술적 연구
× 이론 없는 사례 보고
× 일반화 과잉
```

### 심사자 대응
```
일반적 코멘트:

"Contribution not clear"
→ 기존 연구와 명확히 비교
→ "This study advances... by..."

"Context too specific"
→ 전이가능성 논의 강화
→ 조건 명시

"Methods lack rigor"
→ 분석 과정 상세화
→ 신뢰성 확보 방법 추가
```
