## AI 연구와 질적연구

### 왜 질적연구가 중요한가?

```
AI 연구의 특성:
├── 빠른 기술 변화
├── 사회적 영향 불확실
├── 인간-AI 상호작용 복잡
├── 윤리적 이슈
└── 맥락 의존성

질적연구 가치:
├── 사용자 경험 심층 이해
├── 예상치 못한 영향 발견
├── 윤리적 딜레마 탐구
├── 채택/저항 과정
├── 조직 변화 역동
└── 미래 시나리오 탐색
```

### 주요 연구 영역

```
1. AI 채택과 사용
├── 개인의 AI 채택 과정
├── 조직의 AI 도입
├── 저항과 수용
└── 학습과 적응

2. 인간-AI 협업
├── 의사결정 협력
├── 신뢰 구축
├── 역할 분담
└── 책임 귀속

3. AI의 사회적 영향
├── 고용 영향
├── 기술 변화
├── 불평등 이슈
└── 프라이버시

4. AI 윤리
├── 편향과 공정성
├── 투명성과 설명가능성
├── 책임 소재
└── 가치 정렬

5. AI와 정체성
├── 직업 정체성 변화
├── 인간-기계 경계
├── 전문성 재정의
└── 자아 개념
```

## 이론적 프레임워크

### 1. 기술수용모델 확장

```
TAM의 한계:
├── 정적 모델
├── 맥락 무시
├── 과정 포착 불가
└── 사회적 요인 약함

질적연구로 확장:
├── 수용 과정 탐구
├── 저항 이유 이해
├── 맥락 효과
├── 시간적 역동성
└── 사회적 영향
```

### 2. 사회기술시스템 이론

```
핵심 개념:
├── 기술 + 사회 = 시스템
├── 공동 최적화
├── 상호의존성
└── 창발적 특성

질적연구 적용:
├── 기술-조직 상호작용
├── 의도하지 않은 결과
├── 적응 과정
└── 시스템 역동
```

### 3. 물질성 이론 (Materiality)

```
Orlikowski & Scott (2008):
├── 기술의 물질성
├── 사회물질성
├── 얽힘 (Entanglement)
└── 수행성

연구 접근:
├── 일상적 실천
├── 기술과 인간 얽힘
├── 의미 형성
└── 관계적 존재론
```

### 4. 알고리즘 거버넌스

```
핵심 이슈:
├── 알고리즘 의사결정
├── 투명성
├── 책임
└── 권력

질적연구 주제:
├── 알고리즘 경험
├── 저항과 대응
├── 거버넌스 구조
└── 윤리적 긴장
```

## 데이터 수집 전략

### AI 사용자 인터뷰

```
영역:

1. 사용 맥락
├── 어떤 AI 도구 사용?
├── 어떤 상황에서?
├── 얼마나 자주?
└── 어떤 목적으로?

2. 경험
├── 처음 사용했을 때?
├── 어떤 점이 좋은가?
├── 어떤 점이 불편한가?
├── 예상과 다른 점?
└── 변화된 점?

3. 신뢰와 인식
├── AI를 신뢰하나?
├── 어떤 상황에서?
├── 한계는 무엇?
├── AI에 대한 생각?
└── 미래 전망?

4. 영향
├── 일에 어떤 영향?
├── 역할 변화?
├── 새로운 기술 필요?
├── 관계 변화?
└── 정체성 변화?
```

### 조직 AI 도입 연구

```
다층 데이터:

1. 경영진 관점
├── 도입 결정 과정
├── 기대와 목표
├── 전략적 고려
└── 성과 평가

2. 중간관리자
├── 구현 경험
├── 저항 관리
├── 팀 역동
└── 도전과 해결

3. 현장 직원
├── 일상적 경험
├── 학습 과정
├── 적응 전략
└── 우려와 희망

4. IT/개발팀
├── 기술적 도전
├── 통합 이슈
├── 유지보수
└── 발전 방향
```
