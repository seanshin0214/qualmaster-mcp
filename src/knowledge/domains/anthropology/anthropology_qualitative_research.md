# 인류학 (Anthropology) 질적연구 완벽 가이드

## 1. 인류학의 분과와 접근

### 인류학의 4대 분과
```
인류학 분과:
├── 문화인류학 (Cultural Anthropology)
│   ├── 민족지적 연구
│   ├── 상징과 의미
│   └── 사회 조직
│
├── 사회인류학 (Social Anthropology)
│   ├── 영국 전통
│   ├── 사회 구조
│   └── 친족과 정치
│
├── 고고인류학 (Archaeology)
│   ├── 물질문화
│   ├── 과거 사회 복원
│   └── 역사적 과정
│
└── 형질인류학 (Biological Anthropology)
    ├── 인류 진화
    ├── 영장류학
    └── 법의인류학
```

### 이론적 전통
```
고전 이론:
├── 진화론 (Morgan, Tylor)
├── 기능주의 (Malinowski, Radcliffe-Brown)
├── 구조주의 (Lévi-Strauss)
└── 문화상대주의 (Boas)

현대 이론:
├── 해석인류학 (Geertz)
│   └── 두꺼운 기술
├── 정치경제 (Wolf)
│   └── 권력과 역사
├── 실천이론 (Bourdieu, Ortner)
│   └── 구조와 행위
├── 포스트모던 (Clifford, Marcus)
│   └── 민족지 비평
└── 존재론적 전환 (Viveiros de Castro)
    └── 다자연주의
```

## 2. 민족지적 방법론

### 참여관찰의 정수
```
Malinowski의 원칙:
"일상의 불가피한 세부 사항을 
현장 노트에 기록하라"

참여의 유형:
├── 완전 참여자
├── 참여자로서 관찰자
├── 관찰자로서 참여자
└── 완전 관찰자

핵심 실천:
1. 장기 현지 체류 (최소 12개월)
2. 현지어 습득
3. 일상 공유
4. 상세한 기록
5. 총체적 관점
```

### 현장 노트 작성
```
기록 유형:

1. 스크래치 노트
   - 현장에서 즉시 기록
   - 키워드, 약어
   - 기억 촉발용

2. 상세 현장 노트
   - 당일 저녁 작성
   - 구체적 묘사
   - 대화, 행동, 맥락

3. 방법론 노트
   - 연구 과정 성찰
   - 접근 전략
   - 윤리적 딜레마

4. 분석 메모
   - 떠오르는 통찰
   - 패턴 인식
   - 이론적 아이디어

5. 개인 일기
   - 감정적 반응
   - 자기 성찰
   - 개인적 경험
```

### 인터뷰 기법
```
인류학적 인터뷰:

비구조화 인터뷰:
- 자연스러운 대화
- 참여자 주도
- 열린 질문

생애사 (Life History):
- 개인 서사
- 시간적 전개
- 사회적 맥락

핵심정보제공자 (Key Informant):
- 전문적 지식
- 문화적 통찰
- 장기 관계

계보학 인터뷰:
- 친족 관계
- 가족사
- 사회적 네트워크
```

## 3. Geertz의 해석인류학

### 두꺼운 기술 (Thick Description)
```
개념:
Gilbert Ryle에서 차용
"눈 깜빡임" vs "윙크"

층위:
1. 행위 자체 (눈 깜빡임)
2. 의도된 신호 (윙크)
3. 패러디
4. 리허설
...

해석의 과제:
"의미의 그물망"을 읽기
"행동의 의미"를 해석
"사회적 담론"을 기록

방법론적 함의:
- 맥락 속에서 이해
- 행위자의 관점
- 의미의 층위
- 문화적 논리
```

### 문화 개념
```
Geertz의 문화 정의:
"역사적으로 전승된 의미의 패턴,
상징에 체현된 의미 체계"

특성:
├── 공적 (Public)
│   └── 머릿속이 아닌 사회적
├── 상징적 (Symbolic)
│   └── 의미를 담은 기호
├── 해석적 (Interpretive)
│   └── 읽혀야 할 텍스트
└── 맥락적 (Contextual)
    └── 상황 속의 의미
```

## 4. 민족지 글쓰기

### Writing Culture 논쟁
```
Clifford & Marcus (1986):

비판:
├── 권위의 문제
│   └── 누가 말할 권리가 있는가?
├── 재현의 정치
│   └── 어떻게 "타자"를 재현하는가?
├── 수사학적 장치
│   └── 민족지의 문학적 구성
└── 부분적 진리
    └── 완전한 객관성의 불가능

응답:
├── 대화적 민족지
├── 다성적 텍스트
├── 협력적 연구
└── 위치성 성찰
```

### 민족지 장르
```
고전적 민족지:
- 통합적, 전체론적
- 현재 시제
- 객관적 톤

성찰적 민족지:
- 연구 과정 노출
- 관계성 강조
- 자기 성찰

자문화기술지 (Autoethnography):
- 연구자 = 주제
- 개인 경험
- 수행적 글쓰기

다중현장 민족지 (Multi-sited):
- 연결 추적
- 비교 분석
- 글로벌 흐름
```

## 5. 주요 연구 영역

```
├── 결혼 교환

새로운 접근:

├── 퀴어 친족

방법론:
- 계보 작성
- 가구 조사
- 생애사
- 참여관찰
```

### 종교와 의례
```
연구 주제:
├── 신앙과 실천
├── 의례 과정
├── 상징과 신화
├── 치유와 영성

이론적 틀:

방법론:
- 의례 관찰
- 종교 전문가 인터뷰
- 텍스트 분석
- 물질문화 연구
```

```
연구 영역:
├── 질병과 문화
├── 치유 체계
├── 생의학 비판
├── 신체와 체현
└── 글로벌 건강

핵심 개념:
├── Disease vs Illness vs Sickness
├── 설명 모형 (Explanatory Models)
├── 고통의 사회적 맥락
└── 생명윤리와 신기술

방법론:
- 임상 민족지
- 환자 내러티브
- 의료인 관찰
- 정책 분석
```

## 6. 윤리적 고려

### 인류학 윤리 강령
```
AAA 윤리 원칙:

1. 피해 금지 (Do No Harm)
   - 신체적/심리적 피해
   - 사회적/경제적 피해
   - 장기적 영향 고려

2. 투명성 (Transparency)
   - 연구 목적 공개
   - 방법 설명
   - 결과 공유

3. 동의 (Consent)
   - 정보에 기반한 동의
   - 지속적 동의
   - 철회 가능성

4. 비밀보장 (Confidentiality)
   - 익명성
   - 데이터 보호
   - 민감 정보

5. 상호성 (Reciprocity)
   - 연구 혜택 공유
   - 관계의 호혜성
   - 역량 강화
```

### 탈식민주의적 성찰
```
비판적 질문:
├── 연구의 정치적 함의는?
├── 누구를 위한 연구인가?
├── 권력 관계는 어떠한가?
├── 지식의 소유권은?
└── 재현의 윤리는?

실천:
├── 협력적 연구
├── 토착 방법론 존중
├── 결과 환원
└── 장기 관계 구축
```
