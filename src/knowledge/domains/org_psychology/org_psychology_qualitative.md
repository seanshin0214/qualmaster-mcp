## 조직심리학과 질적연구

### 왜 질적연구가 필요한가?

```
양적 연구 한계:
├── 설문 편향 (CMV)
├── 맥락 탈락
├── 인과 메커니즘 불명확
├── 새로운 현상 측정 한계
└── 개인 경험의 풍부함 손실

질적연구 강점:
├── 심리적 과정 심층 이해
├── 맥락 속 경험
├── 의미 형성 과정
├── 새로운 구성개념 발견
└── 이론 개발
```

### 주요 연구 영역

```
1. 직무 태도와 경험
├── 직무 만족/불만족의 경험
├── 몰입의 심리
├── 번아웃 과정
└── 의미 있는 일

2. 정체성과 자아
├── 직업 정체성
├── 조직 동일시
├── 정체성 작업
└── 역할 전환

3. 정서와 웰빙
├── 직장 정서 경험
├── 정서 노동
├── 스트레스 대처
└── 회복

4. 대인관계
├── 리더-구성원 관계
├── 팀 역동
├── 갈등 경험
└── 신뢰 구축

5. 경력 발달
├── 경력 전환
├── 성장 경험
├── 멘토링
└── 네트워킹
```

## 주요 저널

### Top-tier 저널

```
1. Journal of Applied Psychology (JAP)
   ├── IF: ~8
   ├── 주로 양적, 질적 드묾
   └── 방법론적 엄격성 높음

2. Organizational Behavior and Human Decision Processes (OBHDP)
   ├── IF: ~5
   ├── 의사결정 과정
   └── 질적 가능

3. Journal of Organizational Behavior (JOB)
   ├── IF: ~8
   ├── 조직행동 전문
   └── 질적연구 환영

4. Journal of Vocational Behavior (JVB)
   ├── IF: ~6
   ├── 경력/직업 연구
   └── 질적 수용

5. Work & Stress
   ├── IF: ~5
   ├── 직무 스트레스
   └── 질적연구 전통
```

### 질적연구 친화적 저널

```
1. Human Relations
   ├── IF: ~5
   ├── 비판적 관점
   └── 질적 전통 강함

2. Journal of Management Inquiry
   ├── 반성적 접근
   └── 다양한 형식

3. Qualitative Research in Organizations and Management
   ├── 질적연구 전문
   └── 방법론 논의

4. Work, Employment & Society
   ├── 사회학적 관점
   └── 질적연구 다수
```

## 핵심 이론적 프레임워크

### 1. 정체성 이론 (Identity Theory)

```
핵심 개념:
├── 사회적 정체성
├── 역할 정체성
├── 자기 개념
└── 정체성 작업

질적연구 질문:
├── 직업 정체성은 어떻게 형성되는가?
├── 조직 변화 시 정체성은?
├── 역할 갈등은 어떻게 경험되는가?
└── 정체성 위협에 어떻게 대응하는가?
```

### 2. 센스메이킹 (Sensemaking)

```
핵심 아이디어:
├── 모호성 속 의미 형성
├── 사후적 해석
├── 정체성 기반
└── 사회적 과정

적용:
├── 조직 변화 경험
├── 위기 대응
├── 역할 전환
└── 직무 의미 형성
```

### 3. 정서 조절 (Emotion Regulation)

```
개념:
├── 표면 행위
├── 심층 행위
├── 정서 부조화
└── 정서 자원

질적연구 접근:
├── 정서 노동 경험
├── 조절 전략
├── 맥락적 요인
└── 장기 효과
```

### 4. 심리적 계약 (Psychological Contract)

```
개념:
├── 암묵적 기대
├── 위반과 파괴
├── 이행과 초과
└── 재협상

질적연구 가치:
├── 계약 내용 탐구
├── 위반 경험
├── 대응 과정
└── 맥락 차이
```

## 데이터 수집 전략

```
도전:
├── 조직 허가
├── 참여자 신뢰
├── 민감한 주제
├── 시간 제약
└── 정치적 역학

대응:
├── 게이트키퍼 관계
├── 비밀보장 강조
├── 유연한 일정
├── 조직 혜택 제시
└── 중립성 유지
```

### 심층 면담 설계

```
경험 탐구:
├── "그 경험에 대해 말씀해 주세요"
├── "어떤 느낌이었나요?"
├── "무엇이 가장 어려웠나요?"
├── "어떤 의미가 있었나요?"
└── "어떻게 대처하셨나요?"

프로빙:
├── 감정 탐색
├── 구체적 상황
├── 시간 흐름
├── 관계 영향
└── 변화 지점
```

### 일지/다이어리 연구

```
적합한 주제:
├── 일상 경험
├── 정서 변동
├── 스트레스/회복
├── 학습 과정
└── 관계 발전

설계:
├── 주기 (일/주)
├── 기간 (4-12주)
├── 형식 (구조화/비구조화)
├── 매체 (앱/종이)
└── 후속 인터뷰
```

## 분석 접근

### 경험 분석

```
IPA 예시:

1. 개별 사례 분석
   ├── 전사본 반복 읽기
   ├── 기술적 코멘트
   ├── 언어적 코멘트
   └── 개념적 코멘트

2. 발현 주제
   ├── 코멘트 → 주제
   ├── 심리학적 개념화
   └── 추상화

3. 사례 간 분석
   ├── 수렴 패턴
   ├── 발산 패턴
   └── 상위 주제
```

### 과정 분석

```
Gioia 방법론:

1차 개념:
├── "처음엔 혼란스러웠어요"
├── "점점 이해가 됐어요"
├── "나만의 방식을 찾았어요"

2차 주제:
├── 초기 혼란
├── 점진적 이해
├── 개인화

총괄 차원:
├── 적응 단계
├── 학습 메커니즘
├── 성장 결과
```

## 글쓰기 전략

### JOB 스타일

```
특징:
├── 이론적 기여 명확
├── 실무 함의
├── 방법론적 투명성
└── 풍부한 데이터
```
