## Entrepreneurship Qualitative Research

## 분야 특성

### 왜 질적연구가 중요한가?

```
창업 현상의 특성:
├── 과정적 (Process-oriented)
├── 맥락 의존적 (Context-dependent)
├── 역동적 (Dynamic)
├── 희소 사건 (Rare events)
└── 복잡한 인과관계

양적연구의 한계:
├── 과정 포착 어려움
├── 새로운 현상 측정 도구 부재
├── 맥락 손실
└── 작은 표본 (특히 성공 창업)
```

### 주요 연구 주제

```
1. 기회 인식/창출 (Opportunity)
├── 어떻게 기회를 발견/창출하는가?
├── 인지 과정
└── 센스메이킹

2. 창업 과정 (Entrepreneurial Process)
├── 시간에 따른 진화
├── 의사결정 과정
└── 피벗과 학습

3. 창업자 경험 (Founder Experience)
├── 실패 경험
├── 정서적 여정
└── 정체성 변화

4. 맥락과 생태계 (Context/Ecosystem)
├── 제도적 환경
├── 네트워크 형성
└── 문화적 영향

5. 사회적/지속가능 창업
├── 사회적 가치 창출
├── 하이브리드 조직
└── 임팩트 측정
```

## 주요 이론적 프레임워크

### 1. Effectuation vs Causation (Sarasvathy)

```
Causation (인과적 논리):
├── 목표 설정 → 수단 탐색
├── 예측에 기반
├── 경쟁 분석
└── 기대 수익 극대화

Effectuation (실현적 논리):
├── 수단에서 출발 → 목표 창출
├── 통제에 기반
├── 파트너십 구축
└── 감당 가능한 손실

질적연구 적합:
├── 의사결정 과정 탐구
├── 인지 패턴 파악
├── 맥락적 요인 이해
└── 시간에 따른 변화
```

### 2. Bricolage (Baker & Nelson)

```
Bricolage = "수중에 있는 것으로 해결"

구성요소:
├── Making do: 즉흥적 해결
├── Combination of resources: 자원 재조합
├── Resources at hand: 기존 자원 활용
└── Refusal of limitation: 제약 거부

질적연구 방법:
├── 자원 활용 과정 추적
├── 창의적 해결책 문서화
├── 제약 극복 내러티브
└── 비교 사례 분석
```

### 3. Entrepreneurial Identity (Fauchart & Gruber)

```
창업자 정체성 유형:
├── Darwinian: 경제적 이익 추구
├── Communitarian: 공동체 기여
└── Missionary: 사회 변화 추구

연구 방법:
├── 정체성 내러티브 분석
├── 정체성 구성 과정
├── 정체성 갈등과 협상
└── 시간에 따른 변화
```

### 4. Entrepreneurial Cognition

```
주요 개념:
├── Mental models
├── Heuristics & biases
├── Pattern recognition
├── Counterfactual thinking
└── Prospection

질적연구 방법:
├── Think-aloud protocol
├── Cognitive mapping
├── Retrospective interview
└── Critical incident technique
```

## 권장 방법론

### 1. 과정 연구 (Process Research)

```
접근:
├── 종단적 데이터 수집
├── 실시간 추적 + 회고적 재구성
├── 사건 순서 분석
└── 내러티브 전략

분석:
├── 시간적 브래킷팅
├── 과정 코딩
├── 시각적 맵핑
└── 패턴 매칭
```

### 2. 사례연구 (Case Study)

```
적합한 상황:
├── "어떻게" "왜" 질문
├── 현상과 맥락 경계 불분명
├── 희소 사건 (유니콘, 대실패)
└── 새로운 현상

설계:
├── 단일 사례: 극단적/독특한 경우
├── 다중 사례: 복제/확장 논리
├── 내재적: 사례 자체 관심
└── 도구적: 현상 이해 수단
```

### 3. 문화기술지 (Ethnography)

```
적합한 상황:
├── 창업 생태계 연구
├── 액셀러레이터/인큐베이터
├── 코워킹 스페이스
└── 창업 공동체

접근:
├── 참여관찰
├── 장기간 몰입
├── 현장 노트
└── 다양한 데이터 소스
```

### 4. 내러티브 연구 (Narrative)

```
적합한 상황:
├── 창업자 여정
├── 실패와 회복
├── 정체성 변화
└── 의미 형성

분석:
├── 줄거리 구조
├── 주인공-조력자-적대자
├── 전환점 (Turning points)
└── 주제적 분석
```

## 데이터 수집 전략

### 창업자 인터뷰 특이사항

```
도전:
├── 바쁜 일정
├── 회고적 편향
├── 성공 귀인 편향
├── 선택적 기억
└── 기밀 정보

대응:
├── 유연한 일정
├── 다중 시점 인터뷰
├── 삼각화 (문서, 관찰)
├── 프로빙 기술
└── 비밀보장 약속
```

### 실시간 데이터 수집

```
방법:
├── 창업자 일지
├── 정기 인터뷰 (주/월)
├── 회의 관찰
├── 이메일/메시지 분석
└── 소셜미디어 추적

장점:
├── 회고 편향 감소
├── 과정 포착
├── 풍부한 맥락
└── 시간적 정확성
```

### 아카이브 데이터

```
소스:
├── 투자유치 문서 (피치덱)
├── 사업계획서
├── 미디어 기사
├── 소셜미디어
├── 특허/지재권
├── 웹사이트 아카이브
└── 규제 문서

활용:
├── 삼각화
├── 시간적 맥락
├── 공식 vs 비공식 비교
└── 변화 추적
```

## 분석 접근

### Gioia 방법론 적용

```
특히 적합:
├── 새로운 개념 개발
├── 과정 모델 구축
├── 창업 현상 이론화
└── 귀납적 이론 구축

예시 - 기회 인식 연구:

1차 개념:
├── "우연히 발견했다"
├── "문제를 겪고 해결책 생각"
├── "트렌드를 보고"

2차 주제:
├── 우연적 발견
├── 문제 중심 탐색
├── 트렌드 기반 인식

총괄 차원:
├── 수동적 기회 인식
├── 능동적 기회 탐색
├── 기회 창출
```

### 과정 코딩

```
Van de Ven & Poole 접근:

1. 사건 식별
   ├── 무엇이 일어났나?
   ├── 언제?
   ├── 누가 관여?
   └── 결과는?

2. 사건 순서화
   ├── 시간순 배열
   ├── 인과 연결
   └── 단계 식별

3. 패턴 발견
   ├── 반복 시퀀스
   ├── 전환점
   └── 경로 의존성
```

## 글쓰기 전략

### JBV 스타일

```
특징:
├── 과정 강조
├── 풍부한 데이터
├── 이론적 기여 명확
└── 실무적 함의

구조:
1. Introduction: 현상과 퍼즐
2. Theoretical Background: 관련 이론
3. Method: 상세한 절차
4. Findings: 데이터 기반 주장
5. Discussion: 이론적 기여
```

### ETP 스타일

```
특징:
├── 이론 개발 강조
├── 개념적 명확성
├── 방법론적 엄격성
└── 일반화 가능성

주의:
├── 이론과 데이터 균형
├── 과정 모델 시각화
├── 명제 도출
└── 경계 조건
```
