# 엔터프레뉴어십 이론 마스터 가이드: 신급 질적연구

## 1. 엔터프레뉴어십 이론의 진화

### 패러다임 전환
```
이론적 발전:
├── 1단계: 특성 접근 (Trait Approach)
│   ├── 창업자는 누구인가?
│   ├── 성취동기 (McClelland)
│   └── 위험감수성, 통제위치
│
├── 2단계: 행동 접근 (Behavioral Approach)
│   ├── 창업자는 무엇을 하는가?
│   ├── 기회 인식/활용
│   └── 자원 동원
│
├── 3단계: 과정 접근 (Process Approach)
│   ├── 창업은 어떻게 전개되는가?
│   ├── Effectuation
│   ├── Bricolage
│   └── Lean Startup
│
└── 4단계: 맥락 접근 (Contextual Approach)
    ├── 창업은 어디서 일어나는가?
    ├── Entrepreneurial Ecosystems
    ├── Institutional Entrepreneurship
    └── Social Entrepreneurship
```

## 2. 핵심 이론 심층 분석

### Effectuation Theory (Sarasvathy, 2001)
```
인식론적 기반:
- 기사도적 합리성 비판
- 불확실성 vs 위험 구분
- 전문 창업가 연구 기반

5대 원칙:

1. Bird-in-hand (가용 수단 원칙)
   "나는 누구인가? 무엇을 아는가? 누구를 아는가?"
   → 목표에서 수단으로 (Causation)
   → 수단에서 목표로 (Effectuation)

2. Affordable Loss (감당 가능 손실)
   기대 수익 극대화 ≠ 감당 가능 손실 최소화
   → 실패해도 버틸 수 있는 범위

3. Crazy Quilt (조각보 원칙)
   경쟁 분석 < 파트너십 구축
   → 이해관계자를 공동창업자로

4. Lemonade (레몬에이드 원칙)
   우연을 활용
   → 예상치 못한 것을 기회로

5. Pilot-in-the-plane (조종사 원칙)
   예측 < 통제
   → 미래를 예측하지 말고 만들어라

질적연구 적용:
├── 의사결정 과정 추적
├── 실시간 관찰/인터뷰
├── Retrospective 심층 인터뷰
└── Think-aloud 프로토콜
```

### Bricolage (Baker & Nelson, 2005)
```
정의:
"제한된 자원으로 새로운 가치를 창출하는 과정"

3가지 핵심 요소:

1. Making Do (있는 것으로 해결)
   - 표준적 자원 거부
   - 창의적 재사용
   - 즉흥적 해결

2. Combination of Resources (자원 재조합)
   - 기존 자원의 새로운 용도
   - 다목적 활용
   - 물리적/지적/사회적 자원

3. Refusal to Enact Limitations (한계 거부)
   - 규범적 제약 무시
   - 가능성 재정의
   - 규칙 파괴

유형:
├── Parallel Bricolage: 여러 프로젝트 동시
├── Selective Bricolage: 특정 영역에만
└── Network Bricolage: 관계를 통한

연구 방법:
- 종단적 사례연구
- 자원 동원 과정 추적
- 제약 상황 관찰
- 창의적 해결 분석
```

### Entrepreneurial Identity (Fauchart & Gruber, 2011)
```
3가지 창업자 정체성:

1. Darwinian Identity (다윈형)
   - 자기 이익 추구
   - 경쟁 중심
   - 경제적 성과 지향
   - "나는 부자가 되고 싶다"

2. Communitarian Identity (공동체형)
   - 특정 집단 지향
   - 커뮤니티 기여
   - 소속감 중시
   - "우리 업계를 위해"

3. Missionary Identity (선교사형)
   - 사회 변화 추구
   - 가치 기반
   - 세상을 바꾸려는 열망
   - "세상을 더 낫게"

질적연구 접근:
├── 내러티브 정체성 분석
├── 생애사 인터뷰
├── 정체성 작업(Identity Work) 관찰
└── 시간에 따른 정체성 변화 추적
```

### Entrepreneurial Passion (Cardon et al., 2009)
```
영역 특수적 열정:

1. Passion for Inventing
   - 새로운 제품/서비스 창조
   - 문제 해결 희열
   - 발명가 정체성

2. Passion for Founding
   - 벤처 설립 과정
   - 조직 구축
   - 창업가 정체성

3. Passion for Developing
   - 성장과 확장
   - 스케일업
   - 개발자 정체성

질적연구 주제:
├── 열정의 발현과 표현
├── 팀 내 열정 전이
├── 열정의 어두운 면
└── 열정과 웰빙
```

## 3. 현상 중심 연구

### Entrepreneurial Failure
```
핵심 문헌:
├── Shepherd (2003): 비탄 회복
├── Cope (2011): 실패로부터 학습
├── Byrne & Shepherd (2015): 감정 회복
└── Singh et al. (2007): 낙인

연구 질문:
├── 실패는 어떻게 경험되는가?
├── 실패에서 무엇을 배우는가?
├── 낙인은 어떻게 관리되는가?
├── 재기는 어떻게 이루어지는가?

방법론:
├── 현상학적 인터뷰
├── 내러티브 분석
├── 종단적 추적
└── 감정 데이터 수집
```

### Entrepreneurial Exit
```
연구 주제:
├── Exit 의사결정 과정
├── Exit 이후 정체성
├── 연쇄 창업 (Serial Entrepreneurship)
├── Exit과 웰빙
└── 가족기업 승계

이론적 틀:
├── 정체성 이론
├── 이별 이론 (Detachment)
├── 경력 전환
└── 감정 조절
```

### Entrepreneurial Ecosystems
```
구성 요소 (Stam, 2015):
├── 문화 (Culture)
├── 네트워크 (Networks)
├── 리더십 (Leadership)
├── 금융 (Finance)
├── 인재 (Talent)
├── 지식 (Knowledge)
├── 지원 서비스 (Support Services)
└── 물리적 인프라

질적 접근:
├── 생태계 형성 과정 연구
├── 핵심 행위자 역할
├── 네트워크 다이내믹스
├── 정책 개입 효과
└── Multi-site 비교

방법론:
├── 네트워크 민족지
├── 다중 이해관계자 인터뷰
├── 역사적 과정 추적
└── 참여관찰
```

## 4. 특수 맥락 연구

### Social Entrepreneurship
```
정의 논쟁:
├── 사회적 가치 창출 중심
├── 혼합 가치 (Blended Value)
├── 시스템 변화
└── 지속가능성

이론적 틀:
├── Institutional Entrepreneurship
├── Social Value Creation
├── Hybrid Organizing
└── Scaling Social Impact

연구 주제:
├── 사회적 기업가 정체성
├── 긴장 관리 (Mission vs Money)
├── 사회적 임팩트 측정
├── 스케일링 전략
└── 생태계와 정책
```

### Family Entrepreneurship
```
핵심 주제:
├── 세대 간 승계
├── 가족 기업 거버넌스
├── 가족-사업 인터페이스
├── 장수 기업 비결
└── 기업가적 지향성

이론적 틀:
├── 사회정서적 부 (SEW)
├── 스튜어드십 이론
├── 가족 시스템 이론
└── 세대 간 전이

방법론:
├── 다세대 인터뷰
├── 가족 역사 연구
├── 참여관찰
└── 문서/사진 분석
```

## 5. JBV/ETP 출판 전략

### 성공적 질적연구 공식
```
JBV 선호 요소:
□ Compelling phenomenon
□ Process-oriented question
□ Gioia methodology (often)
□ Clear theoretical contribution
□ Rich data structure

ETP 차별화:
□ 실무 함의 강조
□ Teaching implications
□ Family business 연결
□ 사례 교육적 활용
```

### Data Structure 예시
```
1st Order Concepts
├── "처음엔 아는 것부터 시작했어요"
├── "잃어도 괜찮은 만큼만 투자했죠"
├── "우연히 만난 사람이 파트너가 됐어요"
├── "계획대로 안 되면 방향을 바꿨어요"
└── "예측보다는 통제할 수 있는 걸 했어요"
         ↓
2nd Order Themes
├── Means-driven action
├── Affordable loss logic
├── Contingent partnerships
├── Adaptive pivoting
└── Control over prediction
         ↓
Aggregate Dimensions
├── NON-PREDICTIVE STRATEGY
├── STAKEHOLDER CO-CREATION
└── UNCERTAINTY EMBRACING

이론적 통합:
→ Effectual Entrepreneuring Process
```
