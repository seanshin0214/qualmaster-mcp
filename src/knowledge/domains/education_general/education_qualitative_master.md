# 교육학 질적연구 마스터 가이드

## 1. 교육학 질적연구의 기반

### 교육 연구의 패러다임
```
패러다임 전쟁과 화해:
├── 실증주의 (Positivism)
│   ├── 객관적 지식
│   ├── 인과관계 규명
│   └── 일반화 추구
│
├── 해석주의 (Interpretivism)
│   ├── 의미와 이해
│   ├── 맥락 중시
│   └── 참여자 관점
│
├── 비판이론 (Critical Theory)
│   ├── 권력과 불평등
│   ├── 해방적 관심
│   └── 사회 변화
│
└── 포스트모던/포스트구조주의
    ├── 다원성과 차이
    ├── 담론과 권력
    └── 해체와 재구성
```

### 교육 연구의 고유성
```
교육 현상의 특성:
├── 복잡성 (Complexity)
│   └── 다층적 요인의 상호작용
├── 맥락성 (Contextuality)
│   └── 상황 의존적 현상
├── 규범성 (Normativity)
│   └── 가치와 목적 내재
├── 실천성 (Practicality)
│   └── 실천 개선 지향
└── 관계성 (Relationality)
    └── 인간 관계의 중심성
```

## 3. 핵심 연구 영역

### 교수-학습 연구
```
연구 주제:
├── 교사 사고 (Teacher Thinking)
│   ├── 실천적 지식
│   ├── 반성적 실천
│   └── 신념과 태도
│
├── 학습 경험 (Learning Experiences)
│   ├── 학습자 관점
│   ├── 의미구성
│   └── 정체성 형성
│
├── 교실 상호작용 (Classroom Interaction)
│   ├── 담화 분석
│   ├── 참여 구조
│   └── 권력 관계
│
└── 교육과정 경험 (Curriculum Experience)
    ├── 실행된 교육과정
    ├── 숨은 교육과정
    └── 학생 경험

방법론:
├── 교실 민족지
├── 내러티브 탐구
├── 현상학적 연구
└── 사례연구
```

### 교사 연구 (Teacher Research)
```
교사 정체성 (Teacher Identity):
├── 정체성 형성 과정
├── 정체성의 다층성
├── 갈등과 협상
└── 시간에 따른 변화

교사 발달 (Teacher Development):
├── 초임 교사 경험
├── 전문성 발달 경로
├── 멘토링과 협력
└── 번아웃과 회복력

교사 지식 (Teacher Knowledge):
├── Shulman의 PCK
├── 실천적 지식
├── 지식의 체화
└── 지식 공유

방법론:
├── 생애사 연구
├── 내러티브 탐구
├── 자기연구 (Self-Study)
├── 실행연구 (Action Research)
└── 집단 전기 (Collective Biography)
```

### 학생 연구 (Student Research)
```
학생 경험:
├── 학교 경험의 의미
├── 또래 관계
├── 정체성 협상
└── 저항과 순응

학생 목소리 (Student Voice):
├── 참여적 연구
├── 권한 부여
├── 학생 관점
└── 변화 행위자

특수 집단:
├── 소외 계층 학생
├── 이주 배경 학생
├── 장애 학생
├── 영재 학생
└── 퀴어 학생

방법론:
├── 현상학적 인터뷰
├── 포토보이스
├── 일지/저널
├── 포커스 그룹
└── 참여관찰
```

## 4. 이론적 프레임워크

### 사회문화이론 (Sociocultural Theory)
```
Vygotsky 전통:

핵심 개념:
├── 근접발달영역 (ZPD)
├── 비계설정 (Scaffolding)
├── 매개 (Mediation)
├── 내면화 (Internalization)
└── 문화적 도구

확장:
├── 활동이론 (Activity Theory)
│   └── Engeström의 확장된 모형
├── 상황학습 (Situated Learning)
│   └── Lave & Wenger
└── 대화주의 (Dialogism)
    └── Bakhtin

연구 적용:
├── 교실 담화 분석
├── 협력학습 연구
├── 도구 매개 학습
└── 학습 공동체
```

### 비판적 교육학 (Critical Pedagogy)
```
Freire 전통:

핵심 개념:
├── 의식화 (Conscientização)
├── 은행예금식 교육 비판
├── 문제제기식 교육
├── 프락시스 (Praxis)
└── 대화 (Dialogue)

발전:
├── Giroux: 비판적 교육학
├── hooks: 참여적 교육학
├── McLaren: 혁명적 비판교육학
└── Apple: 비판적 교육과정 이론

연구 적용:
├── 권력 관계 분석
├── 저항과 대항 헤게모니
├── 사회정의 교육
├── 참여적 행동연구
└── 해방적 교육 실천
```

### 포스트구조주의 접근
```
핵심 개념:
├── 담론 (Discourse)
├── 주체화 (Subjectification)
├── 권력/지식
├── 수행성 (Performativity)
└── 해체 (Deconstruction)

교육 적용:
├── Foucault와 학교
│   ├── 규율과 감시
│   ├── 시험과 정상화
│   └── 통치성
├── Butler와 젠더 교육
│   ├── 젠더 수행성
│   ├── 퀴어 교육학
│   └── 정체성 정치
└── Derrida와 교육과정
    ├── 텍스트 해체
    ├── 이분법 비판
    └── 차연

방법론:
├── 담론 분석
├── 계보학
├── 해체적 읽기
└── 정책 고고학
```

## 5. 연구 방법론

### 교육 민족지
```
연구 현장:
├── 학교 전체
├── 교실
├── 교무실
├── 학생 공간
├── 가정-학교 연계
└── 온라인 공간

관찰 초점:
├── 일상적 상호작용
├── 의례와 루틴
├── 권력 관계
├── 문화적 실천
└── 저항과 순응

특수 접근:
├── 비판적 민족지
├── 수행적 민족지
├── 디지털 민족지
└── 다중현장 민족지
```

### 내러티브 탐구 (Narrative Inquiry)
```
Clandinin & Connelly 접근:

3차원 공간:
├── 시간적 (Temporal)
│   └── 과거-현재-미래
├── 사회적 (Social)
│   └── 개인-사회 상호작용
└── 장소 (Place)
    └── 물리적/상상적 공간

연구 과정:
1. 현장 진입
2. 현장 텍스트 구성
3. 연구 텍스트로 전환
4. 의미 협상

적용:
├── 교사 이야기
├── 학생 경험
├── 교육과정 이야기
└── 학교 이야기
```

### 자기연구 (Self-Study)
```
특성:
├── 실천가-연구자
├── 자기 탐구
├── 협력적 성찰
├── 공적 지식 생산
└── 실천 개선

유형:
├── 교사교육자 자기연구
├── 교사 자기연구
├── 협력적 자기연구
└── 발달적 자기연구

방법:
├── 반성적 저널
├── 동료 대화
├── 비디오 분석
├── 학생 피드백
└── 비판적 친구
```

## 6. 데이터 구조 예시

### 교사 정체성 연구
```
1st Order Concepts
├── "처음엔 학생들 앞에서 떨렸어요"
├── "선배 선생님처럼 되고 싶었어요"
├── "내 방식을 찾기 시작했어요"
├── "실패해도 다시 시도해요"
└── "이제 저만의 철학이 있어요"
         ↓
2nd Order Themes
├── Initial vulnerability
├── Role modeling
├── Finding own voice
├── Resilience building
└── Philosophical grounding
         ↓
Aggregate Dimensions
├── IDENTITY STRUGGLE
├── PROFESSIONAL SOCIALIZATION
└── AUTHENTIC SELF-MAKING

이론적 통합:
→ Teacher Identity Becoming Process
```
