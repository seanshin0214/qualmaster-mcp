# 수학과 질적연구 완벽 가이드: 수학철학, 수학교육, 수학사

## 1. 수학철학에서의 질적 탐구

### 수학의 존재론
```
수학적 대상의 본성:
├── 플라톤주의 (Platonism)
│   ├── 수학적 대상은 독립적으로 존재
│   ├── 수학자는 발견자
│   └── Gödel, Hardy
│
├── 형식주의 (Formalism)
│   ├── 수학은 기호 조작
│   ├── 무의미한 게임
│   └── Hilbert
│
├── 직관주의 (Intuitionism)
│   ├── 수학은 정신적 구성
│   ├── 배중률 거부
│   └── Brouwer
│
├── 논리주의 (Logicism)
│   ├── 수학은 논리로 환원
│   ├── Frege, Russell
│   └── 집합론적 역설로 위기
│
└── 사회구성주의
    ├── 수학적 지식은 사회적 산물
    ├── 증명은 사회적 합의
    └── Lakatos, Ernest
```

### 수학의 인식론
```
수학적 지식의 특성:
├── 선험성 (A priori)
│   └── 경험 독립적?
├── 필연성 (Necessity)
│   └── 필연적 진리?
├── 확실성 (Certainty)
│   └── 절대적 확실성?
└── 적용가능성 (Applicability)
    └── 왜 물리세계에 적용되는가?

질적 탐구 질문:
- 수학자들은 어떻게 새로운 수학을 창조하는가?
- 증명은 어떻게 설득력을 갖는가?
- 수학적 직관은 무엇인가?
- 아름다움은 수학에서 어떤 역할을 하는가?
```

## 2. 수학사 연구 방법론

### 역사적 사례연구
```
주요 사례:
1. 비유클리드 기하학의 발견
   - 평행선 공리 문제
   - Gauss, Lobachevsky, Bolyai
   - 수학적 진리 개념의 변화

2. 무한의 수학화 (Cantor)
   - 집합론의 탄생
   - 무한의 비교
   - 연속체 가설

3. 수학 기초 위기 (1900년대 초)
   - Russell의 역설
   - 세 학파의 논쟁
   - Gödel의 불완전성

4. 페르마의 마지막 정리 증명 (Wiles)
   - 350년의 도전
   - 현대 수학의 통합
   - 증명 검증 과정

연구 방법:
- 원전 분석
- 서신/노트 연구
- 맥락적 재구성
- 개념 발전 추적
```

### Lakatos의 수학철학
```
"수학적 발견의 논리" (Proofs and Refutations):

증명과 반박의 변증법:
1. 원시적 추측 (Primitive Conjecture)
2. 증명 시도
3. 반례 (Counterexample) 등장
4. 개념 재정의 (Monster-barring)
5. 보조정리 편입 (Lemma-incorporation)
6. 정제된 추측

방법론적 함의:
- 수학도 오류가능
- 증명은 역사적으로 발전
- 비형식적 수학의 중요성
- 휴리스틱의 역할
```

## 3. 수학교육 연구 (Mathematics Education Research)

### 주요 이론적 프레임워크
```
1. 구성주의 (Constructivism)
   - Piaget: 인지적 구성
   - von Glasersfeld: 급진적 구성주의
   - 학습자의 능동적 의미구성

2. 사회문화이론 (Sociocultural Theory)
   - Vygotsky: 근접발달영역
   - 문화적 도구로서의 수학
   - 담론 참여로서의 학습

3. 현실적 수학교육 (RME)
   - Freudenthal: 수학화
   - 맥락적 문제에서 출발
   - 수평적/수직적 수학화

4. 수학적 지식 (Mathematical Knowledge)
   - Shulman: PCK
   - Ball: MKT (Mathematical Knowledge for Teaching)
   - 가르치기 위한 수학 지식
```

### 연구 방법론
```
질적 연구 설계:

1. 임상 인터뷰 (Clinical Interview)
   - Piaget 전통
   - 학생 사고 탐색
   - 유연한 질문

2. 설계 연구 (Design Research)
   - 교수학습 실험
   - 반복적 개선
   - 이론-실천 통합

3. 교실 연구 (Classroom Research)
   - 비디오 분석
   - 담화 분석
   - 참여관찰

4. 교사 연구 (Teacher Research)
   - 수업 사례연구
   - 전문성 발달
   - 반성적 실천
```

### 수학교육 연구 저널
```
최상위:
├── Journal for Research in Mathematics Education (JRME)
│   └── NCTM 공식 저널
├── Educational Studies in Mathematics
│   └── 국제 수학교육
├── ZDM - Mathematics Education
│   └── 독일 기반, 국제적
└── Mathematical Thinking and Learning
    └── 인지과학 연계

특화 저널:
├── For the Learning of Mathematics
│   └── 교사 실천 중심
├── International Journal of Science and Mathematics Education
│   └── 과학-수학 통합
└── Journal of Mathematical Behavior
    └── 학습자 행동 분석
```

## 4. 증명 연구

### 증명의 본성
```
증명의 기능 (de Villiers):
├── 검증 (Verification)
├── 설명 (Explanation)
├── 체계화 (Systematization)
├── 발견 (Discovery)
├── 소통 (Communication)
└── 지적 도전 (Intellectual Challenge)

증명 유형:
├── 형식적 증명 (Formal Proof)
├── 엄밀한 논증 (Rigorous Argument)
├── 설명적 증명 (Explanatory Proof)
├── 시각적 증명 (Visual Proof)
└── 실험적 확인 (Empirical Verification)
```

### 증명 학습 연구
```
연구 질문:
1. 학생들은 증명을 어떻게 이해하는가?
2. 증명 구성 과정은 어떠한가?
3. 증명의 역할에 대한 인식은?
4. 효과적인 증명 교수법은?

방법론:
- Think-aloud 프로토콜
- 증명 읽기/쓰기 분석
- 동료 상호작용 관찰
- 교사 피드백 분석
```

## 5. 수학적 실천 연구

### 수학자 연구
```
연구 주제:
├── 수학적 창의성
│   ├── 영감의 순간
│   ├── 미적 판단
│   └── 문제 선택
│
├── 증명 실천
│   ├── 증명 작성 과정
│   ├── 검증과 수정
│   └── 협력적 증명
│
├── 수학 공동체
│   ├── 동료 심사
│   ├── 세미나 문화
│   └── 학파 형성
│
└── 수학적 커뮤니케이션
    ├── 논문 작성
    ├── 발표
    └── 비형식적 논의

연구 방법:
- 수학자 인터뷰
- 노트/초고 분석
- 세미나 관찰
- 서신/이메일 분석
```

### 수학적 문제해결
```
Pólya의 4단계:
1. 문제 이해 (Understanding)
2. 계획 수립 (Planning)
3. 실행 (Carrying Out)
4. 반성 (Looking Back)

Schoenfeld의 프레임워크:
├── 자원 (Resources)
├── 휴리스틱 (Heuristics)
├── 통제 (Control)
└── 신념 체계 (Belief Systems)

질적 연구:
- 문제해결 과정 비디오 분석
- 전문가-초보자 비교
- 메타인지 탐구
```

## 6. 수학과 사회

### 민족수학 (Ethnomathematics)
```
D'Ambrosio의 정의:
다양한 문화에서 발전한 수학적 실천

연구 영역:
├── 토착 수학 (Indigenous Mathematics)
├── 일상 수학 (Everyday Mathematics)
├── 직업 수학 (Workplace Mathematics)
└── 역사적 수학 전통

연구 방법:
- 민족지적 연구
- 참여관찰
- 도구/인공물 분석
- 구술 역사
```

### 수학과 젠더
```
연구 주제:
├── 수학 성취 차이
├── 자기효능감 차이
├── 수학 정체성 형성
├── 여성 수학자 경험
└── 교실 상호작용 패턴

방법론:
- 생애사 인터뷰
- 교실 담화 분석
- 정체성 내러티브
- 제도 분석
```
