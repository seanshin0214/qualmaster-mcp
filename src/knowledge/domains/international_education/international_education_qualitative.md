# International & Comparative Education 질적연구 완벽 가이드

## 1. 분야 개관

### 인식론적 위치
국제교육학은 교육 현상의 문화적, 역사적, 정치적 맥락을 중시하며, 질적연구가 분야의 핵심 방법론이다.

```
국제교육학 질적 패러다임:
├── Comparative Education
│   ├── Cross-national comparison
│   ├── Policy borrowing/lending
│   └── 맥락의 중요성
│
├── International Education
│   ├── Student mobility
│   ├── Internationalization
│   └── Global citizenship
│
├── Development Education
│   ├── Education for development
│   ├── Critical pedagogy
│   └── Decolonizing approaches
│
└── Multicultural Education
    ├── Cultural diversity
    ├── Identity negotiation
    └── Inclusive pedagogy
```

## 2. 최상위 저널

## 4. 핵심 이론 프레임워크

### Policy Borrowing and Lending (Steiner-Khamsi)
```
개념:
- 교육정책의 국가 간 이동
- 맥락에서 분리된 정책 차용
- 선택적 수용과 변형

연구 질문:
- 왜 특정 정책이 차용되는가?
- 어떻게 현지화되는가?
- 누가 이익을 보는가?

질적 접근:
- Policy discourse analysis
- Actor network analysis
- Comparative case study
```

### World Culture Theory (Meyer)
```
핵심 주장:
- 전 세계 교육의 동질화
- 서구 모델의 확산
- 제도적 동형화

비판적 질문:
- 표면적 vs 실제 변화
- 탈결합(decoupling) 현상
- 저항과 재해석

방법론:
- Institutional ethnography
- Document analysis
- Comparative observation
```

### Postcolonial Theory in Education
```
핵심 학자:
- Ngugi wa Thiong'o (언어)
- Spivak (subaltern)
- Bhabha (hybridity)

연구 주제:
- 식민 교육의 유산
- 언어와 권력
- 지식의 식민화
- Indigenous epistemologies

방법론적 함의:
- Researcher positionality 필수
- Voice와 representation
- Decolonizing methodology
```

### Internationalization (Knight)
```
정의:
"목적, 기능, 제공에 국제적/글로벌 차원을 통합하는 과정"

분석 수준:
- Institutional level
- National policy level
- Regional/global level

질적 탐구:
- 의미구성
- 구현 과정
- 이해관계자 경험
- 의도치 않은 결과
```

## 5. 방법론적 접근

### 비교 사례연구
```
설계 유형:
1. Most Similar Systems Design (MSSD)
   - 유사 맥락에서 차이 설명
   - 변이 요인 식별

2. Most Different Systems Design (MDSD)
   - 상이한 맥락에서 유사성 설명
   - 공통 요인 식별

3. Hierarchical Design
   - 국가-지역-학교 수준
   - 다층적 분석

사례 선정 기준:
□ 이론적 관련성
□ 정보의 풍부성
□ 접근 가능성
□ 비교 가능성
```

### 초국가적 민족지 (Multi-sited Ethnography)
```
Marcus (1995) 접근:
"Follow the..."
- People (이주민, 유학생)
- Thing (교과서, 정책 문서)
- Metaphor (담론, 개념)
- Plot (서사, 이야기)
- Life (생애사)
- Conflict (갈등, 논쟁)

적용:
- 유학생의 다중 현장 경험
- 정책의 글로벌 순환
- 교육 상품의 이동
```

### Narrative & Life History
```
적합한 주제:
- 이주 교사의 경험
- 유학생의 정체성 협상
- 교육을 통한 사회이동

분석 접근:
- Thematic narrative analysis
- Structural narrative analysis
- Dialogical narrative analysis
```

## 6. 연구 윤리 특수 고려

### 다문화/국제 맥락 윤리
```
고려사항:
□ 언어와 통역
   - 통역자 훈련
   - 뉘앙스 손실 인식
   - 다국어 보고

□ 권력 불균형
   - Global North 연구자 특권
   - 참여자 취약성
   - 착취적 관계 방지

□ 문화적 민감성
   - 동의 개념의 문화적 차이
   - 기밀성 이해 차이
   - 시간과 관계 개념

□ 상호성(Reciprocity)
   - 연구 이익 공유
   - 역량 강화 기여
   - 지속적 관계 유지
```

### 취약 집단 연구
```
예시 집단:
- 난민/이주민 학생
- 미등록 이주민
- 저개발국 아동
- 분쟁 지역 참여자

추가 보호:
- 익명성 강화
- 데이터 보안
- 트라우마 인식 접근
- 후속 지원 계획
```

## 8. 데이터 구조 예시: 유학생 적응 연구

```
1st Order Concepts
├── "처음엔 수업 따라가기 힘들었어요"
├── "교수님 찾아가기 무서웠어요"
├── "같은 나라 친구들이랑만 어울렸어요"
├── "점점 현지 친구도 생기더라고요"
└── "이제는 두 문화 다 이해해요"
         ↓
2nd Order Themes
├── Academic adjustment challenges
├── Faculty-student cultural gaps
├── Co-national clustering
├── Gradual intercultural contact
└── Bicultural identity development
         ↓
Aggregate Dimensions
├── ACCULTURATIVE STRESS
├── SOCIAL NETWORK EVOLUTION
└── IDENTITY TRANSFORMATION

이론적 통합:
→ Dynamic Acculturation Process Model
```
