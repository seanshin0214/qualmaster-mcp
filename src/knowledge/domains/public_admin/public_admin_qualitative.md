## Public Administration Qualitative Research

## 행정학과 질적연구

### 왜 질적연구가 중요한가?

```
행정 현상의 특성:
├── 맥락 의존성
├── 복잡한 이해관계
├── 정치적 역동
├── 제도적 제약
├── 가치 갈등
└── 민주적 책임

질적연구 가치:
├── 정책 과정 심층 이해
├── 다양한 이해관계자 관점
├── 제도적 맥락
├── 권력 역학
├── 의도하지 않은 결과
└── 정책 실패 원인
```

### 주요 연구 영역

```
1. 정책 과정 (Policy Process)
├── 의제 설정
├── 정책 형성
├── 정책 집행
└── 정책 평가

2. 조직과 관료제
├── 조직 문화
├── 리더십
├── 의사결정
└── 변화 관리

3. 공공 가치와 윤리
├── 공익 정의
├── 윤리적 딜레마
├── 책임성
└── 투명성

4. 시민 참여
├── 거버넌스
├── 공동 생산
├── 시민 경험
└── 민주적 참여

5. 일선 관료제
├── 재량권 행사
├── 시민과의 상호작용
├── 직무 스트레스
└── 전문성
```

## 주요 저널

### Top-tier 저널

```
1. Public Administration Review (PAR)
   ├── IF: ~5

   └── 실무 관련성

2. Journal of Public Administration Research & Theory (JPART)
   ├── IF: ~5

   ├── IF: ~4
   ├── 거버넌스 중심
```

## 이론적 프레임워크

### 1. 제도주의 (Institutionalism)

```
유형:
├── 역사적 제도주의
├── 사회학적 제도주의
├── 합리적 선택 제도주의
└── 담론적 제도주의

질적연구 적용:
├── 제도 형성 과정
├── 제도 변화
├── 경로 의존성
├── 아이디어의 역할
└── 행위자와 구조
```

### 2. 일선 관료제 (Street-Level Bureaucracy)

```
Lipsky (1980):
├── 재량권 (Discretion)
├── 자원 제약
├── 대처 기제
└── 정책-실천 괴리

질적연구 주제:
├── 재량 행사 과정
├── 시민 상호작용
├── 딜레마 경험
├── 대처 전략
└── 정체성
```

### 3. 협력적 거버넌스 (Collaborative Governance)

```
핵심 개념:
├── 다중 이해관계자
├── 숙의 과정
├── 공동 의사결정
└── 네트워크

질적연구 가치:
├── 협력 과정
├── 권력 역학
├── 신뢰 구축
├── 갈등 관리
└── 결과 평가
```

### 4. 공공 가치 이론 (Public Value)

```
Moore (1995):
├── 공공 가치 창출
├── 정당성과 지지
├── 운영 역량
└── 전략적 삼각형

질적연구 적용:
├── 공공 가치 정의
├── 가치 창출 과정
├── 가치 갈등
├── 시민 인식
└── 평가 방법
```

## 권장 방법론

### 1. 정책 사례연구

```
유형:
├── 단일 정책 심층
├── 비교 정책 분석
├── 시간적 변화 추적
└── 다층 분석

데이터:
├── 정책 문서
├── 이해관계자 인터뷰
├── 회의록
├── 미디어 분석
└── 통계 자료
```

### 2. 과정 추적 (Process Tracing)

```
정의:
├── 인과 메커니즘 규명
├── 사건 순서 분석
├── 증거 기반 추론
└── 대안 설명 검증

단계:
├── 가설적 메커니즘 설정
├── 증거 수집
├── 메커니즘 검증
├── 대안 배제
└── 인과 추론
```

### 3. 비교 사례연구

```
설계:
├── 최대 유사 시스템 (MSSD)
├── 최대 차이 시스템 (MDSD)
├── 이중 비교
└── 구성적 비교

분석:
├── 조건 식별
├── 결과 비교
├── 메커니즘 규명
└── 이론적 일반화
```

### 4. 문화기술지

```
적합한 상황:
├── 조직 문화 연구
├── 일선 관료 연구
├── 시민 경험
└── 정책 현장

접근:
├── 장기 현장 연구
├── 참여 관찰
├── 심층 면담
├── 문서 분석
└── 일상 관찰
```

## 분석 접근

### 정책 과정 분석

```
Kingdon 다중흐름:

1. 문제 흐름
├── 문제 정의
├── 지표/사건
└── 피드백

2. 정책 흐름
├── 대안 생성
├── 전문가 공동체
└── 기술적 타당성

3. 정치 흐름
├── 국민 여론
├── 이익집단
└── 정권 변화

4. 정책 창
├── 흐름의 결합
├── 정책 기업가
└── 기회 포착
```

```
접근:
├── 조건 식별 (질적)
├── 사례 비교
├── 패턴 발견
├── QCA 가능 (체계화)
└── 이론적 해석
```

## 글쓰기 전략

### PAR 스타일

```
특징:
├── 실무 관련성 강조
├── 접근성 있는 글쓰기
├── 정책 함의 명확
└── 학제적 청중

구조:
1. Introduction: 정책 문제
2. Literature: 이론적 배경
3. Context: 정책 맥락
4. Method: 연구 설계
5. Findings: 체계적 제시
6. Discussion: 정책 함의
```

### 정책 함의 작성

```
포함 요소:
├── 정책 결정자를 위한
├── 집행 관료를 위한
├── 시민을 위한
├── 제도 개선
└── 향후 정책 방향

표현:
├── 구체적 권고
├── 실행 가능성
├── 맥락 고려
├── 한계 인정
└── 추가 연구 필요성
```
