# 교육정책학 질적연구 완벽 가이드

## 1. 교육정책학의 학문적 기반

### 정책 연구 접근법
```
전통적 접근:
├── 합리주의 (Rationalism)
│   ├── 단계 모형 (Stages Heuristic)
│   ├── 비용-편익 분석
│   └── 증거기반 정책
│
├── 점증주의 (Incrementalism)
│   ├── Lindblom의 "기어가기"
│   ├── 제한된 합리성
│   └── 점진적 변화
│
└── 다중흐름 모형 (Multiple Streams)
    ├── Kingdon의 프레임워크
    ├── 문제, 정책, 정치 흐름
    └── 정책 창 (Policy Window)

비판적 접근:
├── 비판적 정책분석
│   ├── 권력과 이데올로기
│   ├── 담론 분석
│   └── 구조적 불평등
│
├── 포스트구조주의
│   ├── Ball의 정책 사회학
│   ├── 정책으로서의 담론
│   └── 정책 기업가
│
└── 페미니스트 정책분석
    ├── 젠더 렌즈
    ├── 교차성
    └── 돌봄의 정치
```

### Ball의 정책 사이클
```
Policy Cycle (Ball et al.):

1. Context of Influence
   - 정책 담론 형성
   - 이해관계자 영향력
   - 이념적 기반

2. Context of Policy Text Production
   - 공식 문서 작성
   - 협상과 타협
   - 레토릭과 상징

3. Context of Practice
   - 학교 현장 해석
   - 정책 번역 (Translation)
   - 실행의 다양성

4. Context of Outcomes
   - 의도적/비의도적 결과
   - 불평등에 미친 영향
   - 1차/2차 효과

5. Context of Political Strategy
   - 불평등 해소 전략
   - 정치적 행동
   - 사회 정의
```

## 3. 핵심 연구 영역

```

├── 주체 위치 (Subject Positions)
│   └── 정책이 구성하는 주체
├── 진리 레짐 (Regime of Truth)
│   └── 무엇이 지식으로 간주되는가
├── 배제/포함 (Inclusion/Exclusion)
│   └── 누가 말할 수 있는가
└── 권력/지식 관계
    └── 담론과 권력의 연결

적용 주제:
- "학교 선택" 담론
- "책무성" 담론
- "수월성" 담론
- "형평성" 담론

방법론:
- 정책 문서 분석
- 미디어 담론 분석
- 인터뷰 담론 분석
- 역사적 담론 추적
```

## 4. 연구 방법론

### 비판적 정책분석 (CPA)
```
Diem et al. (2014) 프레임워크:

9가지 핵심 개념:
1. 정책 생산의 맥락
2. 정책 과정
3. 정책 결과/영향
4. 분배 효과
5. 정책과 권력 관계
6. 사회 계층화 기여
7. 비주류 집단 목소리
8. 역사적 뿌리
9. 동원 가능성

분석 절차:
1. 정책 텍스트 비판적 읽기
2. 맥락적 배경 조사
3. 권력 관계 분석
4. 결과/영향 추적
5. 대안적 해석 제시
```

### 정책 민족지
```
Shore & Wright 접근:

특성:
- 정책을 "사회적 삶의 창"으로
- 제도와 일상의 연결
- 거버넌스의 인류학

연구 방법:
- 다중현장 민족지
- 정책 입안자 인터뷰
- 현장 실행자 관찰
- 문서 추적

연구 질문:
- 정책은 어떻게 "사실"이 되는가?
- 정책 언어는 어떤 현실을 구성하는가?
- 정책의 의도치 않은 효과는?
```

### WPR 접근법 (What's the Problem Represented to be?)
```
Bacchi의 분석 프레임워크:

6가지 질문:
1. 문제는 어떻게 표상되는가?
2. 이 표상의 전제/가정은?
3. 이 표상은 어떻게 나왔는가?
4. 논쟁되지 않는 것은 무엇인가?
5. 이 표상이 미치는 효과는?
6. 이 표상은 어디서 생산/방어/저항되는가?

적용 예시:
정책: "학업 성취도 격차 해소"

WPR 분석:
Q1: 문제 = 특정 학생들의 낮은 성취
Q2: 전제 = 개인적 결손, 측정 가능
Q3: 역사 = 인적자본론, 경쟁력 담론
Q4: 침묵 = 구조적 불평등, 교육과정
Q5: 효과 = 학생/가족 책임화, 테스트 중심
Q6: 저항 = 교사 운동, 비판적 학자
```

## 5. 주요 정책 이슈 연구

### 신자유주의 교육정책
```
핵심 요소:
├── 시장화 (Marketization)
│   ├── 학교 선택
│   ├── 바우처
│   └── 차터 스쿨
│
├── 민영화 (Privatization)
│   ├── PPP (Public-Private Partnership)
│   ├── 교육 서비스 계약
│   └── 에듀테크 기업
│
├── 책무성 (Accountability)
│   ├── 표준화 시험
│   ├── 성과급
│   └── 학교 평가
│
└── 관리주의 (Managerialism)
    ├── 성과 관리
    ├── 데이터 기반 의사결정
    └── 효율성 담론

비판적 연구 질문:
- 누가 이익을 보는가?
- 어떤 지식이 가치 있게 되는가?
- 교사 전문성에 미치는 영향은?
- 형평성에 미치는 영향은?
```

### 교육 형평성 정책
```
연구 주제:
├── 다문화 교육정책
├── 특수교육 정책
├── 저소득층 지원 정책
├── 농산어촌 교육정책
└── 이주민 교육정책

분석 렌즈:

```

```
1st Order Concepts
├── "위에서 시키니까 하는 거죠"
├── "우리 학교 상황에 안 맞아요"
├── "서류만 늘어나요"

         ↓
2nd Order Themes
├── Compliance without commitment
├── Context mismatch
├── Administrative burden
├── Implementation gap
└── Interpretive variation
         ↓
Aggregate Dimensions
├── SYMBOLIC ADOPTION
├── DECOUPLING
└── LOCALIZED TRANSLATION

이론적 통합:
→ Mediated Policy Enactment Model
```
