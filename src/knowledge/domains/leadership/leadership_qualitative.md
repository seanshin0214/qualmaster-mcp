# 리더십 질적연구 가이드
## Leadership Qualitative Research

---

## 왜 질적연구가 필요한가?

### 리더십 연구의 한계 극복

```
양적연구의 한계:
├── 설문 편향 (Common method bias)
├── 팔로워 인식 의존
├── 맥락 무시
├── 과정 포착 어려움
└── 새로운 현상 측정 한계

질적연구의 장점:
├── 리더십 실제 행동 관찰
├── 맥락적 요인 이해
├── 리더-팔로워 상호작용
├── 시간에 따른 발달
└── 새로운 리더십 형태 탐구
```

### 질적연구가 특히 적합한 주제

```
1. 리더십 개발 (Leadership Development)
├── 어떻게 리더가 되어가는가?
├── 결정적 경험 (Crucible experiences)
├── 정체성 변화
└── 학습 과정

2. 리더십의 어두운 면 (Dark Side)
├── 독성 리더십
├── 나르시시즘
├── 비윤리적 행동
└── 팔로워 영향

3. 맥락적 리더십 (Contextual Leadership)
├── 위기 상황
├── 문화 간 차이
├── 산업 특수성
└── 조직 변화기

4. 새로운 리더십 형태
├── 분산 리더십
├── 공유 리더십
├── 서번트 리더십
├── 진정성 리더십
└── 디지털 리더십
```

## 주요 이론적 프레임워크

### 1. Identity-based Leadership

```
핵심 개념:
├── 리더 정체성 (Leader Identity)
├── 팔로워 정체성
├── 집단 정체성
└── 정체성 작업 (Identity Work)

질적연구 질문:
├── 리더 정체성은 어떻게 형성되는가?
├── 정체성 전환 과정은?
├── 정체성 갈등은 어떻게 해결되는가?
└── 맥락이 정체성에 미치는 영향은?
```

### 2. Leadership as Practice (LAP)

```
핵심 주장:
├── 리더십 = 개인 특성 X
├── 리더십 = 사회적 실천
├── 관계 속에서 출현
└── 맥락에 내재

연구 방법:
├── 민족지학
├── 대화 분석
├── 미시분석
└── 그림자 연구 (Shadowing)
```

### 3. Relational Leadership

```
핵심 아이디어:
├── 관계에서 리더십 구성
├── 사회 구성주의
├── 공동 의미 형성
└── 상호 영향

연구 초점:
├── 리더-팔로워 상호작용
├── 대화와 담론
├── 관계 역동성
└── 권력 협상
```

### 4. Authentic Leadership

```
구성요소:
├── 자기인식
├── 관계적 투명성
├── 균형 잡힌 정보처리
└── 내면화된 도덕

질적연구 접근:
├── 리더 삶의 이야기
├── 결정적 사건 탐구
├── 진정성 경험
└── 발달 과정
```

## 권장 방법론

### 1. 내러티브 연구 (Narrative Approach)

```
적합한 주제:
├── 리더십 발달
├── 결정적 경험
├── 정체성 구성
└── 리더 여정

방법:
├── 생애사 인터뷰
├── 내러티브 분석
├── 스토리텔링
└── 자기 서술

분석:
├── 줄거리 구조
├── 주제 분석
├── 정체성 포지셔닝
└── 문화적 내러티브
```

### 2. 그림자 연구 (Shadowing)

```
정의:
연구자가 리더를 따라다니며 관찰

장점:
├── 실제 행동 관찰
├── 맥락 속 리더십
├── 비공식적 상호작용
└── 암묵적 지식

절차:
├── 종일 또는 여러 날
├── 모든 활동 기록
├── 즉각 메모
├── 추후 인터뷰
└── 문서 수집
```

### 3. 사례연구 (Case Study)

```
유형:

1. 단일 사례 심층
├── 뛰어난 리더
├── 극단적 상황
├── 독특한 맥락
└── 계시적 사례

2. 다중 사례 비교
├── 성공 vs 실패
├── 다른 맥락
├── 다른 스타일
└── 복제 논리
```

### 4. 근거이론 (Grounded Theory)

```
적합한 상황:
├── 기존 이론 부족
├── 새로운 현상
├── 과정 탐구
└── 이론 개발 목표

주의:
├── 진정한 귀납적 접근
├── 지속적 비교
├── 이론적 표본추출
└── 포화까지 진행
```

## 글쓰기 전략

### Leadership Quarterly 스타일

```
특징:
├── 이론적 기여 명확
├── 방법론적 엄격성
├── 풍부한 데이터
└── 실무 함의

구조:
1. Hook: 리더십의 미해결 질문
2. Gap: 기존 연구 한계
3. Method: 상세한 절차
4. Findings: 체계적 제시
5. Discussion: 이론적 확장
```

### 주의사항

```
피해야 할 것:
├── 지나친 영웅화
├── 단순한 묘사
├── 이론 없는 스토리
├── 맥락 무시
└── 비판적 시각 부재

지향해야 할 것:
├── 균형 잡힌 관점
├── 이론적 해석
├── 맥락적 이해
├── 복잡성 인정
└── 비판적 성찰
```
