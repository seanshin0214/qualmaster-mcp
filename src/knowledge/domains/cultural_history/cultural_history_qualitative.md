# 문화사 (Cultural History) 질적연구 완벽 가이드

## 1. 문화사의 학문적 정체성

### 문화사의 전환
```
역사학의 전환들:

1. 사회사에서 문화사로 (Cultural Turn)
   ├── 1970-80년대 전환
   ├── 언어학적 전환 영향
   ├── 문화 개념의 재정의
   └── 의미와 표상 중시

2. 새로운 문화사 (New Cultural History)
   ├── Lynn Hunt (1989)
   ├── 문화의 정치학
   ├── 상징과 의례
   └── 일상생활의 문화

3. 지구사적 전환 (Global Turn)
   ├── 초국가적 접근
   ├── 연결된 역사
   ├── 문화 접촉과 혼종
   └── 비유럽 중심
```

### 핵심 개념
```
문화사의 핵심 개념:

1. 표상 (Representation)
   - 의미 구성 방식
   - 언어와 이미지
   - 권력과의 관계

2. 실천 (Practice)
   - 일상적 행위
   - 체화된 앎
   - 물질문화와의 관계

3. 정체성 (Identity)
   - 개인/집단 정체성
   - 젠더, 인종, 계급
   - 역사적 구성

4. 기억 (Memory)
   - 집단 기억
   - 기념과 망각
   - 역사의 정치
```

## 2. 방법론적 접근

### 미시사 (Microhistory)
```
특성:
├── "이례적 정상" (Exceptional Normal)
├── 단서 패러다임 (Clue Paradigm)
├── 명시적 방법론 성찰
└── 역사적 맥락 복원

대표 학자:
├── Carlo Ginzburg
│   └── 《치즈와 구더기》
├── Natalie Zemon Davis
│   └── 《마르탱 게르의 귀향》
└── Emmanuel Le Roy Ladurie
    └── 《몽타유》

연구 방법:
- 집중적 문서 분석
- 맥락적 재구성
- 상향식 접근
- 미시/거시 연결
```

### 일상사 (Alltagsgeschichte)
```
특성:
├── 평범한 사람들의 역사
├── 일상적 경험
├── 아래로부터의 역사
└── 주관적 경험 중시

주제:
├── 노동과 여가
├── 가족과 젠더
├── 소비와 물질문화
├── 신체와 감정
└── 공간과 이동

방법론:
- 구술사
- 일기/편지 분석
- 사진/영상 분석
- 물질문화 연구
```

### 담론 분석 (Discourse Analysis)
```
Foucault의 영향:

분석 개념:
├── 에피스테메 (Episteme)
├── 담론 형성 (Discursive Formation)
├── 권력/지식
└── 계보학 (Genealogy)

적용:
- 의학사: 광기, 질병 담론
- 성의 역사: 섹슈얼리티 담론
- 감옥의 역사: 규율과 처벌
- 통치성: 자기의 테크놀로지

방법론:
- 텍스트 분석
- 제도 분석
- 실천 분석
- 주체화 분석
```

## 3. 주요 연구 영역

### 시각 문화사
```
연구 대상:
├── 회화와 조각
├── 사진
├── 영화
├── 광고와 포스터
├── 건축과 도시 경관
└── 일상의 시각적 요소

분석 방법:
├── 도상학 (Iconography)
│   └── Panofsky: 3수준 분석
├── 기호학 (Semiotics)
│   └── Barthes: 신화 분석
├── 시각 체제 (Scopic Regime)
│   └── Jay: 시각의 역사성
└── 시각적 전환 (Visual Turn)
    └── Mitchell: 픽처 이론

연구 질문:
- 이미지는 무엇을 "보여주는가"?
- 누가, 왜 이 이미지를 만들었는가?
- 당대에 어떻게 "보여졌는가"?
- 시각 체제는 어떻게 변화하는가?
```

### 신체사 (History of the Body)
```
연구 주제:
├── 신체 관념의 역사
├── 질병과 의료
├── 성과 재생산
├── 노동하는 몸
├── 아름다움의 역사
└── 장애의 역사

이론적 틀:
├── Foucault: 생명권력
├── Butler: 수행적 젠더
├── Bourdieu: 아비투스
└── Merleau-Ponty: 체화

방법론:
- 의학 담론 분석
- 광고/미디어 분석
- 일기/편지 분석
- 물질문화 연구
```

### 감정사 (History of Emotions)
```
핵심 개념:

1. 감정 체제 (Emotional Regime)
   - Reddy의 개념
   - 규범적 감정 표현
   - 감정 고통 (Emotional Suffering)

2. 감정 공동체 (Emotional Community)
   - Rosenwein의 개념
   - 공유된 감정 규범
   - 감정의 사회적 형성

3. 감정 표현 양식 (Emotionology)
   - Stearns 부부 개념
   - 감정에 관한 태도
   - 역사적 변화

연구 방법:
- 일기/서간문 분석
- 문학 텍스트 분석
- 시각 자료 분석
- 어휘 변화 연구
```

## 4. 사료와 자료

### 사료 유형
```
문자 자료:
├── 공식 기록 (관보, 법령)
├── 사적 기록 (일기, 편지)
├── 출판물 (신문, 잡지, 책)
├── 문학 (소설, 시, 희곡)
└── 구술 증언

비문자 자료:
├── 시각 자료 (회화, 사진, 영화)
├── 물질 문화 (유물, 건축)
├── 소리 자료 (음악, 라디오)
├── 지도와 도면
└── 풍경과 공간

디지털 사료:
├── 디지털화된 아카이브
├── 빅데이터 분석
├── 소셜 미디어
└── 웹 아카이브
```

### 사료 비평
```
외적 비평:
├── 진위 (Authenticity)
├── 시기 (Dating)
├── 저자 (Authorship)
└── 출처 (Provenance)

내적 비평:
├── 의도 (Intention)
├── 맥락 (Context)
├── 청중 (Audience)
├── 장르 (Genre)
└── 언어 (Language)

문화사적 읽기:
- 행간 읽기
- 침묵 분석
- 상징 해독
- 맥락 재구성
```

## 6. 연구 설계

### 문화사 연구 단계
```
1. 문제 설정
   - 역사적 질문
   - 현재적 관심
   - 연구 가능성

2. 사료 조사
   - 아카이브 탐색
   - 사료 선별
   - 접근 가능성

3. 사료 분석
   - 비판적 읽기
   - 맥락화
   - 해석

4. 서술
   - 논증 구성
   - 서사 전략
   - 증거 제시

5. 의미화
   - 역사적 의미
   - 현재적 함의
   - 이론적 기여
```

```

분석적 접근:
- 주제별 분석

혼합 접근:
```
