# 이론경제학 질적연구 완벽 가이드

## 1. 경제학 방법론 논쟁

### 경제학의 인식론
```
방법론적 입장:
├── 실증주의 (Positivism)
│   ├── Friedman: 예측력 중심
│   ├── 가정의 현실성 무관
│   └── "as if" 방법론
│
├── 비판적 실재론 (Critical Realism)
│   ├── Lawson: 경제학 비판
│   ├── 인과적 메커니즘 중시
│   └── 폐쇄체계 비판
│
├── 해석주의 (Interpretivism)
│   ├── 행위자의 의미 중시
│   ├── 맥락적 이해
│   └── 질적 방법 옹호
│
└── 다원주의 (Pluralism)
    ├── 다양한 방법 인정
    ├── 문제 중심 접근
    └── 학파 간 대화
```

## 2. 경제학에서의 질적연구

### 역사적 방법
```
경제사 접근:

1. 경제사적 서사
   - 장기 경제 변동
   - 제도 변화 과정
   - 위기와 전환

2. 사례연구
   - 특정 산업/기업 역사
   - 정책 실험
   - 경제 개혁

3. 비교 역사 분석
   - 국가 간 비교
   - 제도 다양성
   - 경로 의존성

4. 구술 경제사
   - 경제 행위자 증언
   - 일상 경제 경험
   - 노동자/기업가 이야기
```

### 제도 분석
```
연구 접근:

1. 규칙 분석 (Rules Analysis)
   - 공식/비공식 규칙
   - 제도의 형성과 변화
   - 제도적 보완성

2. 행위자 분석
   - 제도 기업가
   - 이해관계자 전략
   - 집합 행동

3. 담론 분석
   - 경제 담론의 구성
   - 정책 담론
   - 시장 이데올로기

4. 과정 분석
   - 제도 변화 과정
   - 경로 의존성
   - 임계 국면 (Critical Juncture)

이론적 틀:
├── 역사적 제도주의
├── 비교 정치경제학
├── 조절 이론 (Regulation Theory)
└── 다양한 자본주의 (VoC)
```

## 3. 핵심 연구 영역

### 시장의 사회적 구성
```
Callon/MacKenzie 전통:

핵심 주장:
"경제학이 경제를 수행한다" (Performativity)

연구 주제:
├── 시장 장치 (Market Devices)
│   ├── 가격 결정 메커니즘
│   ├── 거래 기술
│   └── 분류와 범주화
│
├── 계산 행위 (Calculating)
│   ├── 가치 평가
│   ├── 회계와 측정
│   └── 모형과 알고리즘
│
├── 프레이밍과 오버플로우
│   ├── 경계 설정
│   ├── 외부효과
│   └── 시장 실패
│
└── 물질성 (Materiality)
    ├── 기술과 인프라
    ├── 상품의 물질적 특성
    └── 거래의 물질적 조건

방법론:
├── 시장 민족지
├── 과학기술학 (STS) 접근
├── 행위자 연결망 이론
└── 역사적 분석
```

### 돈과 금융의 사회학
