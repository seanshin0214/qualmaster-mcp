# 이중 해석학 (Double Hermeneutic)
## Anthony Giddens와 사회과학 방법론

---

## 개념 정의

### Giddens의 이중 해석학
```
"사회과학자는 이미 의미 있게 구성된 
 사회 세계를 해석한다"

1차 해석: 사회적 행위자들의 해석
├── 일상인들의 의미 부여
├── 상식적 이해
└── 실천적 지식

2차 해석: 사회과학자의 해석
├── 1차 해석에 대한 해석
├── 이론적 개념화
└── 학문적 언어로 번역
```

### 자연과학과의 차이
```
자연과학 (Single Hermeneutic):
연구자 ───해석───→ 자연 현상
                    (의미 없음)

사회과학 (Double Hermeneutic):
연구자 ───해석───→ 행위자의 해석 ───→ 사회 현상
        (2차)        (1차)           (의미 있음)
```

---

## 이론적 배경

### 세 가지 전통의 통합

#### 1. 현상학 (Husserl, Schutz)
```
생활세계 (Lebenswelt):
├── 일상적 경험의 세계
├── 당연시되는 지식
├── 상호주관적 의미
└── 1차 해석의 터전
```

#### 2. 해석학 (Gadamer, Ricoeur)
```
이해의 조건:
├── 선입견의 역할
├── 해석학적 순환
├── 지평 융합
└── 텍스트의 해석
```

#### 3. 비판 이론 (Habermas)
```
비판적 차원:
├── 이데올로기 비판
├── 왜곡된 의사소통
├── 해방적 관심
└── 성찰적 지식
```

## 질적연구에서의 적용

### Smith의 IPA에서의 이중 해석학

```
참여자 → 자신의 경험을 해석 (1차)
    ↓
연구자 → 참여자의 해석을 해석 (2차)
    ↓
독자 → 연구자의 해석을 해석 (3차)

"연구자는 참여자가 자신의 세계를
 이해하려는 것을 이해하려고 한다"
```

### 삼중 해석학으로의 확장
```
          경험
            ↓
    [1차 해석: 참여자]
            ↓
        참여자 이해
            ↓
    [2차 해석: 연구자]
            ↓
        연구자 이해
            ↓
    [3차 해석: 독자/학계]
            ↓
        지식 순환
            ↓
    [영향: 사회로 복귀]
            ↓
        실천 변화
```

## 실천적 함의

### 1. 연구자 위치성
```
연구자는:
├── 참여자의 의미 세계에 접근
├── 자신의 해석 틀 사용
├── 두 세계의 교차점에 위치
└── 번역자/매개자 역할

요구사항:
├── 참여자 관점 존중
├── 이론적 거리 유지
├── 양방향 이해 추구
└── 반성적 자세
```

### 2. 데이터 수집
```
면담에서:
├── 참여자의 1차 해석 탐구
├── 의미 부여 방식 질문
├── "어떻게 이해하시나요?"
├── "그것이 무엇을 의미하나요?"
└── 경험의 의미에 초점

관찰에서:
├── 행위자들의 의미 세계 접근
├── 상황 정의 방식 파악
├── 암묵적 지식 탐구
└── 문화적 맥락 이해
```

### 3. 데이터 분석
```
1차 분석: 참여자 관점
├── "참여자가 이것을 어떻게 이해하는가?"
├── 참여자 언어 사용
├── 경험적 기술
└── 의미 단위 식별

2차 분석: 연구자 해석
├── "이것이 무엇을 의미하는가?"
├── 이론적 개념 도입
├── 패턴과 연결 탐구
└── 심층적 의미 해석

통합:
├── 두 수준의 연결
├── 참여자 언어 → 학술 언어
├── 경험적 → 개념적
└── 기술적 → 해석적
```

### 4. 글쓰기
```
균형 유지:
├── 참여자 목소리 (1차)
│   - 직접 인용
│   - 그들의 언어
│   - 경험 기술
│
└── 연구자 해석 (2차)
    - 분석적 코멘트
    - 이론적 연결
    - 심층 의미

구조:
"참여자 X는 '...'라고 말했다 [1차].
이것은 ...를 보여주며, 
...와 연결된다 [2차]."
```

## 이중 해석학의 복잡성

### 해석의 층위
```
Level 0: 경험 자체 (전반성적)
    ↓
Level 1: 참여자의 즉각적 해석
    ↓
Level 2: 참여자의 반성적 해석 (면담 시)
    ↓
Level 3: 연구자의 초기 이해
    ↓
Level 4: 연구자의 분석적 해석
    ↓
Level 5: 연구자의 이론적 해석
    ↓
Level 6: 학계의 해석 (출판 후)
    ↓
Level 7: 실천으로의 복귀
```

### 해석의 왜곡 가능성
```
각 층위에서:
├── 의미 손실
├── 의미 추가
├── 의미 변형
├── 오해
└── 과잉해석

대응:
├── 참여자 확인 (Member Checking)
├── 삼각화
├── 동료 검토
├── 반성적 실천
└── 투명한 과정 기술
```

## 방법론적 원칙

### 1. 공감적 이해 (Empathic Understanding)
```
참여자 세계로 들어가기:
├── 판단 유보
├── 경험에 열림
├── 참여자 논리 따라가기
└── 그들의 관점에서 보기
```

### 2. 비판적 거리 (Critical Distance)
```
분석적 관점 유지:
├── 이론적 렌즈 적용
├── 당연시된 것 질문
├── 숨겨진 의미 탐구
└── 대안적 해석 고려
```

### 3. 변증법적 통합
```
공감 ←───────→ 비판
  │              │
  └──────┬───────┘
         │
   해석적 이해
         │
    균형점 찾기
```

## 재귀성 (Reflexivity)

### 이중 해석학과 재귀성
```
연구자의 해석이:
├── 참여자에게 영향 (면담 중)
├── 학문 담론에 영향
├── 사회적 실천에 영향
└── 다시 현상에 영향

순환:
학문적 개념 → 일상 언어로 흡수
예: "자아실현", "스트레스", "번아웃"
    원래 학술 개념 → 일상어가 됨
```

### 연구자 재귀성
```
필수 질문:
├── 나의 선입견은 무엇인가?
├── 어떤 이론적 렌즈를 사용하는가?
├── 나의 위치가 해석에 어떤 영향?
├── 대안적 해석은 무엇인가?
└── 내 해석의 한계는?
```

## 분야별 적용

### 조직연구
```
1차 해석: 조직 구성원들의 의미 부여
├── 조직 문화 이해
├── 역할 정의
├── 성공/실패 귀인
└── 정치적 해석

2차 해석: 연구자의 분석
├── 제도 이론
├── 권력 분석
├── 정체성 이론
└── 센스메이킹
```

### 교육연구
```
1차 해석: 교사/학생의 경험 이해
├── 교실 상황 정의
├── 학습 경험의 의미
├── 관계의 이해
└── 성공/좌절 경험

2차 해석: 연구자의 분석
├── 교육학 이론
├── 발달 심리학
├── 사회화 이론
└── 권력/저항
```

### 의료/간호 연구
```
1차 해석: 환자/의료인의 경험
├── 질병 경험
├── 돌봄의 의미
├── 신체와 자아
└── 회복의 의미

2차 해석: 연구자의 분석
├── 현상학적 해석
├── 사회학적 분석
├── 의료인류학
└── 젠더/권력
```

## 품질 기준

### 1차 해석 충실성
```
- 참여자 관점 정확히 포착?
- 그들의 언어 사용?
- 맥락 고려?
- 왜곡 최소화?
```

### 2차 해석 적절성
```
- 이론적 연결 타당?
- 분석 깊이 충분?
- 통찰 제공?
- 학문적 기여?
```

### 통합의 균형
```
- 두 수준 연결?
- 긴장 인정?
- 투명한 과정?
- 반성적 자세?
```
