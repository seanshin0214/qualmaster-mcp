# Gadamer의 철학적 해석학 (Philosophical Hermeneutics)
## Hans-Georg Gadamer (1900-2002)

## 핵심 저작
- **Truth and Method (Wahrheit und Methode, 1960)**: 해석학의 금자탑
- **Philosophical Hermeneutics (1976)**: 에세이 모음
- **Reason in the Age of Science (1981)**

---

## 핵심 개념 체계

### 1. 선입견 (Vorurteil / Prejudice)

#### 계몽주의 비판
```
계몽주의: 선입견 = 극복해야 할 오류
Gadamer: 선입견 = 이해의 필수 조건

"선입견은 이해의 장애물이 아니라 이해의 조건이다"
```

#### 정당한 선입견 vs 부당한 선입견
```
정당한 선입견 (Legitimate Prejudice):
├── 전통에서 비롯된 것
├── 검증을 통해 정당화된 것
└── 이해를 가능하게 하는 것

부당한 선입견 (Illegitimate Prejudice):
├── 맹목적 권위에서 비롯된 것
├── 성급한 판단
└── 이해를 왜곡하는 것
```

#### 연구 적용
```
연구자의 선입견:
1. 학문적 훈련에서 온 관점
2. 이론적 프레임워크
3. 방법론적 가정
4. 개인적 경험

→ 배제가 아닌 "자각"이 핵심
→ Reflexivity(반성성)의 철학적 근거
```

### 2. 해석학적 순환 (Hermeneutic Circle)

#### 전통적 해석학적 순환
```
부분 ←→ 전체

텍스트의 개별 문장 이해 → 전체 맥락 필요
전체 맥락 이해 → 개별 문장 이해 필요
```

#### Gadamer의 확장: 존재론적 순환
```
          해석자
         ↗    ↖
        ↙      ↘
    텍스트 ←→ 전통
        ↘      ↗
         ↖    ↙
          의미

해석자의 선입견(전통) → 텍스트 접근
텍스트 → 선입견 수정/확인
수정된 이해 → 새로운 접근
→ 무한 순환 (but 발전적)
```

#### 질적연구에서의 순환
```
1차 순환: 데이터 ↔ 코드
├── 개별 인용문 → 코드 생성
└── 코드 체계 → 인용문 재해석

2차 순환: 코드 ↔ 주제
├── 코드 모음 → 주제 도출
└── 주제 → 코드 재검토

3차 순환: 주제 ↔ 전체 이해
├── 주제들 → 전체 해석
└── 전체 해석 → 주제 재정의

메타 순환: 연구자 ↔ 현상
├── 연구자 이해 → 현상 접근
└── 현상 → 연구자 변화
```

### 3. 지평융합 (Fusion of Horizons / Horizontverschmelzung)

#### 지평(Horizon)의 의미
```
지평 = 특정 관점에서 볼 수 있는 모든 것을 포함하는 범위

개인의 지평:
├── 역사적 위치
├── 문화적 배경
├── 언어적 전통
├── 경험의 총체
└── 선입견의 집합
```

#### 지평융합의 과정
```
연구자 지평          참여자 지평
    │                    │
    └───────┬───────────┘
            │
      [대화/만남]
            │
    ┌───────┴───────┐
    │               │
  확장된         확장된
 연구자 지평    참여자 지평
    │               │
    └───────┬───────┘
            │
      [융합된 지평]
            │
      새로운 이해
```

#### 연구에서의 지평융합
```
수준 1: 연구자-참여자 지평융합
- 면담에서의 상호 이해
- 공동 의미 구성
- 양측 모두의 변화

수준 2: 연구자-텍스트 지평융합
- 전사본/문서 해석
- 과거와 현재의 대화
- 텍스트의 의미 실현

수준 3: 연구자-전통 지평융합
- 이론적 전통과의 대화
- 선행연구와의 연결
- 학문 공동체에의 기여
```

### 4. 작용사적 의식 (Wirkungsgeschichtliches Bewußtsein)

#### 개념
```
Effective Historical Consciousness
= 역사가 우리에게 미치는 영향에 대한 인식

우리는 역사 안에 있다:
├── 역사를 객관적으로 볼 수 없음
├── 역사가 우리를 형성함
├── 우리의 이해도 역사적
└── 완전한 자기이해 불가능
```

#### 연구 함의
```
연구자의 위치성(Positionality):
1. 학문적 전통에 의해 형성됨
2. 시대적 담론에 영향받음
3. 제도적 맥락에 놓여있음
4. 이 모든 것을 완전히 벗어날 수 없음

→ 겸손한 지식 주장
→ 해석의 잠정성 인정
→ 대안적 해석 열어두기
```

---

### 5. 언어와 이해

#### 언어의 존재론적 지위
```
"이해될 수 있는 존재는 언어이다"
(Being that can be understood is language)

언어 ≠ 도구
언어 = 이해의 매체이자 한계

우리는 언어 안에서 사고한다
→ 언어가 세계를 열어준다
→ 동시에 언어가 이해를 제약한다
```

#### 대화(Gespräch)의 중요성
```
진정한 대화의 특징:
1. 결과를 미리 알 수 없음
2. 대화가 참여자를 이끌어감
3. 상호 변화가 일어남
4. 새로운 이해가 창발함

면담 = 대화:
├── 반구조화: 열린 대화 허용
├── 경청: 참여자가 이끌도록
├── 유연성: 예상치 못한 방향 허용
└── 상호성: 연구자도 변화
```

## Gadamer 해석학의 질적연구 적용

### 방법론적 함의

#### 1. 연구 설계
```
전통적 접근:
"편견을 배제하고 객관적으로 접근하라"

Gadamerian 접근:
"선입견을 자각하고 검증하며 연구하라"

실천:
- 연구 시작 전 선입견 기록
- 연구 과정에서 지속적 반성
- 선입견의 변화 추적
- 최종 해석에 영향 논의
```

#### 2. 데이터 수집
```
면담 = 대화로서:
├── 연구자도 참여자로부터 배움
├── 예상치 못한 방향 허용
├── 상호 이해 추구
└── 권력 관계 성찰

관찰 = 지평 확장으로서:
├── 연구자의 지평 한계 인식
├── 참여자 세계로 들어감
├── 점진적 이해 심화
└── 자기 변화 허용
```

#### 3. 데이터 분석
```
해석학적 순환 적용:

1단계: 전체 읽기
   - 전체적 인상 형성
   - 선입견 자각

2단계: 부분 분석
   - 세부 코딩
   - 전체 맥락 참조

3단계: 전체 재구성
   - 부분에서 전체로
   - 이해 심화

4단계: 새로운 부분 분석
   - 변화된 이해로 재접근
   - 순환 반복

5단계: 융합
   - 연구자-데이터 지평 융합
   - 새로운 이해 도출
```

#### 4. 글쓰기
```
해석의 잠정성 인정:
- "이 해석은..." 표현
- 대안적 해석 언급
- 연구자 위치성 명시
- 지평의 한계 인정

BUT: 약한 주장이 아님
- 충분한 근거 제시
- 논리적 일관성
- 설득력 있는 논증
```

## 비판과 대응

### 주요 비판

#### 1. 상대주의 비판
```
비판: "모든 해석이 동등하다는 것인가?"

Gadamer 대응:
- 해석은 전통에 의해 제약됨
- 텍스트가 해석을 제한함
- 대화에서 검증됨
- 더 나은/적절한 해석 있음
```

#### 2. 보수주의 비판 (Habermas)
```
비판: "전통을 무비판적으로 수용하는가?"

Gadamer 대응:
- 전통 내에서 비판 가능
- 반성적 거리 가능
- 하지만 전통 밖은 불가능

연구적 함의:
- 비판적 관점 유지 가능
- 하지만 완전한 외부자 시점 환상
```

#### 3. 방법 무시 비판
```
비판: "방법 없이 어떻게 학문인가?"

대응:
- 방법 반대가 아님
- 방법만능주의 반대
- 진리는 방법 너머에도 있음
- 해석의 기예(art) 중요
```

