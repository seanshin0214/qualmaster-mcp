# 미학 (Aesthetics) 질적연구 완벽 가이드

## 1. 미학의 철학적 기초

### 미학의 핵심 문제
```
미학의 주요 질문:
├── 예술이란 무엇인가? (정의 문제)
│   ├── 재현론 (Representation)
│   ├── 표현론 (Expression)
│   ├── 형식주의 (Formalism)
│   └── 제도론 (Institutional Theory)
│
├── 미적 경험이란 무엇인가?
│   ├── 무관심성 (Disinterestedness)
│   ├── 미적 태도 (Aesthetic Attitude)
│   └── 미적 속성 (Aesthetic Properties)
│
├── 예술적 가치는 어디서 오는가?
│   ├── 내재적 가치 vs 외재적 가치
│   ├── 인지적 vs 비인지적 가치
│   └── 도덕적 가치와의 관계
│
└── 미적 판단은 객관적인가?
    ├── Kant: 무관심적 만족
    ├── Hume: 이상적 비평가
    └── 상대주의 vs 실재론
```

### 핵심 이론가
```
고전 미학:
├── Plato: 예술은 모방의 모방
├── Aristotle: 카타르시스, 미메시스
├── Kant: 미적 판단력 비판
│   ├── 무관심성 (Disinterestedness)
│   ├── 목적 없는 합목적성
│   ├── 주관적 보편성
│   └── 숭고 (Sublime)
└── Hegel: 예술사의 변증법

현대 미학:
├── Dewey: 경험으로서의 예술
├── Goodman: 예술의 언어
├── Danto: 예술계 (Artworld)
├── Dickie: 제도론
└── Walton: 범주와 관례
```

## 2. 예술 정의 이론

### 주요 이론
```
1. 재현론 (Representation Theory)
   - 예술은 현실의 모방
   - 문제: 비재현적 예술 설명 불가

2. 표현론 (Expression Theory)
   - Collingwood: 감정의 표현
   - Croce: 직관과 표현
   - 문제: 모든 표현이 예술인가?

3. 형식주의 (Formalism)
   - Bell: 의미 있는 형식
   - 순수한 형식적 속성이 중요
   - 문제: 맥락 무시

4. 제도론 (Institutional Theory)
   - Dickie: 예술계의 승인
   - 사회적 맥락 강조
   - 문제: 순환성

5. 역사적 정의 (Historical Definition)
   - Levinson: 의도 역사
   - 예술 전통과의 관계
   - 문제: 급진적 혁신 설명
```

### 미적 경험 연구
```
현상학적 접근:
├── Husserl: 지향성과 미적 대상
├── Heidegger: 예술작품의 근원
├── Merleau-Ponty: 체화된 지각
└── Dufrenne: 미적 경험의 현상학

연구 방법:
- 깊은 현상학적 인터뷰
- 예술 감상 관찰
- 예술가 창작 과정 연구
- 미술관/갤러리 민족지
```

## 3. 예술 비평과 해석

### 해석 이론
```
1. 의도주의 (Intentionalism)
   - 작품 의미 = 작가 의도
   - Hirsch: 유효한 해석
   - 실제 의도 vs 가상 의도

2. 반의도주의 (Anti-intentionalism)
   - Wimsatt & Beardsley: 의도 오류
   - 텍스트 자체가 의미 결정
   - 작가의 죽음 (Barthes)

3. 다원주의 (Pluralism)
   - 다중 해석 가능
   - Margolis: 상대적 타당성
   - 해석 공동체

4. 맥락주의 (Contextualism)
   - 역사적/문화적 맥락
   - 사회적 의미
   - 수용미학
```

### 비평 실천 연구
```
연구 질문:
├── 비평가는 어떻게 판단하는가?
├── 비평적 글쓰기의 수사학
├── 비평 담론의 권력 관계
└── 비평과 예술 시장

방법론:
- 비평문 담론 분석
- 비평가 인터뷰
- 비평 역사 연구
- 제도 분석
```

## 4. 예술사회학

### Bourdieu의 접근
```
핵심 개념:
├── 문화자본 (Cultural Capital)
│   └── 예술 감상 능력
├── 장 (Field)
│   └── 예술장의 자율성
├── 아비투스 (Habitus)
│   └── 취향의 사회적 형성
└── 구별짓기 (Distinction)
    └── 취향과 계급

연구 주제:
- 미술관 관람 패턴
- 예술교육과 계급
- 취향의 사회화
- 예술가의 사회적 위치
```

### 예술계 연구
```
Becker의 "Art Worlds":
예술 = 협력적 네트워크의 산물

분석 대상:
├── 예술가
├── 딜러/큐레이터
├── 비평가
├── 수집가
├── 관객
├── 지원 인력
└── 제도

방법론:
- 예술계 민족지
- 네트워크 분석
- 경력 경로 연구
- 제도 분석
```

## 5. 예술창작 연구

### 창작 과정 연구
```
연구 접근:

1. 심리학적 접근
   - 창의성 연구
   - 영감과 노력
   - 몰입 (Flow)

2. 현상학적 접근
   - 창작 경험의 본질
   - 체화된 앎
   - 재료와의 대화

3. 사회문화적 접근
   - 협력적 창작
   - 문화적 맥락
   - 제도적 제약

방법론:
- 예술가 심층 인터뷰
- 작업실 관찰
- 스케치/초고 분석
- 자전적 자료 분석
```

### 예술 기반 연구 (Arts-Based Research)
```
유형:
├── A/r/tography
│   └── 예술가-연구자-교사
├── Performative Inquiry
│   └── 수행적 탐구
├── Poetic Inquiry
│   └── 시적 표현
└── Visual Methodology
    └── 시각적 방법론

원칙:
- 예술적 과정이 연구 방법
- 미적 형식이 지식 표현
- 체화된 앎 강조
- 감각적 경험 중시
```
