# 정의론 (Theory of Justice) 질적연구 가이드

## 1. 정의론의 철학적 기초

### 주요 정의 이론
```
1. 자유주의적 평등주의
   └── Rawls: 정의론
       ├── 원초적 입장 (Original Position)
       ├── 무지의 베일 (Veil of Ignorance)
       ├── 두 정의 원칙
       │   ├── 평등한 자유 원칙
       │   └── 차등 원칙 (Difference Principle)
       └── 중첩적 합의 (Overlapping Consensus)

2. 자유지상주의 (Libertarianism)
   └── Nozick: 소유권 이론
       ├── 취득의 정의
       ├── 이전의 정의
       └── 교정의 정의

3. 공동체주의 (Communitarianism)
   ├── MacIntyre: 덕 윤리와 전통
   ├── Sandel: 자아와 공동체
   ├── Taylor: 인정과 정체성
   └── Walzer: 영역들의 정의

4. 역량 접근 (Capability Approach)
   ├── Sen: 자유로서의 발전
   └── Nussbaum: 핵심 역량 목록

5. 비판적 정의론
   ├── Young: 억압의 다섯 얼굴
   ├── Fraser: 재분배와 인정
   └── Honneth: 인정투쟁
```

### 정의의 영역
```
분배적 정의:
├── 자원 배분
├── 기회의 평등
├── 결과의 평등
└── 필요 기반 분배

절차적 정의:
├── 공정한 절차
├── 의사결정 참여
├── 투명성
└── 책임성

인정의 정의:
├── 정체성 존중
├── 문화적 인정
├── 사회적 지위
└── 상호 존중

회복적 정의:
├── 피해 회복
├── 관계 복원
├── 공동체 치유
└── 화해
```

## 2. Rawls 정의론 심화

### 원초적 입장과 무지의 베일
```
사고실험:

원초적 입장 (Original Position):
- 자유롭고 평등한 도덕적 인격체
- 상호 무관심 (Mutual Disinterest)
- 합리적 선택

무지의 베일 (Veil of Ignorance):
차단되는 정보:
├── 사회적 지위
├── 자연적 재능
├── 인종, 성별, 민족
├── 선의 구상 (Conception of Good)
├── 세대
└── 심리적 특성

허용되는 정보:
├── 일반적 사회 사실
├── 정치/경제 이론
├── 인간 심리 일반
└── 정의의 상황

질적연구 적용:
- 시민들의 정의 직관 연구
- 사고실험 반응 분석
- 문화 간 비교 연구
```

### 두 정의 원칙
```
제1원칙 (평등한 자유):
"각 인격체는 모든 사람에게 적용되는 
동등한 기본 자유 체계와 양립하는
평등한 기본 자유 체계에 대한 
동등한 권리를 가진다."

제2원칙 (차등 원칙):
a) 최소 수혜자에게 최대 이익
b) 공정한 기회균등 조건 하에
   모든 직위가 모두에게 개방

우선성 규칙:
1. 자유의 우선성
2. 정의의 효율성에 대한 우선성
```

## 3. 인정의 정치학

### Fraser의 이원론적 정의
```
두 차원:
1. 재분배 (Redistribution)
   - 경제적 불의
   - 착취, 배제, 박탈
   - 해결: 경제 재구조화

2. 인정 (Recognition)
   - 문화적 불의
   - 지배, 비인정, 무시
   - 해결: 문화적 재평가

관계:
├── 상호 환원 불가
├── 동시 발생 가능
├── 긴장 관계 존재
└── 참여 동등성 (Participatory Parity)

적용 사례:
- 성별: 재분배 + 인정
- 인종: 재분배 + 인정
- 성적 지향: 주로 인정
- 계급: 주로 재분배
```

### Honneth의 인정투쟁
```
세 가지 인정 양식:

1. 사랑 (Love)
   - 친밀한 관계
   - 자기 신뢰 (Self-confidence)
   - 무시 형태: 학대, 폭력

2. 법 (Law)
   - 권리로서의 인정
   - 자기 존중 (Self-respect)
   - 무시 형태: 권리 박탈

3. 연대 (Solidarity)
   - 사회적 가치 평가
   - 자기 존중 (Self-esteem)
   - 무시 형태: 모욕, 비하

질적연구 주제:
- 인정 경험의 현상학
- 무시 경험의 영향
- 인정투쟁의 동학
```

## 4. 질적연구 방법론

### 정의 경험 연구
```
현상학적 접근:
"정의/불의를 경험한다는 것은 무엇인가?"

연구 질문:
├── 사람들은 정의를 어떻게 경험하는가?
├── 불의 경험은 어떻게 인식되는가?
├── 정의 추구 동기는 무엇인가?
└── 문화에 따라 정의 개념은 다른가?

방법론:
- 심층 현상학적 인터뷰
- 내러티브 탐구
- 생애사 연구
- 민족지적 관찰
```

### 정의 담론 분석
```
분석 초점:

1. 제도적 담론
   - 법원 판결문
   - 정책 문서
   - 입법 논쟁

2. 공적 담론
   - 미디어 논의
   - 사회운동 담론
   - 정치적 수사

3. 일상 담론
   - 일상 대화
   - 소셜 미디어
   - 개인 내러티브

방법론:
- 비판적 담론 분석
- 프레임 분석
- 수사학적 분석
- 텍스트 분석
```

### 정의 실천 연구
```
연구 영역:

1. 사법 정의
   - 법정 민족지
   - 판결 과정 연구
   - 배심원 의사결정

2. 회복적 정의
   - 프로그램 평가
   - 참여자 경험
   - 과정 관찰

3. 사회운동
   - 정의 프레이밍
   - 동원 과정
   - 정체성 정치

4. 조직적 정의
   - 절차적 공정성
   - 분배적 공정성
   - 상호작용적 공정성
```
