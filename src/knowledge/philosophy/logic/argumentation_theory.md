# 논증이론 (Argumentation Theory) 완벽 가이드

## 1. 논증이론의 학문적 위치

### 정의와 범위
```
논증이론의 학제적 성격:
├── 논리학 (Logic)
│   └── 타당성, 형식적 구조
├── 수사학 (Rhetoric)
│   └── 설득, 청중 적응
├── 변증론 (Dialectic)
│   └── 대화, 비판적 검토
└── 화용론 (Pragmatics)
    └── 맥락, 언어 사용
```

### 핵심 학파
```
1. Pragma-Dialectics (Amsterdam 학파)
   - van Eemeren & Grootendorst
   - 비판적 토론의 규범 이론
   - 오류를 규칙 위반으로 분석

2. Informal Logic (Windsor 학파)
   - Johnson & Blair
   - 일상 논증 분석
   - 좋은 논증의 기준: RSA

3. New Rhetoric
   - Perelman & Olbrechts-Tyteca
   - 청중 중심 접근
   - 보편 청중 개념

4. Toulmin Model
   - Stephen Toulmin
   - 필드 의존적 논증
   - 6요소 모형
```

## 2. Pragma-Dialectics 심화

### 비판적 토론의 4단계
```
1. Confrontation Stage (대립 단계)
   - 의견 차이 확인
   - 쟁점 명확화
   - 역할 배분 (주창자/비판자)

2. Opening Stage (개시 단계)
   - 출발점 합의
   - 토론 규칙 합의
   - 입증 책임 배분

3. Argumentation Stage (논증 단계)
   - 논증 제시
   - 비판적 질문
   - 반박과 재반박

4. Concluding Stage (결론 단계)
   - 결과 확인
   - 의견 차이 해소 여부 평가
   - 주창 철회 또는 비판자 양보
```

### 논증 도식 (Argumentation Schemes)
```
인과 논증 (Causal Argumentation):
- 사건 A가 사건 B를 야기했다
- 비판적 질문: 다른 원인은 없는가?

유비 논증 (Argument from Analogy):
- A와 B가 속성 X, Y, Z를 공유한다
- A는 속성 W를 가진다
- 따라서 B도 속성 W를 가진다
- 비판적 질문: 관련된 차이점은?

권위 논증 (Argument from Authority):
- E는 영역 D의 전문가이다
- E는 P를 주장한다
- P는 영역 D에 속한다
- 따라서 P는 참이다
- 비판적 질문: E가 편향되어 있지는 않은가?

결과 논증 (Argument from Consequences):
- 행위 A가 결과 C를 가져온다
- C는 좋다/나쁘다
- 따라서 A를 해야/하지 말아야 한다
- 비판적 질문: C가 정말 발생하는가?
```

### 전략적 조작 (Strategic Maneuvering)
```
세 가지 측면:

1. Topic Potential (주제 잠재력)
   - 논점 선택
   - 유리한 프레이밍
   - 의제 설정

2. Audience Demand (청중 요구)
   - 청중 분석
   - 공유 전제 활용
   - 설득력 극대화

3. Presentational Devices (표현 장치)
   - 수사적 기법
   - 언어적 선택
   - 강조와 축소

탈선 (Derailment):
전략적 조작이 합리성 규범을 위반할 때
→ 오류 (fallacy)로 전환
```

## 3. Walton의 논증 도식

### 주요 도식 유형
```
1. 추정 논증 (Argument from Position to Know)
   A는 P를 알 위치에 있다
   A는 P라고 진술한다
   따라서 P는 참이다

2. 전문가 의견 (Argument from Expert Opinion)
   E는 영역 D의 전문가이다
   E는 명제 A가 참이라고 주장한다
   A는 영역 D 내의 명제이다
   따라서 A는 참이다

3. 일반적 관행 (Argument from Popular Practice)
   대부분의 사람이 X를 한다
   따라서 X를 하는 것이 합리적이다

4. 약속 (Argument from Commitment)
   A는 명제 P에 동의했다
   P는 Q를 함축한다
   따라서 A는 Q에도 동의해야 한다

5. 규칙 적용 (Argument from Rules)
   규칙 R이 있다: 조건 C가 충족되면 결론 Q
   조건 C가 충족된다
   따라서 결론 Q가 따른다
```

### 대화 유형
```
1. 설득 대화 (Persuasion Dialogue)
   - 의견 차이 해소
   - 입증 책임

2. 탐구 대화 (Inquiry Dialogue)
   - 진리 탐구
   - 공동 증거 평가

3. 협상 대화 (Negotiation Dialogue)
   - 이해 조정
   - 합의 도출

4. 정보 탐색 대화 (Information-Seeking)
   - 지식 획득
   - 질문-답변

5. 심의 대화 (Deliberation Dialogue)
   - 행동 결정
   - 최선의 행위 모색

6. 논쟁 대화 (Eristic Dialogue)
   - 승리 목적
   - 상대 굴복
```

## 4. 수사학적 논증

### Perelman의 신수사학
```
핵심 개념:

1. 청중 (Audience)
   - 특수 청중 (Particular Audience)
   - 보편 청중 (Universal Audience)
   - 자기 자신 (Self)

2. 출발점 (Starting Points)
   - 사실과 진리
   - 추정과 가치
   - 위계와 선호

3. 연합 기법 (Association Techniques)
   - 준논리적 논증 (Quasi-Logical)
   - 현실 구조 기반 논증
   - 현실 구조 확립 논증

4. 해리 기법 (Dissociation Techniques)
   - 개념 분리
   - 현상/실재 구분
   - 재정의
```

### 수사학적 상황 (Rhetorical Situation)
```
Bitzer의 모형:

1. Exigence (긴급성)
   - 수사적 담화를 요청하는 문제
   - 담화로 수정 가능해야 함

2. Audience (청중)
   - 변화의 매개자
   - 결정과 행동 능력

3. Constraints (제약)
   - 청중과 상황에 영향
   - 기존 신념, 태도, 전통
```

## 5. 질적연구 적용

### 논증 분석 방법론
```
1단계: 논증 재구성
├── 논증 요소 식별
├── 암묵적 전제 복원
└── 논증 구조 도식화

2단계: 논증 평가
├── 형식적 타당성
├── 전제의 수용가능성
├── 결론의 적절성
└── 오류 식별

3단계: 맥락 분석
├── 대화 유형 파악
├── 제도적 맥락
├── 참여자 역할
└── 권력 관계

4단계: 해석
├── 전략적 의도
├── 사회문화적 함의
├── 비판적 평가
```

### 코딩 체계 예시
```
논증 요소 코딩:
├── CLAIM: 주장
├── DATA: 근거/증거
├── WARRANT: 보증
├── BACKING: 뒷받침
├── QUALIFIER: 한정어
├── REBUTTAL: 반박/예외
└── IMPLICIT: 암묵적 전제

논증 도식 코딩:
├── CAUSE: 인과 논증
├── ANALOGY: 유비 논증
├── EXPERT: 전문가 논증
├── EXAMPLE: 예시 논증
├── SIGN: 징후 논증
└── VALUE: 가치 논증

대화 행위 코딩:
├── ASSERT: 주장하기
├── QUESTION: 질문하기
├── CHALLENGE: 도전하기
├── DEFEND: 방어하기
├── CONCEDE: 양보하기
└── RETRACT: 철회하기
```
