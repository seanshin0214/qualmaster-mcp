# AMJ Grounded Theory 성공 사례 분석

## 사례: Gioia et al. (2010) AMJ 출판 여정

### 논문 정보
- **제목**: "Where do we go from here? Toward a theory of organizational identity transformation"
- **저널**: Academy of Management Journal
- **핵심 기여**: 조직 정체성 변화의 과정 이론 개발

## 1. 성공 요인 분석

### 1.1 Compelling Phenomenon
```
Hook 구조:
"Organizations sometimes face the challenge of
transforming who they are..." 

왜 성공했나:
✓ 보편적 관심사 (정체성 변화)
✓ 경영 실무 관련성 높음
✓ 기존 이론의 한계 명확
✓ "How" 질문으로 과정 초점
```

### 1.2 이론적 Gap 정의
```
기존 연구:
- 정체성 "유지" 연구 풍부
- 변화 시 어떻게 되는가?
- 과정 연구 부재

Gap Framing:
"We know much about identity maintenance,
but surprisingly little about how
organizations actually transform their
identities..."
```

### 1.3 방법론적 엄밀성
```
Gioia Methodology 정립:
├── 1st Order Concepts (참여자 언어)
├── 2nd Order Themes (연구자 해석)
└── Aggregate Dimensions (이론)

데이터 구조표:
- 시각적으로 명확
- 귀납적 과정 증명
- 재현 가능성 시사
```

## 2. 논문 구조 분석

### Introduction (2-3 pages)
```
Paragraph 1: Phenomenon introduction
- 조직 정체성 변화의 도전

Paragraph 2: What we know
- 정체성 이론 요약

Paragraph 3: What we don't know (GAP)
- 변화 과정 미해명

Paragraph 4: What this study does
- "We develop a grounded theory of..."

Paragraph 5: Contributions preview
- 세 가지 기여 예고
```

### Theoretical Background (4-5 pages)
```
균형 잡힌 문헌검토:
- 너무 많지 않게 (grounded theory)
- 핵심 개념 정의
- "Sensitizing concepts" 역할
- Gap 명확화

Sensitizing Concepts:
- Organizational identity
- Identity threat
- Sensemaking
- Sensegiving
```

### Methods (3-4 pages)
```
Research Context:
- 왜 이 조직인가?
- 변화 상황의 적합성

Data Sources:
- 인터뷰 (N=40)
- 관찰
- 문서 자료
- 삼각검증

Analysis Approach:
- 1st order analysis
- 2nd order analysis
- Aggregate dimensions
- Theoretical saturation

Trustworthiness:
- Informant validation
- Multiple investigators
- Audit trail
```

### Findings (10-12 pages)
```
구조:
1. Data Structure Table (핵심!)
2. 각 Aggregate Dimension 설명
3. 풍부한 인용문 지원
4. 과정 모델 도출

Data Structure 예시:
1st Order → 2nd Order → Aggregate
"우리가 누군지 다시 생각해야 했어요"
→ Identity questioning → IDENTITY DISRUPTION

서술 원칙:
- 인용문으로 시작
- 연구자 해석 제시
- 개념 간 관계 설명
```

### Discussion (4-5 pages)
```
Contribution 명시:
1. Process model of identity transformation
2. Concept of "liminal identity"  
3. Role of sensegiving in change

Boundary Conditions:
- 어떤 조직에 적용 가능?
- 어떤 조건에서?

Future Research:
- 구체적 제안
- 이론 확장 방향
```

## 4. 핵심 성공 요소

### 연구 설계
```
✓ Phenomenon 선택의 적절성
  - 이론적으로 중요
  - 경험적으로 접근 가능
  - 과정 관찰 가능

✓ 맥락의 풍부함
  - 변화가 진행 중인 조직
  - 접근 용이
  - 다양한 데이터 원천
```

### 분석 전략
```
✓ 체계적 귀납
  - 명확한 단계
  - 투명한 과정
  - 재현 가능성

✓ 시각화
  - Data structure table
  - Process model
  - 한눈에 이해 가능
```

### 글쓰기
```
✓ Compelling narrative
  - 첫 문장부터 흡입력
  - 논리적 흐름
  - 깔끔한 문장

✓ Balance
  - 이론과 데이터 균형
  - 인용문과 해석 균형
  - 엄밀성과 가독성 균형
```
