# 교육학 저널 질적연구 투고 가이드
## Education Journals for Qualitative Research

---

## Top-tier 교육학 저널

### 1. American Educational Research Journal (AERJ)

```
기본 정보:
├── 출판사: AERA (Sage)
├── IF: ~5
├── 게재율: ~10%
├── 질적연구: 환영

섹션:
├── Social and Institutional Analysis
├── Teaching, Learning, and Human Development
└── 각 섹션 별도 심사

질적연구 기대:
├── 방법론적 엄격성
├── 이론적 기여
├── 교육 실무 함의
└── 일반화 가능성 논의

구조:
1. Introduction: 교육적 중요성
2. Theoretical Framework
3. Methods: 상세한 절차
4. Findings
5. Discussion: 교육적 함의
```

### 2. Review of Educational Research (RER)

```
기본 정보:
├── 출판사: AERA (Sage)
├── IF: ~11 (매우 높음)
├── 유형: 리뷰 논문 전문

유형:
├── Systematic Review
├── Meta-analysis
├── Meta-synthesis (질적)
├── Integrative Review
└── Critical Review

질적 메타합성 가능:
├── 질적연구 종합
├── 방법론적 엄격성 필요
├── ENTREQ 가이드라인
└── 이론적 통합
```

### 3. Educational Researcher (ER)

```
기본 정보:
├── 출판사: AERA (Sage)
├── IF: ~6
├── 형식: 짧은 논문 (8,000 단어)
├── 광범위 독자층

특징:
├── 논쟁적 주제 환영
├── 정책 관련성
├── 접근성 있는 글쓰기
├── 빠른 심사

질적연구:
├── 방법론적 논문
├── 개념적 논문
├── 경험적 논문
└── 논평/비평
```

### 4. Teachers College Record (TCR)

```
기본 정보:
├── 출판사: Columbia University
├── IF: ~2
├── 역사: 1900년 창간
├── 다양한 주제

특징:
├── 실무 지향
├── 다양한 방법론
├── 교사 교육
├── 정책 분석

질적연구 환영:
├── 사례연구
├── 문화기술지
├── 내러티브
└── 비판적 연구
```

### 5. Journal of Teacher Education (JTE)

```
기본 정보:
├── 출판사: AACTE (Sage)
├── IF: ~4
├── 초점: 교사 교육

주제:
├── 예비 교사 교육
├── 현직 교사 발달
├── 교사 정체성
├── 교수법 개발

질적연구 유형:
├── Self-study
├── 실행 연구
├── 사례연구
├── 내러티브 연구
```

## 질적연구 전문/친화적 저널

### 1. Qualitative Inquiry

```
기본 정보:
├── 출판사: Sage
├── IF: ~2
├── 방법론적 혁신 중시

특징:
├── 실험적 글쓰기 환영
├── 예술 기반 연구
├── 비판적 접근
├── 방법론 논의

유형:
├── 자문화기술지
├── 퍼포먼스 문화기술지
├── 시각적 방법
└── 창의적 표현
```

### 2. International Journal of Qualitative Studies in Education (IJQSE)

```
기본 정보:
├── 출판사: Taylor & Francis
├── IF: ~2
├── 질적연구 전문

범위:
├── K-12 교육
├── 고등교육
├── 평생교육
├── 비형식 교육

특징:
├── 국제적 관점
├── 비판적 연구
├── 다양한 방법론
└── 이론적 깊이
```

### 3. Anthropology & Education Quarterly (AEQ)

```
기본 정보:
├── 출판사: AAA (Wiley)
├── IF: ~2
├── 인류학적 접근

특징:
├── 문화기술지 중심
├── 장기 현장 연구
├── 문화와 교육
├── 비교 연구

주제:
├── 학교 문화
├── 교실 상호작용
├── 다문화 교육
└── 언어와 정체성
```

## 교육학 질적연구 특이사항

### 연구 현장 접근

```
도전:
├── IRB 승인 복잡
├── 학교/교육청 승인
├── 학부모 동의
├── 학생 동의/승낙
└── 민감한 주제

대응:
├── 조기 관계 구축
├── 게이트키퍼 협력
├── 명확한 혜택 제시
├── 비밀보장 강조
└── 연구 결과 공유
```

### 교사-연구자 이중 역할

```
유형:
├── 현직 교사 연구자
├── 대학원생 겸 교사
├── 연구 휴직
└── 실행 연구자

고려사항:
├── 역할 경계
├── 권력 관계
├── 반성적 실천
├── 윤리적 이슈
└── 시간 관리
```

### 참여적 연구 전통

```
유형:
├── 참여적 실행 연구 (PAR)
├── 공동체 기반 연구 (CBPR)
├── 청소년 참여 연구 (YPAR)
└── 교사 연구 (Teacher Research)

특징:
├── 참여자 = 공동연구자
├── 실천 변화 목표
├── 민주적 과정
└── 역량 강화
```

## 글쓰기 특이사항

### 교육적 함의 강조

```
필수 요소:
├── 교사에 대한 함의
├── 학교에 대한 함의
├── 정책에 대한 함의
├── 교사 교육 함의
└── 학생에 대한 함의

표현:
"본 연구 결과는 교사들이...
 교사 교육 프로그램에서는...
 정책 입안자들은..."
```

### 일반화 논의

```
질적연구에서:
├── 전이가능성 논의
├── 맥락 상세 기술
├── 독자 판단 권한
├── 이론적 일반화
└── 자연주의적 일반화
```
