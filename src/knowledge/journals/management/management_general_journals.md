# Management 경영학 질적연구 최상위 저널 가이드

## 1. FT50 & UTD24 최상위 저널

### Academy of Management Journal (AMJ)
```
Impact Factor: ~13.5
세계 최고 경영학 저널

에디터 선호:
□ 강력한 이론적 기여
□ 현상 중심 연구 환영
□ Phenomenon-driven theorizing
□ 방법론적 혁신

질적연구 비중: ~25%

선호 방법론 순위:
1. Grounded Theory (Gioia 방법론)
2. Multiple Case Study
3. Ethnography
4. Process Research

워드 제한: 12,000 단어

Desk Rejection 피하기:
□ 첫 3페이지에 contribution 명확히
□ "Why should readers care?" 대답
□ Data structure table 포함 (Gioia시)
□ 현상의 compelling한 서술

수정 요청 패턴:
- "Contribution not clear enough"
- "Why this context?"
- "Theoretical mechanism unclear"
```

## 3. 핵심 이론과 방법론

### Gioia Methodology
```
가장 영향력 있는 귀납적 이론구축 방법

3단계 구조:
1st Order Concepts → 참여자 언어
2nd Order Themes → 연구자 해석
Aggregate Dimensions → 이론적 개념

데이터 구조표 필수 요소:
├── 왼쪽: 대표 인용문
├── 중앙: 1차/2차 개념
└── 오른쪽: 집합 차원

주요 참고:
- Gioia, Corley, & Hamilton (2013), ORM
- 인용수 10,000+ 
```

### Process Research (Langley)
```
과정 연구 전략:

1. Narrative Strategy
   - 풍부한 서사 구성
   - 시간순 전개

2. Quantification Strategy
   - 사건의 수량화
   - 패턴 시각화

3. Alternate Templates Strategy
   - 다중 이론 렌즈 적용
   - 가장 적합한 이론 선택

4. Grounded Theory Strategy
   - 귀납적 개념 발견
   - 과정 모델 구축

5. Visual Mapping Strategy
   - 다이어그램 표현
   - 인과 관계 시각화

6. Temporal Bracketing Strategy
   - 시기 구분
   - 각 시기 비교

7. Synthetic Strategy
   - 사례 간 공통 과정
   - 추상화된 모델
```

### Institutional Ethnography (Dorothy Smith)
```
특성:
- 페미니스트 사회학 기원
- 일상 경험에서 출발
- 지배 관계(ruling relations) 분석

조직 연구 적용:
- 텍스트와 권력
- 업무의 사회적 조직
- 제도적 담론

분석 초점:
- 지역적 경험 → 제도적 과정
- 텍스트 매개 조정
- 사회적 관계 추적
```

### Sensemaking (Weick)
```
핵심 속성 (7가지):
1. Grounded in identity
2. Retrospective
3. Enactive of environments
4. Social
5. Ongoing
6. Focused on cues
7. Driven by plausibility

연구 방법:
- Critical incident 분석
- Narrative 수집
- Real-time observation
- 의미구성 과정 추적

대표 연구:
- Mann Gulch disaster (Weick, 1993)
- Bristol Royal Infirmary (Weick & Sutcliffe)
```

## 4. 데이터 구조 예시: 조직 변화 연구

```
1st Order Concepts (참여자 언어)
├── "처음엔 왜 바꾸는지 몰랐어요"
├── "리더가 비전을 설명해줬죠"
├── "동료들이랑 얘기하면서 이해했어요"
├── "이제는 제가 설명해요"
└── "우리 정체성의 일부가 됐어요"
         ↓
2nd Order Themes (연구자 해석)
├── Initial confusion
├── Top-down communication
├── Peer sensegiving
├── Role as change agent
└── Identity integration
         ↓
Aggregate Dimensions (이론적 개념)
├── DISRUPTED SENSEMAKING
├── MULTI-LEVEL SENSEGIVING
└── IDENTITY TRANSFORMATION

이론적 통합:
→ Cascading Sensemaking Model of Change
```

## 5. 저널별 제출 전략

### AMJ 타겟팅
```
Hook 전략:
"We know a lot about X, but surprisingly
little about HOW/WHY..."

강조점:
1. Phenomenon (compelling 현상)
2. Theory gap (명확한 공백)
3. Findings (data structure)
4. Contribution (3개 이상)

피해야 할 표현:
× "explore" (탐색적이라는 인상)
× "this study fills a gap" (gap filling 혐오)
× "no research has examined" (부정적 프레이밍)

대신 사용:
○ "advances understanding of"
○ "reveals the process by which"
○ "develops theory that explains"
```

### ASQ 타겟팅
```
차별화:
- 더 긴 시간 투자 (2-3년 심사)
- 이론적 깊이 극대화
- 역사적/제도적 관점 환영

필수 요소:
1. Novel theoretical insight
2. Surprising finding
3. Methodological sophistication
4. Beautiful writing
```

### JMS 타겟팅
```
유럽적 관점:
- 비판적 이론 환영
- 사회적 맥락 강조
- 실천적 함의 중시

Special Issue 활용:
- 정기적 CFP 확인
- 주제 적합성 평가
- 게스트 에디터 관계 구축
```

## 6. 심사 대응 전략

### 일반적 Reviewer Comments
```
"Contribution not clear"
→ Discussion 첫 문단 재구성
→ Contribution statement 표 추가
→ What we knew / What we now know

"Sampling justification"
→ Theoretical sampling 논리 강화
→ Information power 개념 활용
→ Context importance 강조

"Trustworthiness concerns"
→ Lincoln & Guba 기준 명시적 논의
→ 추가 검증 절차 기술
→ 연구자 반성성 추가

"Generalizability"
→ Transferability로 프레이밍
→ Boundary conditions 명시
→ 이론적 일반화 강조
```
