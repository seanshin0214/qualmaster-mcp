# 후기실증주의 패러다임 (Postpositivism)

## 핵심 전제

### 존재론 (Ontology)
- **비판적 실재론 (Critical Realism)**: 객관적 현실 존재하나 불완전하게 파악
- 현실의 복잡성: 완전한 인과관계 규명 어려움
- 확률적 인과: 결정론에서 확률론으로

### 인식론 (Epistemology)
- **수정된 객관주의**: 연구자 영향 인정하되 최소화 노력
- 객관성 추구: 편견 통제 기법 적용
- 반증 가능성: Popper의 영향

### 방법론 (Methodology)
- **수정된 실험적**: 준실험, 자연 실험
- 비판적 다원주의: 다양한 방법 수용
- 삼각화: 다중 방법으로 확인

## 실증주의와의 차이

| 측면 | 실증주의 | 후기실증주의 |
|------|---------|------------|
| 현실관 | 완전히 파악 가능 | 불완전하게 파악 |
| 객관성 | 절대적 | 추구하되 한계 인정 |
| 검증 | 검증 (Verification) | 반증 (Falsification) |
| 인과 | 결정론적 | 확률론적 |

## 질적연구와의 관계

### 후기실증주의적 질적연구

**특징**
- 신뢰도와 타당도 강조
- 코딩 일치도 측정
- 일반화 가능성 추구
- 체계적 절차 선호

**적용 예**
- 내용분석 (Content Analysis)
- 구조화된 관찰
- Yin의 사례연구
- 일부 근거이론 (Glaser)

### 품질 기준 적용

| 양적 기준 | 질적 적용 |
|---------|---------|
| 내적 타당도 | 신빙성 (Credibility) |
| 외적 타당도 | 전이가능성 (Transferability) |
| 신뢰도 | 의존가능성 (Dependability) |
| 객관성 | 확인가능성 (Confirmability) |

## 비판

### 구성주의자들의 비판
- 여전히 객관성 신화
- 권력 관계 간과
- 연구자 위치성 경시

### 방어 논리
- 실용적 접근
- 정책 연구에 적합
- 축적적 지식 가능

## 대표 학자
- Karl Popper: 반증주의
- Thomas Kuhn: 패러다임 전환
- Imre Lakatos: 연구 프로그램

## 현대적 적용

### 혼합방법 연구
- 질적-양적 통합의 인식론적 기반
- 실용주의와의 연결

### 근거 기반 실천
- 의료, 교육, 정책 분야
- 체계적 문헌고찰
- 메타분석
