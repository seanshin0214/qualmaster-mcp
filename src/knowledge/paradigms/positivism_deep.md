# 실증주의 패러다임 (Positivism)

## 핵심 전제

### 존재론 (Ontology)
- **순박한 실재론 (Naive Realism)**: 객관적 현실이 독립적으로 존재
- 단일한 현실: 모든 관찰자에게 동일한 현실
- 측정 가능성: 현실은 관찰과 측정을 통해 파악 가능

### 인식론 (Epistemology)
- **객관주의**: 연구자와 연구 대상은 독립적
- 가치 중립: 연구자의 가치가 연구에 영향을 미치지 않음
- 일반화 추구: 보편적 법칙과 원리 발견

### 방법론 (Methodology)
- **실험적/조사적**: 가설-연역적 접근
- 양적 방법 선호: 통계적 분석, 인과관계 규명
- 조작적 정의: 개념의 측정 가능한 지표화

## 역사적 발전

### 논리실증주의 (Logical Positivism)
- 1920-30년대 비엔나 학파
- 검증 가능성 원리
- 형이상학적 진술 배제

### 비판적 합리주의 (Popper)
- 반증 가능성 (Falsifiability)
- 과학적 지식의 잠정성
- 추측과 논박의 방법

## 질적연구에서의 한계

### 연구 목적의 제한
- 깊은 이해보다 일반화 추구
- 맥락적 의미 간과
- 참여자 관점 경시

### 실증주의 전통의 질적연구
- 내용분석 (Content Analysis)
- 구조화된 관찰
- 범주적 코딩

## 질적연구와의 관계

### 긴장 영역
1. 객관성 vs 해석성
2. 일반화 vs 특수성
3. 측정 vs 의미
4. 가치중립 vs 가치개입

### 통합 시도
- 혼합방법 연구의 실증주의적 기반
- 양적 타당도 개념의 적용 시도

## 대표 학자
- Auguste Comte (사회학의 아버지)
- Émile Durkheim (사회적 사실)
- Karl Popper (반증주의)

## 현대적 의의

실증주의는 질적연구의 직접적 기반은 아니지만:
- 비교 기준점 제공
- 엄밀성 논의의 출발점
- 혼합방법론에서의 역할
