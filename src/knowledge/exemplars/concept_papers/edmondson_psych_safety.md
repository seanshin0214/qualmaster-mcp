# Edmondson의 Psychological Safety 개념 분석

## 논문 정보

- **저자**: Amy C. Edmondson
- **대표 논문**:
  - "Psychological Safety and Learning Behavior in Work Teams" (1999, ASQ)
  - "The Fearless Organization" (2019, 저서)
- **핵심 개념**: Psychological Safety (심리적 안전감)

## 개념 정의

### 정의
"팀 내에서 대인관계 위험(interpersonal risk)을 감수하는 것이 안전하다는 팀 구성원들의 공유된 믿음"

### 핵심 요소
1. **공유된 믿음**: 개인이 아닌 팀 수준
2. **대인관계 위험**: 무안, 거절, 처벌의 두려움
3. **안전감**: 자유롭게 발언하고 행동할 수 있음

## 개념 개발 과정

### 이론적 기반
- **Schein & Bennis (1965)**: 조직 변화와 심리적 안전
- **Kahn (1990)**: 심리적 안전과 직원 몰입
- **학습 이론**: 조직 학습과 실수 공유

### Edmondson의 기여
1. **팀 수준 구성개념화**: 개인에서 팀으로 확장
2. **측정 도구 개발**: 7문항 척도
3. **실증적 검증**: 병원 팀 연구

## 개념의 구조

### 이론적 명제

**명제 1**: 심리적 안전감은 팀 학습 행동을 촉진한다
```
심리적 안전감 → 학습 행동 (질문, 피드백 추구, 실험, 실수 논의)
```

**명제 2**: 팀 학습 행동은 팀 성과를 향상시킨다
```
학습 행동 → 팀 성과
```

**명제 3**: 심리적 안전감은 팀 학습 행동을 통해 간접적으로 성과에 영향
```
심리적 안전감 → 학습 행동 → 팀 성과
```

### 심리적 안전감의 선행요인

**리더 행동**
- 접근 가능성 (Accessibility)
- 실수에 대한 반응
- 참여 초대

**팀 구조**
- 역할 명확성
- 목표 상호의존성

**지원적 맥락**
- 조직 문화
- 보상 시스템

## 개념의 차별화

### 무엇이 아닌가?

| 심리적 안전감 ≠ | 구별점 |
|---------------|-------|
| 신뢰 (Trust) | 신뢰는 개인 간, 심리적 안전감은 집단 수준 |
| 집단 응집력 | 응집력 높은 팀도 안전감 낮을 수 있음 |
| 쉬운 업무 환경 | 높은 기준과 양립 가능 |
| 갈등 회피 | 오히려 건설적 갈등 촉진 |

### 핵심 구별
"심리적 안전감은 건설적 비판을 편안하게 하는 것이지, 비판 자체를 피하는 것이 아니다"

## 측정 도구

### 7문항 척도 (Edmondson, 1999)
1. 이 팀에서 실수를 하면 자주 비난받는다 (R)
2. 이 팀원들은 어려운 문제를 제기할 수 있다
3. 이 팀 사람들은 때때로 다른 사람이 다르다고 거부한다 (R)
4. 이 팀에서는 안심하고 위험을 감수할 수 있다
5. 이 팀의 다른 구성원들에게 도움을 구하기 어렵다 (R)
6. 이 팀의 어느 누구도 나의 노력을 의도적으로 훼손하지 않을 것이다
7. 이 팀원들과 일할 때 나의 고유한 기술과 재능이 가치있게 여겨진다

*(R) = 역코딩 문항*

## 이론적 기여

### 왜 중요한가?

**실용적 기여**
- 팀 효과성의 핵심 요인 규명
- 리더십 개발 시사점
- 조직 개입의 레버리지 포인트

**이론적 기여**
- 학습 이론과 팀 연구 연결
- 분석 수준 문제 해결
- 미시-거시 연결

### Google의 Project Aristotle
- 효과적 팀의 5가지 요인 중 1위
- Edmondson의 개념을 실증적으로 확인
- 산업계 주목

## 개념 구축 전략 분석

### 강점

**현상 관찰에서 출발**
- 병원 팀의 실수 보고 패턴 관찰
- 역설적 발견: 좋은 팀이 더 많은 실수 보고
- 귀납적 통찰

**기존 개념의 확장**
- Schein의 개념을 팀 수준으로
- 측정 가능하게 조작화
- 실증적 검증

**명확한 경계 설정**
- 유사 개념과 구별
- 수준 명시 (팀 수준)
- 범위 조건 설정

### 논문 구조 분석

```
1. 도입: 왜 팀이 학습하는가의 퍼즐
2. 이론: 심리적 안전감 개념 발전
3. 가설: 심리적 안전감 → 학습 → 성과
4. 방법: 51개 병원 팀 연구
5. 결과: 가설 지지
6. 논의: 이론적, 실용적 함의
```

## 교훈: 개념 논문 작성을 위해

### 성공 요인
1. **관찰된 퍼즐에서 시작**: 의외의 패턴
2. **기존 개념의 창의적 확장**: 수준 전환
3. **명확한 정의와 경계**: 무엇이고 무엇이 아닌지
4. **측정 가능한 조작화**: 검증 가능성
5. **다중 검증**: 이론과 실증의 결합
