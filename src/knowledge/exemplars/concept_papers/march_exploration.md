# March의 Exploration vs Exploitation 개념 분석

## 논문 정보

- **저자**: James G. March
- **논문**: "Exploration and Exploitation in Organizational Learning" (1991, Organization Science)
- **인용**: 30,000+ (가장 많이 인용된 경영학 논문 중 하나)
- **핵심 개념**: Exploration/Exploitation Trade-off

## 개념 정의

### Exploration (탐색)
"새로운 대안의 탐색. 변이, 위험 감수, 실험, 유연성, 발견, 혁신"

특징:
- 새로운 가능성 추구
- 불확실성 수용
- 장기적 이익
- 변이 생성

### Exploitation (활용)
"기존 확실성의 정교화. 선택, 효율성, 선택, 구현, 실행"

특징:
- 기존 역량 활용
- 확실성 추구
- 단기적 이익
- 효율성

## 핵심 논증

### 트레이드오프 논리

**희소 자원 경쟁**
```
자원 (시간, 주의, 돈)
       ↓
분배 결정 필요
       ↓
Exploration ←→ Exploitation
```

**자기강화 역학**
- Exploitation 성공 → 더 많은 Exploitation
- Exploration 부재 → 적응 실패
- 역량 함정 (Competency Trap)

### 근시안 문제 (Myopia)

**시간적 근시**
- 단기 성과 선호
- 장기 학습 무시
- 근시안적 조직

**공간적 근시**
- 인접 영역 선호
- 원거리 탐색 회피

## 개념 구축 전략 분석

### 1. 컴퓨터 시뮬레이션

**March의 방법**
- 수학적 모델링
- 시뮬레이션 실험
- 파라미터 변화 분석

**시뮬레이션 모델 (단순화)**
```
조직 코드 (Organization Code): 신념 체계
개인 (Individuals): 자신의 신념
학습:
  - 개인 → 조직 (Socialization)
  - 조직 → 개인 (Exploration)
```

### 2. 은유적 표현

**Exploration/Exploitation 은유**
- 직관적 이해 가능
- 광범위한 적용
- 기억하기 쉬움

### 3. 이분법적 구조

**대비를 통한 명확화**
| Exploration | vs | Exploitation |
|------------|---|-------------|
| 새로움 | | 기존 |
| 변이 | | 선택 |
| 위험 | | 효율 |
| 장기 | | 단기 |

### 4. 역설적 긴장

**양면성 (Ambidexterity) 개념으로 발전**
- 둘 다 필요
- 균형의 어려움
- 조직적 과제

## 이론적 기여

### 왜 영향력 있나?

**보편성**
- 조직 수준
- 개인 수준
- 국가 수준
- 생태계 수준

**다양한 적용 영역**
- 혁신 관리
- 전략적 갱신
- 조직 학습
- 기업가정신

**후속 연구 촉발**
- Organizational Ambidexterity
- Dynamic Capabilities
- Strategic Renewal

### 논문의 구조

```
1. Introduction: 학습의 문제
2. Model: 수학적 모델 제시
3. Simulation Results: 결과
4. Implications:
   - 탐색 vs 활용 트레이드오프
   - 근시안 문제
   - 적응의 역설
5. Conclusion: 조직 학습의 딜레마
```
