# Weick의 Sensemaking 개념 분석

## 논문 정보
- **저자**: Karl E. Weick
- **대표 저작**:
  - "Sensemaking in Organizations" (1995, Sage)
  - "The Social Psychology of Organizing" (1979)
- **핵심 개념**: Sensemaking (의미형성/센스메이킹)

## 개념 정의

### Weick의 정의
> "Sensemaking is about the ways people generate what they interpret."
> "센스메이킹은 사람들이 자신이 해석하는 것을 생성하는 방식에 관한 것이다."

### 핵심 아이디어
- 조직 구성원들이 모호한 상황을 이해하려는 과정
- 사후적으로 의미를 부여하는 활동
- 행동과 해석의 상호작용

## Sensemaking의 7가지 속성

### 1. Grounded in Identity Construction
```
"나는 누구인가?" → 해석에 영향
- 자아 개념이 센스메이킹의 기반
- 정체성이 무엇을 주목하는지 결정
- 역할과 책임이 해석 틀 제공
```

### 2. Retrospective
```
"지나간 후에야 이해한다"
- 경험 후 의미 부여
- 현재 시점에서 과거 재해석
- "우리는 행동한 후에야 안다"
```

### 3. Enactive of Sensible Environments
```
"환경을 만들어낸다"
- 수동적 발견 X
- 능동적 환경 창조
- 자기충족적 예언
```

### 4. Social
```
"함께 의미를 만든다"
- 사회적 상호작용을 통해
- 공유된 의미 형성
- 타인의 행동이 단서 제공
```

### 5. Ongoing
```
"멈추지 않는 흐름"
- 지속적 과정
- 시작과 끝 불분명
- 경험의 연속 속에서
```

### 6. Focused on and by Extracted Cues
```
"단서에 집중"
- 환경에서 단서 추출
- 선택적 주목
- 단서가 이야기 촉발
```

### 7. Driven by Plausibility Rather than Accuracy
```
"정확성보다 타당성"
- 완벽한 정보 불필요
- 그럴듯함이 중요
- 행동 가능하면 충분
```

## 개념 발전 과정

### 이론적 배경
```
사회심리학       → 인지적 측면
├── Festinger   → 인지부조화
├── Garfinkel   → 민속방법론
└── Schutz      → 현상학

조직이론         → 조직적 맥락
├── March & Simon → 제한된 합리성
├── Thompson    → 불확실성
└── Perrow      → 복잡성
```

### 진화 과정
```
1969: "The Social Psychology of Organizing" 1판
      - 조직화 과정으로서의 센스메이킹

1979: "The Social Psychology of Organizing" 2판
      - 해석(enactment) 개념 강화

1988: "Enacted Sensemaking in Crisis Situations"
      - 위기상황 적용

1993: "The Collapse of Sensemaking in Organizations"
      - Mann Gulch 화재 분석

1995: "Sensemaking in Organizations"
      - 7가지 속성 정리
      - 체계적 이론화
```

## 개념화 기법 분석

### 1. 풍부한 예시 활용
```
Mann Gulch 화재 사례:
- 극적인 스토리텔링
- 구체적 행동 묘사
- 이론-현상 연결
- 기억에 남는 설명
```

### 2. 기존 개념과의 연결
```
센스메이킹 vs 해석 (Interpretation)
├── 해석: 외부 자극에 대한 반응
└── 센스메이킹: 자극을 만들어내는 과정

센스메이킹 vs 이해 (Understanding)
├── 이해: 결과
└── 센스메이킹: 과정
```

### 3. 역설과 긴장
```
"How can I know what I think until I see what I say?"
(내가 무슨 말을 하는지 보기 전까지
 내가 무엇을 생각하는지 어떻게 알 수 있나?)

→ 행동이 사고에 선행
→ 통념에 대한 도전
→ 기억에 남는 표현
```

### 4. 시각적 모델
```
        환경
         ↓
      제정(Enactment)
         ↓
      선택(Selection)
         ↓
      보유(Retention)
         ↓
        환경
         (순환)
```
