# Hotfix: Claude Desktop MCP 설정 오류 수정

**날짜**: 2025-11-30
**유형**: Configuration Hotfix
**심각도**: Medium

## 문제 상황

Claude Desktop에서 QualMaster MCP 사용 시 "Tool execution failed" 오류 발생

## 원인 분석

`claude_desktop_config.json` 설정 파일에서 `"disabled": true` 옵션이 활성화되어 있어 MCP 서버가 비활성화 상태였음

```json
// 문제 설정
"qualmaster": {
  "command": "node",
  "args": ["...\\dist\\index.js"],
  "disabled": true  // <-- 원인
}
```

## 해결 방법

1. 설정 파일 위치: `%APPDATA%\Claude\claude_desktop_config.json`
2. `disabled: true` 옵션 제거
3. Claude Desktop 재시작

```json
// 수정된 설정
"qualmaster": {
  "command": "node",
  "args": ["C:\\Users\\sshin\\Documents\\qualmaster-mcp-server\\dist\\index.js"]
}
```

## 검증 항목

- [x] ChromaDB 서버 실행 상태 확인 (localhost:8000)
- [x] node_modules 설치 확인
- [x] dist/index.js 빌드 파일 존재 확인
- [x] 서버 단독 실행 테스트 통과

## 재발 방지

- MCP 서버 설정 시 `disabled` 옵션 주의
- 새로운 MCP 추가 후 설정 파일 검토 필수
