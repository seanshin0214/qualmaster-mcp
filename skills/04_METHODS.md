# 질적연구 방법

## 데이터 수집

### 1. 면담 (Interview)

#### 유형
| 유형 | 구조화 정도 | 적합한 상황 |
|------|-----------|-----------|
| 비구조화 | 최소 | 현상학, 탐색 연구 |
| 반구조화 | 중간 | 대부분의 질적연구 |
| 구조화 | 높음 | 다수 면담, 비교 목적 |

#### 면담 가이드 작성

**구조**
1. 도입 질문 (Ice-breaker)
2. 전환 질문 (Transition)
3. 핵심 질문 (Key Questions)
4. 마무리 질문 (Closing)

**질문 유형**
- 경험 질문: "~할 때 어떤 일이 있었나요?"
- 의미 질문: "그것은 당신에게 어떤 의미인가요?"
- 감정 질문: "그때 어떤 느낌이 드셨나요?"

**피해야 할 질문**
- 유도 질문
- 이중 질문
- '왜' 질문 과용 (방어 유발)

### 2. 관찰 (Observation)

#### 참여관찰 스펙트럼
```
완전 참여자 ─── 참여적 관찰자 ─── 관찰적 참여자 ─── 완전 관찰자
```

#### 현장노트 구조
- **기술 노트**: 관찰 사실
- **반성 노트**: 연구자 해석, 감정
- **방법 노트**: 연구 절차 결정

### 3. 문서/인공물

#### 유형
- 공식 문서 (정책, 보고서)
- 개인 문서 (일기, 편지)
- 온라인 자료 (SNS, 포럼)
- 인공물 (물건, 사진)

## 표집 (Sampling)

### 목적적 표집 전략

| 전략 | 설명 | 적합한 상황 |
|------|------|-----------|
| 최대 변이 | 다양한 사례 | 범위 탐색 |
| 동질적 | 유사한 사례 | 하위집단 심층 |
| 극단/일탈 사례 | 특이한 사례 | 통찰 획득 |
| 전형적 사례 | 대표적 사례 | 일반적 이해 |
| 기준 | 특정 조건 충족 | 경험 보유자 |
| 눈덩이 | 참여자 추천 | 접근 어려운 집단 |

### 이론적 표집 (GT용)
- 분석 결과가 다음 표집 결정
- 범주 발전 위해 표집
- 포화까지 지속

### 표본 크기 가이드

| 전통 | 권장 규모 | 근거 |
|------|---------|------|
| 현상학 | 5-25명 | 경험의 본질 |
| 근거이론 | 20-30+명 | 이론적 포화 |
| 문화기술지 | 한 집단 | 문화 전체 |
| 내러티브 | 1-3명 | 심층 이야기 |
| 사례연구 | 1-5사례 | 맥락적 심층 |

## 데이터 분석

### 코딩 (Coding)

#### 코딩 유형
1. **서술적 코딩**: 주제 요약
2. **과정 코딩**: 행위, 동사형 (Gerund)
3. **인-비보 코딩**: 참여자 언어
4. **가치 코딩**: 가치, 태도, 신념
5. **감정 코딩**: 정서 상태

#### 코딩 사이클
```
1차 코딩 (Initial) → 2차 코딩 (Focused) → 범주화 → 주제화
```

### 주제분석 6단계 (Braun & Clarke)
1. 데이터 친숙화
2. 초기 코드 생성
3. 주제 탐색
4. 주제 검토
5. 주제 정의/명명
6. 보고서 작성
