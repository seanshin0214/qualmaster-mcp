# 질적연구 패러다임

## 개요

패러다임(Paradigm)은 연구자의 세계관으로, 존재론, 인식론, 방법론에 대한 기본 가정의 체계입니다.

## 4대 패러다임

### 1. 실증주의 (Positivism)

| 차원 | 입장 |
|------|------|
| 존재론 | 순박한 실재론 - 객관적 현실 존재 |
| 인식론 | 객관주의 - 연구자와 대상 분리 |
| 방법론 | 실험적, 조사적 - 가설 검증 |

**질적연구와의 관계**: 직접적 기반은 아니나, 비교 기준점 제공

### 2. 후기실증주의 (Postpositivism)

| 차원 | 입장 |
|------|------|
| 존재론 | 비판적 실재론 - 불완전하게 파악 가능한 현실 |
| 인식론 | 수정된 객관주의 - 연구자 영향 인정 |
| 방법론 | 수정된 실험적 - 비판적 다원주의 |

**질적연구와의 관계**: 타당도/신뢰도 개념의 질적 적용 시도

### 3. 구성주의/해석주의 (Constructivism/Interpretivism)

| 차원 | 입장 |
|------|------|
| 존재론 | 상대주의 - 다중적, 구성된 현실 |
| 인식론 | 주관주의 - 연구자와 참여자 공동 구성 |
| 방법론 | 해석적, 변증법적 - 의미 이해 |

**질적연구와의 관계**: 대부분의 질적연구 전통의 기반

### 4. 비판이론/참여적 (Critical Theory/Participatory)

| 차원 | 입장 |
|------|------|
| 존재론 | 역사적 실재론 - 권력에 의해 형성된 현실 |
| 인식론 | 가치 개입적 - 연구는 정치적 |
| 방법론 | 대화적, 변혁적 - 해방과 변화 추구 |

**질적연구와의 관계**: 비판적 문화기술지, 참여적 행동연구

## 패러다임 선택 가이드

### 질문별 적합한 패러다임

| 연구 질문 유형 | 적합한 패러다임 |
|--------------|----------------|
| "경험은 어떠한가?" | 구성주의 |
| "과정은 어떻게 진행되는가?" | 구성주의 |
| "왜 불평등이 지속되는가?" | 비판이론 |
| "어떻게 변화를 이끌 수 있는가?" | 참여적 |

### 패러다임 정합성 체크

연구에서 다음이 일관되어야 합니다:
1. **존재론적 가정** - 현실의 본질
2. **인식론적 가정** - 지식의 본질
3. **방법론적 선택** - 연구 접근법
4. **방법 선택** - 구체적 기법

## 논문에서의 패러다임 기술

### 필수 요소
- 패러다임 명시
- 존재론/인식론적 가정 설명
- 방법론과의 정합성 논증

### 예시 문장
```
"본 연구는 구성주의 패러다임에 기반합니다. 이 관점에서
현실은 개인과 사회의 상호작용을 통해 구성되며,
연구자는 참여자와의 대화를 통해 의미를 공동으로
구성해 나갑니다 (Creswell & Poth, 2018)."
```
