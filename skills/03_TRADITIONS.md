# 질적연구 5대 전통

## 개요

Creswell & Poth(2018)에 따른 5대 질적연구 전통입니다.

## 빠른 비교표

| 전통 | 핵심 질문 | 초점 | 분석 단위 |
|------|----------|------|----------|
| 현상학 | 경험의 본질은? | 체험 | 경험 |
| 근거이론 | 과정/이론은? | 이론 개발 | 과정 |
| 문화기술지 | 문화는 어떠한가? | 문화 기술 | 집단 |
| 내러티브 | 이야기는 무엇인가? | 생애 이야기 | 개인 |
| 사례연구 | 사례의 특성은? | 심층 이해 | 사례 |

## 1. 현상학 (Phenomenology)

### 적합한 상황
- 특정 경험의 본질 탐구
- "~을 경험한다는 것은 어떤 것인가?"
- 체험의 의미 이해

### 주요 접근
- **기술적 현상학** (Husserl): 에포케, 본질 기술
- **해석적 현상학** (Heidegger): 해석학적 순환
- **IPA** (Smith): 심리학적 적용

### 데이터
- 심층 면담 (1-2회, 60-90분)
- 참여자: 5-25명 (경험 보유자)

### 분석 절차 (Moustakas)
1. 에포케 (Bracketing)
2. 수평화 (Horizontalization)
3. 의미 단위 구성
4. 텍스처/구조 기술
5. 본질 통합

## 2. 근거이론 (Grounded Theory)

### 적합한 상황
- 이론이 없거나 부족한 현상
- 과정/단계 탐구
- "어떤 과정으로 ~하는가?"

### 주요 접근
- **고전 GT** (Glaser): 출현 강조
- **체계적 GT** (Strauss & Corbin): 패러다임 모델
- **구성주의 GT** (Charmaz): 해석적 접근

### 데이터
- 면담, 관찰, 문서
- 참여자: 20-30명+
- 이론적 표집

### 코딩 절차
1. **개방 코딩**: 개념 생성
2. **축 코딩**: 범주 관계
3. **선택 코딩**: 핵심 범주, 이론 통합

## 3. 문화기술지 (Ethnography)

### 적합한 상황
- 문화 공유 집단 연구
- 장기간 현장 몰입 가능
- "이 집단의 문화는 어떠한가?"

### 주요 접근
- **고전적**: Malinowski, Geertz
- **비판적**: 권력과 불평등
- **자문화기술지**: 연구자의 경험

### 데이터
- 참여관찰 (6개월-2년+)
- 면담, 문서, 인공물
- 현장노트

### 분석
- 문화적 주제 분석
- 두꺼운 기술 (Thick Description)
- Emic & Etic 관점

## 4. 내러티브 탐구 (Narrative Inquiry)

### 적합한 상황
- 개인의 생애 이야기
- 시간적 순서와 맥락 중요
- "삶의 이야기는 무엇인가?"

### 주요 접근
- **생애사**: 전체 삶
- **구술사**: 역사적 사건
- **회복적 내러티브**: 이야기의 치유력

### 데이터
- 생애사 면담 (다회기)
- 참여자: 1-3명 (심층)
- 문서, 사진, 인공물

### 분석
- 연대기적 재구성
- 내러티브 구조 분석
- 플롯, 등장인물, 배경

## 5. 사례연구 (Case Study)

### 적합한 상황
- 경계 지어진 체계 연구
- 맥락 내에서의 이해
- "이 사례의 특성은?"

### 주요 접근
- **Yin**: 실증적, 다중 사례
- **Stake**: 해석적, 내재적/도구적
- **Merriam**: 교육학적 적용

### 유형
- **내재적**: 사례 자체 관심
- **도구적**: 이슈 이해 도구
- **집합적**: 다중 사례

### 데이터
- 다중 출처: 면담, 관찰, 문서
- 삼각화 강조

### 분석
- 사례 내 분석
- 교차 사례 분석
- 주제적/총체적

## 전통 선택 의사결정 트리

```
연구 질문 분석
      │
      ├─ 경험의 본질? → 현상학
      │
      ├─ 과정/이론 개발? → 근거이론
      │
      ├─ 문화 공유 집단? → 문화기술지
      │
      ├─ 개인 생애 이야기? → 내러티브
      │
      └─ 특정 사례 심층? → 사례연구
```
