# 질적연구 품질 기준

## Lincoln & Guba (1985) 신뢰성

### 4대 기준

#### 1. 신빙성 (Credibility)
**정의**: 결과가 참여자 관점을 정확히 반영

**확보 전략**
| 전략 | 설명 | 적용 |
|------|------|------|
| 장기간 관여 | 충분한 현장 시간 | 맥락 이해 |
| 삼각화 | 다중 출처/방법/연구자 | 확인 |
| 동료 브리핑 | 외부 연구자 검토 | 분석 검증 |
| 참여자 확인 | Member Checking | 해석 검증 |
| 부정적 사례 분석 | 반대 사례 탐색 | 이론 정교화 |

#### 2. 전이가능성 (Transferability)
**정의**: 다른 맥락 적용 가능성

**확보 전략**
- 풍부한 기술 (Thick Description)
- 맥락 상세 기술
- 표집 전략 명시

#### 3. 의존가능성 (Dependability)
**정의**: 연구 과정의 일관성, 추적 가능성

**확보 전략**
- 감사 추적 (Audit Trail)
- 연구 절차 상세 기록
- 외부 감사

#### 4. 확인가능성 (Confirmability)
**정의**: 결과가 데이터에 근거

**확보 전략**
- 데이터→결론 추적 가능
- 반성적 저널
- 연구자 가정 명시

## 삼각화 (Triangulation)

### 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 데이터 삼각화 | 다양한 출처 | 면담 + 관찰 + 문서 |
| 연구자 삼각화 | 다수 연구자 | 공동 코딩 |
| 이론적 삼각화 | 다양한 관점 | 다중 이론 적용 |
| 방법론적 삼각화 | 다양한 방법 | 질적 + 양적 |

### 적용 지침
- 모든 유형 필수 아님
- 연구 목적에 맞게 선택
- 불일치는 배제가 아닌 탐구 대상

## 참여자 확인 (Member Checking)

### 확인 대상
1. **전사본 정확성**
2. **코드/범주 적절성**
3. **해석의 공명**

### 수행 방법
- 개별 확인 (전사본, 요약)
- 집단 확인 (포커스 그룹)
- 서면 피드백

### 주의사항
- 모든 참여자 동의 기대 금물
- 불일치는 데이터로 활용
- 연구자 해석 권위 유지

## 반성성 (Reflexivity)

### 정의
연구자가 자신의 가정, 가치, 경험이 연구에 미치는 영향을 인식하고 성찰

### 반성적 저널 작성

**기록 내용**
- 연구자 배경과 가정
- 현장에서의 감정과 반응
- 분석 과정의 결정과 근거
- 편견 인식과 관리

**예시 질문**
- "나는 왜 이렇게 해석했는가?"
- "내 배경이 어떤 영향을 미쳤는가?"
- "다른 해석 가능성은?"

## Tracy (2010) 8가지 기준

### Big-Tent 접근
1. **가치 있는 주제** (Worthy Topic)
2. **풍부한 엄밀성** (Rich Rigor)
3. **진정성** (Sincerity)
4. **신빙성** (Credibility)
5. **공명성** (Resonance)
6. **의미 있는 기여** (Significant Contribution)
7. **윤리** (Ethics)
8. **의미 있는 일관성** (Meaningful Coherence)

## 실천적 체크리스트

### 연구 설계 단계
- [ ] 패러다임 명시
- [ ] 방법론 정합성 확보
- [ ] 표집 전략 계획

### 데이터 수집 단계
- [ ] 충분한 현장 시간
- [ ] 다중 데이터 출처
- [ ] 현장노트/메모 작성

### 데이터 분석 단계
- [ ] 체계적 코딩 절차
- [ ] 동료 검토
- [ ] 부정적 사례 분석
- [ ] 감사 추적 유지

### 결과 보고 단계
- [ ] 풍부한 기술
- [ ] 참여자 확인
- [ ] 연구자 위치성 기술
- [ ] 한계 명시
