# 컨셉 논문 작성 가이드

## 컨셉 논문이란?

### 정의
실증 데이터 없이 이론적 논증만으로 학술적 기여를 하는 논문

### 유형
1. **개념 개발 논문**: 새 개념 제안
2. **이론 구축 논문**: 새 이론 제안
3. **이론 정교화 논문**: 기존 이론 확장/수정
4. **통합 논문**: 분산된 문헌 통합
5. **비평 논문**: 기존 이론 비판

## 논문 구조

### 1. 서론 (Introduction)

**구성요소**
1. Hook: 관심 끌기
2. Gap: 문제/공백 제시
3. Purpose: 논문 목적
4. Contribution: 예상 기여
5. Roadmap: 논문 구조 안내

**서론 공식**
```
Despite [기존 연구의 진전],
[문제/공백]이 존재한다.
This paper addresses this gap by [목적].
We contribute to [이론/실무]에 by [기여 내용].
```

### 2. 이론적 배경 (Theoretical Background)

**목적**
- 기존 문헌 리뷰
- 이론적 기반 수립
- Gap 정교화

**구조화 전략**
- 시간순
- 주제별
- 논쟁별
- 학파별

### 3. 이론 개발 (Theory Development)

**핵심 섹션**
- 개념 정의와 차원
- 이론적 명제 (Propositions)
- 메커니즘 설명
- 경계 조건

**명제 작성 가이드**

```
Proposition 1: A가 증가하면 B가 증가한다.

Proposition 2: C는 A와 B의 관계를 조절하여,
              C가 높을 때 A→B 관계가 강화된다.
```

### 4. 논의 (Discussion)

**포함 내용**
- 이론적 기여 요약
- 기존 이론과의 관계
- 실무적 함의
- 한계와 향후 연구

## 흔한 실수

### 1. "So What?" 문제
- 기여가 불분명
- **해결**: 서론에서 기여 명시적 진술

### 2. "Old Wine in New Bottles"
- 기존 개념 재포장
- **해결**: 차별점 명확히 논증

### 3. "Logic Gap"
- 논리적 비약
- **해결**: 단계별 논증 강화

### 4. "Laundry List"
- 구조 없는 나열
- **해결**: 논리적 구조화

### 5. "Missing Mechanism"
- 왜/어떻게 설명 부재
- **해결**: 메커니즘 명시적 설명

## 저널별 특성

### AMR (Academy of Management Review)
- 이론 논문 전문
- 12,000 단어
- 높은 추상화 수준

### Organization Science
- 이론 + 실증 모두
- 참신성 강조
- 10,000 단어

### Journal of Management
- 리뷰/통합 논문
- 실무 연계 강조

## 글쓰기 팁

### 명제 vs 가설
- **명제 (Proposition)**: 이론적 진술, 검증 전
- **가설 (Hypothesis)**: 검증 가능한 구체적 진술

### 그림 활용
- 개념 모델 시각화
- 관계/과정 도식화
- 복잡성 단순화

### 인용 전략
- 핵심 이론가 필수 인용
- 최신 문헌 포함
- 지나친 자기 인용 피하기
