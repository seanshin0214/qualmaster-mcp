# 저널 투고 가이드

## 저널 선택 전략

### 고려 요소
1. **범위 적합성**: 연구 주제와 저널 범위 일치
2. **방법론 적합성**: 질적연구 게재 이력
3. **영향력**: Impact Factor, 인용 지수
4. **심사 기간**: 리뷰 소요 시간
5. **게재율**: 경쟁 정도

### 질적연구 친화 저널

| 저널 | 분야 | 특징 |
|------|------|------|
| AMJ | 경영 | Top-tier, 질적 환영 |
| ASQ | 조직 | 이론+방법 우수성 |
| Organization Science | 조직 | 혁신적 접근 |
| Human Relations | 조직/HR | 질적 전통 |
| Qualitative Research | 방법론 | 질적 전문 |
| Qualitative Health Research | 보건 | 간호/의료 |

### 컨셉 논문 전문 저널

| 저널 | IF | 게재율 | 특징 |
|------|---|-------|------|
| AMR | ~12 | ~8% | 이론 전문, 최고 수준 |
| AMP | ~9 | ~15% | 접근성, 실무 연계 |
| Organization Science | ~5 | ~10% | 이론+실증 |

## 투고 전 체크리스트

### 저널 적합성
- [ ] 저널 Aims & Scope 확인
- [ ] 최근 게재 논문 분석
- [ ] 유사 주제 게재 여부
- [ ] 에디터/리뷰어 풀 확인

### 원고 준비
- [ ] Author Guidelines 준수
- [ ] 분량 제한 확인
- [ ] 참고문헌 형식 (APA 등)
- [ ] 익명 처리 (Blind Review)
- [ ] 표/그림 형식

## 커버레터 작성

### 구조

```
[날짜]
Dear Editor-in-Chief [이름],

1단락: 투고 목적
We submit our manuscript titled "..." for consideration
in [저널명].

2단락: 논문 개요
Our paper examines... We contribute to... by...

3단락: 저널 적합성
This work is particularly suited for [저널] because...

4단락: 이해충돌/중복게재
We confirm that this work is original and not
under review elsewhere...

5단락: 마무리
We look forward to your decision.

Sincerely,
[저자명]
```

### 커버레터 팁
- 기여를 명확히 진술
- 저널과의 적합성 강조
- 1페이지 이내
- 에디터 이름 확인

## 심사 과정 이해

### 일반적 흐름
```
투고 → 데스크 검토 → 리뷰어 배정 → 심사 → 결정
      (1-2주)      (2-4주)      (2-4개월)
```

### 결정 유형
1. **Accept**: 수정 없이 게재 (드묾)
2. **Minor Revision**: 소폭 수정 후 게재
3. **Major Revision**: 대폭 수정 후 재심사
4. **Reject & Resubmit**: 거절, 재투고 권장
5. **Reject**: 완전 거절

## R&R 대응 전략

### 대응 원칙
1. **모든 코멘트에 대응**: 빠짐없이
2. **구체적으로**: 변경 내용 명시
3. **존중하며**: 비판에도 감사
4. **논리적으로**: 반박 시 근거 제시

### Response Letter 구조

```
Dear Editor and Reviewers,

Thank you for the constructive feedback. Below, we address
each comment point by point.

---
REVIEWER 1

Comment 1.1: [리뷰어 코멘트]

Response: Thank you for this point. We have [대응 설명].
See revised manuscript, page X, lines Y-Z.

[수정된 텍스트 인용]

---
Comment 1.2: ...
```

### 색상 코딩 시스템
- **녹색**: 동의하고 수정
- **노란색**: 부분 동의, 설명 추가
- **빨간색**: 반박 (주의 필요)

## 리젝션 대응

### 즉각 행동 피하기
- 감정적 반응 자제
- 1-2일 후 재검토
- 객관적 분석

### 리젝션 후 전략
1. **피드백 분석**: 핵심 문제 파악
2. **수정 가능성**: 해결 가능한가?
3. **저널 재선택**: 다른 적합 저널
4. **근본적 재작업**: 필요시 재구성

## 자주 받는 리뷰어 코멘트

### 기여 관련
- "The contribution is not clear"
- "What does this add to what we know?"
→ 서론에서 기여 명확히 진술

### 이론 관련
- "The theoretical logic is weak"
- "The mechanism is underspecified"
→ 논증 강화, 메커니즘 상세화

### 방법 관련
- "The method is not rigorous"
- "Sample size is insufficient"
→ 방법론적 정당화 추가

### 글쓰기 관련
- "The paper is hard to follow"
- "The structure is confusing"
→ 재구성, 명확한 표지 사용
